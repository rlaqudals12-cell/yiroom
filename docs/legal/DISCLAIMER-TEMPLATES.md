## 1. 면책 조항 원칙

### 1.1 법적 근거

- 약사법 제44조: 의약품 판매/조제 행위 금지 → "정보 제공"으로 한정
- 화장품법 제13조: 효능/효과 과대 표현 금지
- 의료법 제27조: 무면허 의료행위 금지 → AI 분석은 "참고 정보"

### 1.2 표현 원칙

```
허용: "~를 고려해볼 수 있어요", "~에 대한 정보예요"
금지: "~를 사용하세요", "~가 치료해요", "의사가 추천하는"
```

---

## 2. 안전성 정보 면책

### 2.1 공통 면책 (모든 Safety 정보에 적용)

```
이 정보는 의료 조언이 아닌 일반 참고 정보예요.
알레르기가 심하거나 건강이 걱정되시면 전문 의료인과 상담해주세요.
```

### 2.2 Level별 면책

**Level 1 BLOCK (알레르겐 탐지)**

```
이 성분은 회원님의 알레르기 정보와 일치하여 제외했어요.
정확한 확인은 전문 의료인과 상담해주세요.
성분명: {ingredientName}
```

**Level 2 WARN (주의 성분)**

```
이 성분은 회원님의 상태에서 주의가 필요할 수 있어요.
사용 전 피부 패치 테스트를 권장해요.
더 궁금하시면 전문가와 상의해주세요.
```

**Level 3 INFO (일반 정보)**

```
참고: EWG 등급이 높은 성분이 포함되어 있어요.
EWG 등급은 일반적인 안전성 참고 지표예요.
```

---

## 3. 분석 결과 면책

### 3.1 AI 분석 공통

```
이 분석 결과는 AI가 제공하는 참고 정보예요.
정확한 진단이나 처방은 전문가와 상담해주세요.
```

### 3.2 피부 분석

```
피부 분석 결과는 이미지 기반 참고 정보예요.
정확한 피부 상태 진단은 피부과 전문의에게 문의해주세요.
특이 증상이 있으시면 의료기관을 방문해주세요.
```

### 3.3 영양 분석

```
영양 분석 결과는 일반적인 식이 참고 정보예요.
특정 질환이나 알레르기가 있으시면
전문 영양사 또는 의사와 상담해주세요.
```

### 3.4 운동 추천

```
운동 추천은 일반적인 건강 정보를 기반으로 해요.
심혈관 질환, 관절 질환 등이 있으시면
운동 전 전문의와 상담해주세요.
```

---

## 4. 제품 추천 면책

### 4.1 어필리에이트 제품

```
추천 제품에는 제휴 링크가 포함될 수 있어요.
추천은 분석 결과 기반이며, 구매 결정은 회원님의 선택이에요.
제품 사용 중 이상 반응이 있으면 즉시 사용을 중단해주세요.
```

### 4.2 성분 호환성

```
성분 호환성 정보는 일반적인 화학 특성에 기반해요.
개인의 피부 반응은 다를 수 있으며,
새 제품 사용 시 패치 테스트를 권장해요.
```

---

## 5. 캡슐 시스템 면책

### 5.1 Daily Capsule

```
오늘의 루틴은 회원님의 프로필 기반 추천이에요.
개인 상황에 따라 조정이 필요할 수 있어요.
```

### 5.2 캡슐 갭 분석

```
부족 아이템 분석은 일반적인 캡슐 구성 기준이에요.
반드시 모든 추천 제품을 구매할 필요는 없어요.
현재 보유 아이템으로도 충분할 수 있어요.
```

## 6. 면책 표시 규칙

### 6.1 위치

| 화면          | 면책 위치             |
| ------------- | --------------------- |
| 분석 결과     | 결과 카드 하단        |
| 제품 추천     | 추천 목록 상단        |
| Safety 경고   | 경고 메시지 바로 아래 |
| Daily Capsule | 루틴 카드 하단        |

### 6.2 스타일

```
색상: text-muted-foreground (회색)
크기: text-xs (12px)
아이콘: ℹ️ 정보 아이콘 (선택)
링크: "자세히 보기" → 전체 면책/개인정보처리방침
```

### 6.3 절대 금지 표현

| 금지 표현         | 대체 표현                                |
| ----------------- | ---------------------------------------- |
| "치료합니다"      | "관리에 도움이 될 수 있어요"             |
| "효과가 있습니다" | "도움이 될 수 있어요"                    |
| "의사가 추천하는" | (사용 금지)                              |
| "피부과 전용"     | (사용 금지)                              |
| "임상 검증"       | (사용 금지, 자체 검증 아님)              |
| "100% 안전"       | "일반적으로 안전한 것으로 알려져 있어요" |
