## 2. 의료기기 해당 여부 판단

### 2.1 의료기기 정의 (의료기기법 제2조)

```
"의료기기"란 사람이나 동물에게 단독 또는 조합하여 사용되는 기구·기계·장치·
재료·소프트웨어 또는 이와 유사한 제품으로서 다음 각 목의 어느 하나에 해당하는
제품을 말한다.

가. 질병을 진단·치료·경감·처치 또는 예방할 목적으로 사용되는 제품
나. 상해 또는 장애를 진단·치료·경감 또는 보정할 목적으로 사용되는 제품
다. 구조 또는 기능을 검사·대체 또는 변형할 목적으로 사용되는 제품
라. 임신을 조절할 목적으로 사용되는 제품
```

### 2.2 이룸 서비스별 판단

| 서비스 | 의료기기 해당? | 근거 |
|--------|---------------|------|
| **PC-1 퍼스널컬러** | ❌ 아님 | 미용 정보, 질병 무관 |
| **S-1 피부분석** | ⚠️ 경계 | "분석"만 허용, "진단" 금지 |
| **C-1 체형분석** | ❌ 아님 | 패션 정보, 치료 목적 아님 |
| **H-1 헤어분석** | ❌ 아님 | 미용 정보 |
| **M-1 메이크업** | ❌ 아님 | 미용 정보 |
| **N-1 영양분석** | ⚠️ 경계 | 일반 건강정보만 허용 |
| **W-1 운동추천** | ❌ 아님 | 피트니스 정보 |

## 3. 용어 사용 가이드

### 3.1 허용/금지 용어 목록

| 금지 용어 | 대체 용어 |
|-----------|----------|
| 진단 | 분석, 확인, 파악 |
| 치료 | 관리, 케어, 개선 |
| 처방 | 추천, 제안 |
| 증상 | 상태, 특징 |
| 질환 | 고민, 문제 |
| 효능 | 도움, 기대 효과 |
| 치료제 | 케어 제품, 화장품 |
| 약 | 제품, 보조제 (건기식) |

### 3.2 문장 변환 예시

```typescript
// 금지 문장 → 허용 문장 변환
const termMapping = {
  // 피부 관련
  "여드름 진단": "여드름 고민 분석",
  "건선 증상": "건조한 피부 상태",
  "피부염 치료": "피부 진정 케어",

  // 영양 관련
  "영양 결핍 진단": "영양 섭취 현황 분석",
  "비타민 처방": "비타민 보충 제안",
  "치료 식단": "건강 식단 추천",

  // 운동 관련
  "재활 치료 운동": "회복 도움 운동",
  "처방 운동": "맞춤 운동 추천",
};
```

### 3.3 AI 프롬프트 가이드라인

```typescript
// Gemini 프롬프트에 포함할 제약 조건
const medicalBoundaryPrompt = `
## 의료 경계 준수 (필수)

다음 규칙을 반드시 준수하세요:

1. **금지 용어 사용 금지**:
   - "진단", "치료", "처방", "증상", "질환" 사용 금지
   - "분석", "관리", "추천", "상태", "고민"으로 대체

2. **의학적 판단 금지**:
   - 특정 질병/질환 언급 금지
   - 의약품 추천 금지
   - 의료 시술 권유 금지

3. **면책 조항 포함**:
   - 모든 분석 결과에 "참고용 정보" 명시
   - 전문의 상담 권장 문구 포함

4. **심각한 상태 감지 시**:
   - 분석 결과 제공하지 않음
   - "전문의 상담을 권장합니다" 메시지만 표시
`;
```

## 4. 면책 조항

### 4.1 필수 면책 문구

```typescript
// 서비스별 면책 조항
const disclaimers = {
  skinAnalysis: `
    본 피부 분석 결과는 AI 기반의 참고 정보입니다.
    의학적 진단이나 처방을 대체하지 않습니다.
    피부 이상이 지속되면 피부과 전문의와 상담하세요.
  `,

  nutritionAnalysis: `
    본 영양 분석 결과는 일반적인 건강 정보입니다.
    개인의 건강 상태에 따라 적합하지 않을 수 있습니다.
    질환이 있거나 특별한 식이 요법이 필요한 경우
    의사 또는 영양사와 상담하세요.
  `,

  productRecommendation: `
    추천된 제품은 AI 분석 기반 참고 정보입니다.
    개인의 피부 반응은 다를 수 있으며,
    알레르기나 이상 반응 발생 시 사용을 중단하고
    전문가와 상담하세요.
  `,
};
```

### 4.2 UI 표시 위치

| 위치 | 면책 조항 유형 |

### 6.2 코드 구현

```typescript
// 금지 용어 필터
const FORBIDDEN_TERMS = [
  '진단', '치료', '처방', '증상', '질환',
  '의약품', '약물', '투약', '복용량',
];

function sanitizeMedicalTerms(text: string): string {
  let sanitized = text;

  const replacements: Record<string, string> = {
    '진단': '분석',
    '치료': '관리',
    '처방': '추천',
    '증상': '상태',
    '질환': '고민',
  };

  for (const [forbidden, allowed] of Object.entries(replacements)) {
    sanitized = sanitized.replace(new RegExp(forbidden, 'g'), allowed);
  }

  return sanitized;
}

// 심각 상태 감지
function detectSevereCondition(analysisResult: AnalysisResult): boolean {
  const severeIndicators = [
    'severe_inflammation',
    'potential_malignancy',
    'requires_medical_attention',
  ];

  return severeIndicators.some(
    indicator => analysisResult.flags?.includes(indicator)
  );
}
```
