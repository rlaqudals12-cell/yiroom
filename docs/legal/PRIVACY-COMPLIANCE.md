## 1. 개요

### 1.1 적용 법률

| 법률 | 약칭 | 관할 |
|------|------|------|
| **개인정보 보호법** | PIPA | 한국 |
| **정보통신망법** | - | 한국 |
| **Act on Protection of Personal Information** | APPI | 일본 |
| **California Consumer Privacy Act** | CCPA | 미국 (캘리포니아) |
| **General Data Protection Regulation** | GDPR | EU |

### 1.2 이룸에서 수집하는 개인정보

```
┌─────────────────────────────────────────────────────────────────┐
│                   수집 개인정보 분류                             │

│   ├── 생년월일 (연령 확인용)                                     │
```

## 2. 동의 획득 요건

### 2.1 필수 동의 항목

| 동의 유형 | 법적 근거 | 필수 여부 |
|-----------|----------|----------|
| **서비스 이용약관** | 민법, 전자상거래법 | 필수 |
| **개인정보 수집/이용** | 개인정보보호법 제15조 | 필수 |
| **민감정보 수집** | 개인정보보호법 제23조 | 별도 동의 필수 |
| **마케팅 수신** | 정보통신망법 제50조 | 선택 |
| **제3자 제공** | 개인정보보호법 제17조 | 별도 동의 필수 |

## 3. 이미지 데이터 처리

### 3.1 얼굴 이미지 처리 원칙

```
┌─────────────────────────────────────────────────────────────────┐
│                   얼굴 이미지 처리 원칙                          │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│   1. 목적 제한                                                   │
│   ├── 수집 목적: AI 분석 (퍼스널컬러, 피부)                      │
│   ├── 다른 목적 사용 금지                                        │
│   └── AI 학습용 사용 시 별도 동의 필요                           │
│                                                                  │
│   2. 최소 수집                                                   │
│   ├── 분석에 필요한 최소 해상도만 저장                           │
│   ├── 원본 이미지는 분석 후 즉시 삭제                            │
│   └── 메타데이터 (위치, 기기 정보) 제거                          │
│                                                                  │
│   3. 보관 기한                                                   │
│   ├── 분석 결과: 서비스 이용 기간                                │
│   ├── 원본 이미지: 분석 완료 후 24시간                           │
│   └── 회원 탈퇴 시: 30일 내 삭제                                 │
│                                                                  │
│   4. 접근 제한                                                   │
│   ├── 본인만 접근 가능 (RLS)                                     │
│   ├── 관리자도 원칙적 접근 불가                                  │
│   └── 법적 요청 시에만 예외                                      │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### 3.2 기술적 보호 조치

```typescript
// 이미지 처리 보안 설정
const imageSecurityConfig = {
  // 저장 전 처리
  preprocessing: {
    stripMetadata: true,           // EXIF 제거
    maxResolution: 1024,           // 최대 해상도 제한
    compression: 'lossy',          // 원본 복원 불가
  },

  // 저장소 설정
  storage: {
    bucket: 'private',             // 비공개 버킷
    encryption: 'AES-256',         // 암호화
    signedUrlExpiry: 3600,         // 1시간 만료
  },

  // 접근 제어
  access: {
    rls: true,                     // RLS 필수
    auditLog: true,                // 접근 로그
    ipRestriction: false,          // IP 제한 (선택)
  },

  // 삭제 정책
  retention: {
    originalImage: 86400,          // 24시간 (초)
    analysisResult: 'until_delete', // 탈퇴 시까지
    deletedUser: 2592000,          // 30일 (초)
  },
};
```

## 4. 개인정보처리방침 필수 항목

### 4.1 법적 필수 기재 사항

```markdown
1. 개인정보의 처리 목적
2. 처리하는 개인정보의 항목
3. 개인정보의 처리 및 보유 기간
4. 개인정보의 제3자 제공에 관한 사항
5. 개인정보처리의 위탁에 관한 사항
6. 정보주체의 권리·의무 및 행사방법
7. 개인정보의 파기에 관한 사항
8. 개인정보 보호책임자에 관한 사항
9. 개인정보 처리방침의 변경에 관한 사항
10. 개인정보의 안전성 확보조치에 관한 사항
```

### 4.2 이룸 특화 기재 사항

| 항목 | 내용 |
|------|------|
| **AI 분석** | 이미지가 AI 모델(Gemini)에 전송되어 분석됨 |
| **해외 전송** | Google Cloud (미국) 서버 처리 고지 |
| **자동화된 결정** | AI 분석 결과가 추천에 사용됨 |
| **이의 신청** | AI 분석 결과에 대한 이의 신청 방법 |

## 5. 정보주체 권리 보장

### 5.1 권리 목록

| 권리 | 설명 | 구현 |
|------|------|------|
| **열람권** | 본인 정보 조회 | 설정 > 내 정보 |
| **정정권** | 잘못된 정보 수정 | 설정 > 정보 수정 |
| **삭제권** | 개인정보 삭제 요청 | 설정 > 회원 탈퇴 |
| **처리정지권** | 처리 일시 정지 | 설정 > 서비스 중지 |
| **이동권** | 데이터 내보내기 | 설정 > 데이터 다운로드 |
| **동의철회권** | 동의 철회 | 설정 > 동의 관리 |

### 5.2 권리 행사 API

```typescript
// 개인정보 관련 API 엔드포인트
const privacyEndpoints = {
  // 내 정보 조회
  getMyData: 'GET /api/user/privacy/data',

  // 데이터 내보내기 요청
  exportData: 'POST /api/user/privacy/export',

  // 데이터 삭제 요청
  deleteData: 'POST /api/user/privacy/delete',

  // 동의 상태 조회
  getConsents: 'GET /api/user/privacy/consents',

  // 동의 철회
  revokeConsent: 'POST /api/user/privacy/consents/{id}/revoke',
};
```

## 6. 위반 시 제재

### 6.1 과태료/과징금

| 위반 유형 | 제재 |
|-----------|------|
| 동의 없는 수집 | 5천만원 이하 과태료 |
| 민감정보 부적절 처리 | 5천만원 이하 과태료 |
| 안전조치 미이행 | 5천만원 이하 과태료 |
| 유출 신고 미이행 | 3천만원 이하 과태료 |
| 개인정보 유출 | 매출 3% 이하 과징금 |

### 6.2 체크리스트

```markdown
## 런칭 전 체크리스트

### 동의 획득
□ 개인정보 수집 동의 UI 구현
□ 민감정보 별도 동의 UI 구현
□ 마케팅 동의 선택적 구현
□ 동의 이력 저장 (audit_logs)

### 처리방침
□ 개인정보처리방침 작성
□ 법적 필수 항목 포함 확인
□ 웹사이트/앱 내 게시
□ 버전 관리 체계 구축

### 기술 조치
□ RLS 정책 적용
□ 이미지 암호화 저장
□ 접근 로그 기록
□ 데이터 삭제 API 구현

### 조직 조치
□ 개인정보 보호책임자 지정
□ 정기 교육 계획

```
