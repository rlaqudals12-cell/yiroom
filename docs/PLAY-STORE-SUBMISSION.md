## 1. 실기기 스모크 테스트 (preview APK)

1. APK를 폰으로 전송(USB/링크) → 설치 (출처 불명 앱 허용)
2. 최소 검증: 로그인 → **홈 5축 프로필 카드** → 통합 분석 1회 → 결과 → 개별 축 카드 진입
3. 통과 기준: 크래시 없음 + 프로필 홈 정상 렌더 + 분석 저장/표시 정상

## 2. Play Console 앱 생성

1. [Play Console](https://play.google.com/console) → **앱 만들기**
   - 앱 이름: `이룸 (Yiroom)` / 기본 언어: 한국어 / 앱 · 무료
2. **테스트 → 내부 테스트** → 새 릴리스 → **AAB 업로드** (production 빌드 산출물)
   - "Google에서 서명 키 관리"(Play App Signing) 동의 → 업로드 키 = EAS 키스토어
3. 테스터 이메일 목록 등록 → 릴리스 시작 → 옵트인 링크로 설치 검증

## 4. 앱 콘텐츠 신고 (심사 필수, 콘솔 내 양식)

- **데이터 보안**: 수집=이메일·이름·사진(얼굴/신체), 용도=AI 분석, 암호화 전송/저장, 삭제 요청 가능(개인정보처리방침 §6과 일치하게 작성)
- **콘텐츠 등급 설문**: 뷰티/라이프스타일, 폭력·도박 없음 → 전체이용가 예상
- 광고 없음 / 타겟 연령층 설정 / 뉴스 앱 아님

## 5. 출시 순서 (권장)

```
내부 테스트 (즉시 배포, 심사 없음)
  → 비공개 테스트 (선택, 심사 있음)
  → 프로덕션 (심사 1~7일)
```

- 첫 프로덕션 심사는 수일 걸릴 수 있음. **내부 테스트 트랙까지가 "출시 직전" 상태.**
- 프로덕션 출시 클릭 전까지 스토어에 공개되지 않음.

## 6. 향후 업데이트 시 (EAS 구독 없을 때)
