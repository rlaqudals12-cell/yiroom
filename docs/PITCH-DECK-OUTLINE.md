## 슬라이드 2: 문제 정의

### 핵심 메시지
뷰티/웰니스 정보는 분산되어 있고, 개인화가 부족합니다.

### 내용

**문제 1: 분산된 정보**
- 퍼스널컬러 → A 앱
- 피부 분석 → B 앱
- 운동 관리 → C 앱
- 영양 관리 → D 앱
- 사용자는 4-5개 앱을 오가며 관리

**문제 2: 연결되지 않는 추천**
- 피부 분석 결과가 제품 추천에 반영 안 됨
- 체형 분석이 운동 추천과 연결 안 됨
- 각 분석이 사일로(Silo)로 존재

**문제 3: 개인화 부족**
- 일반적인 피부 타입 분류 (5가지)
- 개인의 복합적 조건 미반영
- 시간에 따른 변화 추적 불가

### 권장 시각 자료
- 분산된 앱 아이콘들 (흩어진 모습)
- 끊어진 화살표 다이어그램
- 사용자 Pain Point 인용구

### 발표자 노트
```
현재 뷰티/웰니스 시장의 문제는 세 가지입니다.

첫째, 정보가 분산되어 있습니다.
퍼스널컬러, 피부, 운동, 영양을 각각 다른 앱에서 관리해야 합니다.

둘째, 분석 결과가 서로 연결되지 않습니다.
피부 분석 결과가 제품 추천에 반영되지 않고, 체형 분석이 운동 추천과 연결되지 않습니다.

셋째, 개인화가 부족합니다.
단순한 피부 타입 분류를 넘어, 개인의 복합적 조건과 변화를 추적하지 못합니다.
```

## 슬라이드 3: 솔루션

### 핵심 메시지
이룸은 모든 분석을 연결하고, 변화를 기록하며, 맞춤 추천을 제공합니다.

### 내용

**통합 AI 분석 플랫폼**

| 분석 모듈 | 연결된 추천 |
|-----------|-------------|
| PC-1 퍼스널컬러 | 메이크업, 패션 컬러 |
| S-1 피부 분석 | 스킨케어 제품 |
| C-1 체형 분석 | 운동, 패션 스타일 |
| N-1 영양 분석 | 영양제, 건강식품 |
| W-1 운동 관리 | 맞춤 운동 플랜 |

**핵심 가치**
1. **통합**: 5개 분석을 하나의 앱에서
2. **연결**: 분석 결과가 추천에 자동 반영
3. **변화**: 시간에 따른 변화 추적 및 기록
4. **AI 코치**: 맞춤 상담 및 조언

### 권장 시각 자료
- 5개 분석 모듈 연결 다이어그램
- 분석 → 추천 흐름도
- 앱 스크린샷 (대시보드)

### 발표자 노트
```
이룸의 솔루션은 통합 AI 분석 플랫폼입니다.

퍼스널컬러, 피부, 체형, 영양, 운동을 하나의 앱에서 분석하고,
모든 분석 결과가 제품 추천에 자동으로 연결됩니다.

예를 들어, 봄 웜톤 + 복합성 피부 + 건조함이라면,
코랄 계열 메이크업과 세라마이드 함유 제품을 함께 추천합니다.

그리고 AI 웰니스 코치가 개인 맞춤 조언을 제공합니다.
```

## 슬라이드 4: 제품 데모 (1/2) - 분석 기능

### 핵심 메시지
AI 기반 정밀 분석으로 개인의 고유한 특성을 파악합니다.

### 내용

**퍼스널컬러 분석 (PC-1)**
- 얼굴 사진 업로드
- 12가지 시즌 분류 (봄/여름/가을/겨울 x 밝음/진한/뮤트)
- 베스트/워스트 컬러 팔레트 제공

**피부 분석 (S-1)**
- 6개 존 분석 (T존, U존, 볼, 턱, 이마, 코)
- 수분도, 유분도, 모공, 색소, 주름, 민감도
- 피부 타입 및 맞춤 성분 추천

**체형 분석 (C-1)**
- 전신 사진 분석
- 체형 분류 및 비율 분석
- 맞춤 운동 및 패션 스타일 추천

### 권장 시각 자료
- 분석 결과 화면 스크린샷 3개 (나란히 배치)
- 분석 과정 GIF 또는 동영상 (선택)
- 결과 카드 예시

### 발표자 노트
```
이것이 이룸의 분석 기능입니다.

퍼스널컬러 분석은 얼굴 사진 한 장으로 12가지 시즌 중
사용자의 톤을 정밀하게 분류합니다.

피부 분석은 T존, U존 등 6개 영역을 개별 분석하여
복합성 피부의 각 부위별 상태를 파악합니다.

체형 분석은 전신 비율을 분석하여
맞춤 운동과 패션 스타일을 추천합니다.
```

## 슬라이드 5: 제품 데모 (2/2) - 추천 및 코치

### 핵심 메시지
분석 결과가 제품 추천과 AI 코칭으로 자연스럽게 연결됩니다.

### 내용

**맞춤 제품 추천**
- 분석 결과 기반 제품 매칭
- 피부 타입 + 퍼스널컬러 복합 필터
- 다중 채널 구매 옵션 (쿠팡, iHerb, 무신사)

**AI 웰니스 코치**
- 자연어 질문 응답
- 분석 결과를 컨텍스트로 활용
- 개인 맞춤 조언 제공

**변화 추적**
- 피부 상태 변화 그래프
- 운동/영양 기록 및 트렌드
- 웰니스 스코어 대시보드

### 권장 시각 자료
- 제품 추천 화면 스크린샷
- AI 코치 채팅 화면 스크린샷
- 변화 추적 그래프 스크린샷

### 발표자 노트
```
분석 결과는 제품 추천으로 자연스럽게 연결됩니다.

"복합성 피부에 맞는 보습제 추천해줘"라고 물으면,
AI 코치가 사용자의 분석 결과를 참고하여 맞춤 답변을 제공합니다.

그리고 쿠팡, iHerb, 무신사 등 다양한 채널에서
사용자가 원하는 방식으로 구매할 수 있습니다.

변화 추적 기능으로 피부 상태, 운동, 영양의 개선을
시각적으로 확인할 수 있습니다.
```

## 슬라이드 8: 시장 규모

### 핵심 메시지
K-뷰티 글로벌 시장은 빠르게 성장 중, 개인화가 핵심 트렌드

### 내용

**시장 규모**

| 구분 | 규모 | 연 성장률 |
|------|------|----------|
| **TAM** (글로벌 뷰티테크) | $150B (2025) | 15% |
| **SAM** (K-뷰티 + 웰니스앱) | $8B | 12% |
| **SOM** (한국 20-30대 여성) | $500M | 10% |

**K-뷰티 시장 트렌드**
- 2024년 K-뷰티 수출 $10B 돌파
- 개인화/맞춤형 뷰티 수요 증가
- AI 뷰티 분석 앱 사용자 급증

**타겟 사용자**
- 1차: 한국 10대 후반~30대 초반 여성
- 2차: 글로벌 K-뷰티 관심층
- 3차: 남성 그루밍/웰니스 시장

### 권장 시각 자료
- TAM/SAM/SOM 동심원 다이어그램
- K-뷰티 수출 성장 그래프
- 타겟 사용자 페르소나

### 발표자 노트
```
글로벌 뷰티테크 시장은 2025년 1,500억 달러 규모로,
연 15% 성장하고 있습니다.

특히 K-뷰티 시장은 2024년 수출 100억 달러를 돌파했고,
개인화/맞춤형 뷰티에 대한 수요가 빠르게 증가하고 있습니다.

이룸의 초기 타겟은 한국 10대 후반~30대 초반 여성으로,
약 5억 달러 규모의 SOM을 목표로 합니다.

이후 글로벌 K-뷰티 관심층과 남성 시장으로 확장할 계획입니다.
```

## 슬라이드 9: 경쟁 우위

### 핵심 메시지
"분석은 시작일 뿐, 모든 것이 연결되는 유일한 플랫폼"

### 내용

**경쟁 환경**

| 경쟁사 | 유형 | 한계점 |
|--------|------|--------|
| 화해, 글로우픽 | 제품 리뷰 | 분석 기능 없음 |
| 룰루랩, 미러로이드 | 피부 분석 | 단일 기능, 제품 연결 약함 |
| 올리브영, 무신사 | 커머스 | 분석/기록 기능 없음 |
| 마이피트니스팔, 눔 | 건강 앱 | 뷰티 분석 없음 |

**이룸 차별점**

1. **통합 분석**: 5개 모듈 한 곳에서
2. **연결된 추천**: 분석 → 제품 자동 연결
3. **변화 추적**: 시간에 따른 기록
4. **AI 코치**: 맞춤 상담
5. **한국 시장 특화**: K-뷰티 트렌드 반영

**핵심 차별화**
```
"분석만 하는 앱" vs "분석이 연결되는 플랫폼"
```

### 권장 시각 자료
- 경쟁사 비교 매트릭스
- 차별점 아이콘 (5개)
- 포지셔닝 맵 (2x2)
