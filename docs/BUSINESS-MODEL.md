## 0. 궁극의 형태 (P1)

### 이상적 최종 상태

```
"글로벌 1위 AI 웰니스 플랫폼으로,
모든 사용자가 자기 이해를 통해 더 나은 선택을 하고,
그 과정에서 자연스럽게 수익이 창출되는 생태계"
```

### 물리적 한계

| 한계        | 설명                                  |
| ----------- | ------------------------------------- |
| 시장 규모   | 글로벌 뷰티테크 $65B (2028), 국내 $2B |
| 경쟁        | L'Oréal, Ulta 등 대기업 진입          |
| AI 기술     | VLM 정확도 한계 (현재 85-90%)         |
| 사용자 신뢰 | 개인 이미지 데이터 민감성             |

### 100점 기준

| 항목        | 기준       | 가중치 |
| ----------- | ---------- | ------ |
| MAU         | 1,000,000+ | 25%    |

### 수익화 Phase 전략

> **현재 Phase**: Phase 1A (수익 학습 기간)
> **참조**: [ADR-054 v2.0](./adr/ADR-054-affiliate-first-monetization.md)

| Phase               | 조건                               | 수익 믹스                                | ARPU         |
| ------------------- | ---------------------------------- | ---------------------------------------- | ------------ |
| **Phase 1A** (현재) | MAU 0 ~ 3,000                      | 어필리에이트 100%                        | ₩2-4         |
| **Phase 1B**        | MAU 3,000 ~ 14,000                 | 어필리에이트 + 1회 구매                  | ₩50-150      |
| **Phase 2**         | MRR ₩100만 가능 시점 (~MAU 14,000) | 어필리에이트 + 1회 구매 + 구독           | ₩500-1,500   |
| **Phase 3**         | MAU 30,000+                        | 어필리에이트 + 1회 구매 + 구독 + B2B API | ₩2,000-3,000 |

> **광고 수익**: 디스플레이 광고 영구 제외 (핵심 가치 보존). 브랜드 콘텐츠 파트너십은 MAU 50K+ 후 별도 ADR.
> **ARPU 산출 근거**: 쿠팡 3% 커미션 기반 보수적 계산. 상세 → ADR-054 v2.0

### ~~2.4 광고 수익~~ — 영구 삭제 (ADR-054 v2.0)

> **결정**: 디스플레이 광고, Promoted Products, CPC/CPM 네트워크 광고 **영구 제외**.

> **근거**:
>
> - 이룸의 핵심 가치("과학적 AI 분석 신뢰도")와 광고가 상충
> - 배민 울트라콜 사례: "돈 내면 상단 노출" → 사용자 불신 → 공정거래위 조사

>
> **향후 검토 가능**: "브랜드 콘텐츠 파트너십" (광고가 아닌 공동 콘텐츠) — MAU 50K+ 후 별도 ADR.
>
> 상세 → [ADR-054 v2.0](./adr/ADR-054-affiliate-first-monetization.md)
