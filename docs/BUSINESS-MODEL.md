#### 2.1.3 예상 수익 시뮬레이션

```typescript
// 어필리에이트 수익 계산 모델
interface AffiliateRevenueModel {
  mau: number;                    // 월간 활성 사용자
  recommendationCtr: number;      // 추천 클릭률 (5~15%)
  conversionRate: number;         // 구매 전환율 (2~5%)
  avgOrderValue: number;          // 평균 주문 금액
  avgCommissionRate: number;      // 평균 커미션율
}

// 1년차 예상 (MAU 50,000 기준)
const year1Projection = {
  mau: 50000,
  recommendationCtr: 0.08,        // 8%
  conversionRate: 0.03,           // 3%
  avgOrderValue: 45000,           // ₩45,000
  avgCommissionRate: 0.05,        // 5%
};

// 월 매출 = 50,000 × 0.08 × 0.03 × 45,000 × 0.05 = ₩27,000,000
// 연 매출 = ₩324,000,000
```

| 경쟁사 진입 | 높음 | 중간 | 빠른 기능 확장, 차별화 |
| 규제 변경 | 낮음 | 높음 | 법률 자문, 규정 선제 준수 |
| 파트너 이탈 | 낮음 | 중간 | 다중 파트너 전략 |
| 사용자 이탈 | 중간 | 높음 | 리텐션 강화, 커뮤니티 |
