## 상세 설명 (4,000자 이내)

```
거울 + 옷장을 한꺼번에 이해해주는 앱, 이룸

사진 한 장으로 전문가급 분석 — 오프라인 진단(10~30만원)을 무료로.

■ 당신의 시각 정체성 5축을 해석합니다
• 퍼스널 컬러: 12톤 시즌 분류 + 베스트/워스트 컬러 팔레트
• 피부 분석: 7존 정밀 분석 (수분, 유분, 민감도, 주름)
• 체형 분석: 스타일링 원칙 + 오늘 따라 입을 코디 3세트
• 헤어 분석: 얼굴형 + 모발 타입 기반 어울리는 헤어스타일
• 메이크업: 퍼스널컬러 + 피부 기반 상황별 추천 (데일리/오피스/데이트)

■ "나"에 맞는 쇼핑 결정의 기준
모든 분석은 "쇼핑할 때 꺼내 쓸 수 있는 실행 가능한 결과"로 돌아와요.
• 화장품: 색과학(CIE Lab) 기반 정확한 컬러 매칭
• 의류: 체형을 살리는 실루엣 원칙 + 코디 예시
• 헤어·제품: 얼굴형과 라이프스타일에 맞춘 추천

■ 체형은 바꾸는 게 아니라 이해하고 표현하는 것
"당신 체형은 X, 개선 필요" 대신
"당신 체형 X를 살리는 스타일은 Y"를 제안해요.

■ AI 코치
분석 결과를 종합해 맞춤형 뷰티 조언을 해줘요.
스킨케어 루틴, 메이크업 스타일, 헤어 관리까지 한곳에서.

■ 일일 루틴
오늘 해야 할 뷰티 루틴을 하나로 정리해줘요.
매일 체크하면서 나의 변화를 추적할 수 있어요.

■ 커뮤니티
다른 사용자들과 분석 결과를 공유하고, 팁을 나눠보세요.
익명 공유(실루엣/일러스트) 모드로 편안하게.

■ 내 옷장 (준비 중)
보유한 옷으로 가능한 코디를 AI가 찾아줄 예정이에요.
곧 베타로 만나보실 수 있어요.

—

왜 이룸인가요?

▶ 경쟁사 없는 통합 — 얼굴(퍼스널컬러+피부+메이크업+헤어)과 몸(체형)을 한 번에 이해해주는 앱은 이룸뿐.

▶ 전문가 90% 수준을 무료로 — 오프라인 퍼스널컬러 진단은 10~30만원. 이룸은 영원히 무료.

▶ 쇼핑 결정의 단일 근거 — 화장품 고를 때, 옷 살 때, 헤어 바꿀 때 — 하나의 "나 프로필"이 모든 기준이 돼요.

▶ 분석할수록 정확해져요 — 나만의 뷰티 타임라인을 쌓아가면 이룸이 나를 더 잘 이해하게 돼요.

개인정보처리방침: https://yiroom.vercel.app/privacy-policy
이용약관: https://yiroom.vercel.app/terms
```
