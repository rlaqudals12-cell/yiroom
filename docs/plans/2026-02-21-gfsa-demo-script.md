## 사전 준비

### 환경

```bash
# Vercel 배포 환경 사용 (또는 로컬)
# .env.local에 FORCE_MOCK_AI=true 확인
# 브라우저: Chrome (1920x1080, 100% 줌)
# 다크모드 OFF (라이트 모드)
```

### 테스트 계정

- Clerk 계정으로 로그인 (데모용 이메일)
- 이전 분석 결과 없는 **신규 계정** 권장 (깨끗한 대시보드)
- 또는 PC-1 + S-1 + C-1 결과가 있는 **기존 계정** (대시보드 풍성)

### 녹화 세팅

- 해상도: 1920x1080 (Full HD)
- 프레임: 30fps
- 마우스 하이라이트 ON (클릭 위치 시각화)
- 브라우저 탭/북마크 바 숨기기 (깔끔한 화면)

## 촬영 팁

### 화면 전환

- 페이지 이동은 **직접 URL 입력이 아닌 UI 클릭**으로 (자연스러운 플로우)
- 로딩 화면도 포함 (AI 분석 중임을 보여줌)
- Mock 모드에서 로딩은 1-2초 (적절한 대기감)

### 마우스 동작

- 천천히, 의도적으로 이동
- 호버 효과가 있는 요소에서 잠시 멈춤
- 스크롤은 부드럽게

### 자막/나레이션

- **영문 자막** (GFSA는 글로벌 프로그램)
- 나레이션은 영어 또는 자막으로 대체
- 핵심 수치는 텍스트 오버레이: "8 AI Modules", "2,776 Tests", "S-Grade Quality"

### 이미지 준비

- 퍼스널컬러용 셀피 1장 (자연광, 정면, 메이크업 최소)
- 피부 분석용 셀피 1장 (같은 사진 재사용 가능)
- 체형 분석용 전신 사진 1장 (시나리오에 포함 시)
