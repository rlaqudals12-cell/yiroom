## 3. 기술 스택

### 3.1 현재 인프라

| 영역             | 기술                      | 설명                          |
| ---------------- | ------------------------- | ----------------------------- |
| **프론트엔드**   | Next.js 16 + React 19     | App Router, Server Components |
| **호스팅**       | Vercel                    | Edge Functions, ISR           |
| **데이터베이스** | Supabase (PostgreSQL)     | RLS 정책, Edge Functions      |
| **인증**         | Clerk                     | OAuth, 세션 관리              |
| **모바일**       | Expo SDK 54               | React Native, NativeWind      |
| **AI 엔진**      | **Google Gemini 3 Flash** | 핵심 분석 엔진                |
| **언어**         | TypeScript                | Strict Mode, Zod 검증         |

### 3.2 Google 생태계 활용

| 서비스            | 현재 상태  | 활용 계획                |
| ----------------- | ---------- | ------------------------ |
| **Gemini AI**     | ✅ 사용 중 | 모든 AI 분석의 핵심 엔진 |
| **Cloud Vision**  | 📋 계획    | 이미지 전처리 고도화     |
| **BigQuery**      | 📋 계획    | 사용자 행동 분석         |
| **Vertex AI**     | 📋 계획    | 모델 파인튜닝            |
| **Cloud Storage** | 📋 계획    | 백업 스토리지            |

### 3.3 GCP 크레딧 활용 계획

```
우선순위 1: Gemini API 비용 → 월 $50-100 절감
우선순위 2: Cloud Vision → 이미지 품질 검증 고도화
우선순위 3: BigQuery → 데이터 분석 파이프라인
우선순위 4: Vertex AI → 맞춤형 모델 학습
```

## 4. 아키텍처

### 4.1 모노레포 구조

```
yiroom/
├── apps/web/          # Next.js 웹 앱
├── apps/mobile/       # Expo 모바일 앱
├── packages/shared/   # 공통 타입/유틸리티
└── docs/              # 설계 문서
```

### 4.2 AI 파이프라인

```
┌─────────────────────────────────────────────────────────────┐
│                    AI 분석 파이프라인                         │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  사용자 이미지 업로드                                          │
│         │                                                    │
│         ▼                                                    │
│  ┌─────────────┐                                            │
│  │ 이미지 검증  │ → 해상도, sharpness, 조명 체크              │
│  └──────┬──────┘                                            │
│         │                                                    │
│         ▼                                                    │
│  ┌─────────────┐                                            │
│  │ Gemini AI   │ → 피부/컬러/체형 분석 (VLM)                 │
│  └──────┬──────┘                                            │
│         │                                                    │
│         ▼                                                    │
│  ┌─────────────┐                                            │
│  │ 결과 저장   │ → Supabase (PostgreSQL + RLS)              │
│  └──────┬──────┘                                            │
│         │                                                    │
│         ▼                                                    │
│  맞춤 추천 (제품, 운동, 식단)                                  │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```
