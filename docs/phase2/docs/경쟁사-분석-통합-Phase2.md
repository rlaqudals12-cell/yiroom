### 1.3 경쟁사별 상세 분석

#### Nike Training Club (NTC)
```
강점:
  - 글로벌 브랜드 신뢰도
  - 고품질 운동 영상 콘텐츠
  - Nike 에코시스템 연동 (신발, 의류)

약점:
  - 개인화 부족 (체형/목표 미반영)
  - 한국 시장 특화 콘텐츠 부족
  - AI 추천 미약

이룸 대응:
  - C-1 체형 분석 기반 "진짜 맞춤" 추천
  - 한국 연예인 루틴으로 로컬화
```

#### 플랜핏
```
강점:
  - AI 맞춤 운동 추천
  - 호흡법 + 자세 가이드
  - 국내 시장 특화

약점:
  - 체형 분석 없음 (자가 입력만)
  - 크로스 모듈 연동 없음
  - 패션/뷰티 연결 없음

이룸 대응:
  - C-1 실제 이미지 분석 (정확도 ↑)
  - PC-1/S-1/N-1 크로스 모듈 인사이트
  - 호흡법 가이드 벤치마킹 적용
```

#### 짐워크
```
강점:
  - 1RM 기반 과학적 무게 추천
  - 연예인 루틴 콘텐츠
  - 헬스장 특화

약점:
  - 홈트레이닝 콘텐츠 부족
  - 영양/피부 연동 없음
  - AI 개인화 미약

이룸 대응:
  - 1RM 추정 로직 벤치마킹
  - 연예인 루틴 + 같은 PC/체형 필터
  - 홈트 + 헬스장 모두 지원
```

### 1.4 W-1 핵심 차별화 3가지

1. **체형(C-1) 기반 맞춤 운동**
   - 경쟁사: 목표/신체 고민만 고려
   - 이룸: 실제 체형 분석 데이터 기반 + "왜 이 운동인지" 설명

2. **퍼스널 컬러 연동 (유일)**
   - 경쟁사: 전무
   - 이룸: 운동복 + 운동 소품 + 분위기까지 PC 연동
   - 같은 체형+PC 연예인 루틴 제공

3. **통합 웰니스 플랫폼**
   - 경쟁사: 운동만 단독
   - 이룸: 운동 ↔ 피부 ↔ 체형 ↔ 영양 연결
   - 크로스 모듈 인사이트 + 통합 웰니스 리포트

### 2.3 경쟁사별 상세 분석

#### 눔 (Noom)
```
강점:
  - 행동 심리학 기반 코칭 (과학적 접근)
  - 음식 신호등 시스템 (직관적)
  - 높은 유저 리텐션

약점:
  - 유료 인력 코칭 → 비용 높음
  - AI 자동 분석 부족
  - 크로스 모듈 연동 없음

이룸 대응:
  - AI 기반 자동 분석 (비용 효율)
  - 음식 신호등 벤치마킹 적용
  - S-1/C-1/W-1 크로스 모듈 인사이트
```

#### MyFitnessPal
```
강점:
  - 1,400만+ 음식 DB (세계 최대)
  - 바코드 스캔 정확도 높음
  - 글로벌 커뮤니티

약점:
  - AI 음식 인식 없음 (수동 입력)
  - 한국 음식 DB 부족
  - 개인화 추천 미약

이룸 대응:
  - Gemini Vision AI 음식 인식
  - 한국 음식 500종 기본 DB
  - 선호도/스킬/예산 기반 개인화
```

#### 다노
```
강점:
  - 한국 시장 특화
  - 식품 판매 수익 모델
  - 습관 트래킹 게이미피케이션

약점:
  - AI 분석 부족
  - 커머스 중심 (분석 서비스 부족)
  - 타 모듈 연동 없음

이룸 대응:
  - 분석 중심 서비스 (커머스 선택적)
  - 크로스 모듈 시너지
  - 더 깊은 개인화
```

### 2.4 N-1 핵심 차별화 3가지

1. **"사진 한 장으로 끝" - AI 음식 인식**
   - 경쟁사: 수동 입력 or 바코드 스캔
   - 이룸: Gemini Vision으로 음식 사진 → 자동 영양 분석

2. **"내 몸 전체를 위한 식단" - 크로스 모듈 연동**
   - 경쟁사: 영양만 단독
   - 이룸:
     - S-1 연동: "피부에 좋은 음식" 추천
     - C-1 연동: 체형 목표에 맞는 칼로리
     - W-1 연동: 운동 후 단백질 보충 알림

3. **"나만의 맞춤 추천"**
   - 경쟁사: 일반적 권장량
   - 이룸: 선호도 + 요리 스킬 + 예산 + 알레르기 반영

## 3. 이룸 핵심 차별화 전략

### 3.1 통합 경쟁 우위

```
┌─────────────────────────────────────────────────────┐
│                   이룸 통합 웰니스                    │
├─────────────────────────────────────────────────────┤
│                                                     │
│   PC-1 ──────┬────────┬────────┬────────> 전체 연동 │
│   (컬러)     │        │        │                    │
│              ▼        ▼        ▼                    │
│           S-1      C-1      W-1      N-1            │
│          (피부)   (체형)   (운동)   (영양)           │
│              │        │        │        │           │
│              └────────┴────────┴────────┘           │
│                       ▼                             │
│              통합 웰니스 인사이트                    │
│                                                     │
│   경쟁사: 각 영역 단독 앱                            │
│   이룸: 하나의 앱에서 전체 웰니스 관리               │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 3.2 크로스 모듈 시너지 예시

| 시나리오 | 경쟁사 | 이룸 |
|---------|--------|------|
| 피부 트러블 발생 | 피부 앱에서 분석만 | S-1 분석 + N-1 피부 친화 음식 추천 |
| 체중 감량 목표 | 운동 or 식단 앱 개별 사용 | C-1 체형 분석 + W-1 맞춤 운동 + N-1 칼로리 적자 식단 |
| 운동복 고민 | 패션 앱 별도 검색 | PC-1 + C-1 기반 운동복 컬러/핏 추천 |
| 운동 후 식사 | 칼로리 앱에서 수동 입력 | W-1 운동 완료 → N-1 단백질 보충 알림 |

### 3.3 경쟁사 대응 리스크 및 대책

| 리스크 | 가능성 | 대책 |
|--------|--------|------|
| 플랜핏/짐워크 유사 기능 출시 | 중 | 크로스 모듈 연동 강화 (따라올 수 없음) |
| 눔/다노 AI 분석 강화 | 중 | 통합 플랫폼 + 빠른 기능 출시 |
| 대기업 진입 | 저 | 틈새 시장 (한국 10-30대 여성) 집중 |
