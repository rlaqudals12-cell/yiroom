## 6. 베타 테스트 시나리오

### 시나리오 1: 신규 사용자 온보딩
```
1. /workout/onboarding/step1 접속
2. 7단계 입력 완료
3. 결과 화면에서 운동 타입 확인
4. 추천 운동 리스트 확인
5. 주간 플랜 확인
```

### 시나리오 2: 운동 세션 완료
```
1. 주간 플랜에서 오늘 운동 시작
2. 각 운동 세트 완료 체크
3. 휴식 타이머 사용
4. 운동 완료 화면 확인
5. History에서 기록 확인
```

### 시나리오 3: Streak 유지
```
1. 연속으로 3일 운동 완료
2. 3일 마일스톤 달성 확인
3. 배지 획득 확인
4. 다음 마일스톤 (7일) 진행도 확인
```

### 시나리오 4: 크로스 모듈 연동
```
1. 결과 화면에서 스타일 추천 확인 (PC-1)
2. 피부 관리 팁 확인 (S-1)
3. 영양 가이드 확인 (N-1 준비)
4. 쇼핑 링크 클릭 → 외부 사이트 이동
```
