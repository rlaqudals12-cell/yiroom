## 1. 브랜드 전략

### 브랜드 정보
- **이름**: 이룸 (Yiroom)
- **슬로건**: "온전한 나를 찾는 여정" / "Know yourself, wholly"
- **타겟**: 10대 후반~30대 초반 여성
- **시장**: 한국 먼저 → 글로벌 확장

### 포지셔닝
```
"분석은 시작일 뿐"
퍼스널컬러 → 피부 → 체형 → 운동 → 영양
모든 분석이 연결되고, 변화가 기록되고, 성장이 보이는 곳
```

### 경쟁사 차별점
- 올리브영/무신사: 쇼핑 플랫폼
- 이룸: 분석 + 기록 + 연결된 추천 플랫폼
- 장기 목표: 인프라 구축 (무신사/올리브영처럼)

## 4. 수익 모델

### 단계별 전략
```
Phase 1: 사용자 확보 (무료 가치 제공)
├── 모든 기능 무료
├── 재분석 넉넉하게 제공
└── 광고 최소화

Phase 2: 최소 비용 충당
├── 쿠팡 파트너스 (1월 가입 예정)
├── 올리브영 제휴 (추후)
└── 제품 링크 수수료

Phase 3: 광고 (MAU 1만+ 후)
├── 네이티브 광고
├── 브랜드 협찬
└── 배너 (최소화)

Phase 4: 프리미엄 (필요시)
└── 광고 제거 + α (사용자 요청 있을 때)
```

### 구독 서비스
- 초기: ❌ 안 함
- 이유: 플랫폼 비즈니스는 사용자 확보가 우선
- 검토 시점: 사용자 10만+ 후
