# Design Refinement Spec (Refined Expertise v2)

> 이룸 웹 UI/UX 개선 스펙 문서
> 작성일: 2026-01-12

## 1. 개요

### 1.1 목적

기능 변경 없이 UI/UX만 개선하여 "정제된 전문성(Refined Expertise)" 느낌 강화

### 1.2 핵심 원칙

| 원칙            | 설명                                  |
| --------------- | ------------------------------------- |
| **기능 무변경** | 모든 기존 기능 100% 유지              |
| **부담감 금지** | 사용자 압박 요소 추가 금지            |
| **전문가 앱**   | 신뢰성 있는 스타일/뷰티 전문가 느낌   |
| **점진적 적용** | CSS 변수만 수정, 컴포넌트 로직 무변경 |

### 1.3 참고 브랜드

- Aesop (따뜻한 뉴트럴 + 블랙 타이포)
- Glossier (친근한 전문성)
- 탬버린즈 (한국적 세련미)

## 2. 현재 상태 분석

### 2.1 현재 색상 시스템 (OKLCH)

```css
/* 배경 */
--background: oklch(0.98 0.005 270); /* #f8f9fc - 쿨톤 화이트 */

/* 프라이머리 */
--primary: oklch(0.53 0.23 262); /* #2e5afa - 이룸 블루 */

/* 모듈별 색상 */
--module-workout: oklch(0.85 0.15 45); /* Orange */
--module-nutrition: oklch(0.75 0.15 150); /* Green */
--module-skin: oklch(0.8 0.12 350); /* Pink */
--module-body: oklch(0.75 0.15 250); /* Blue */
--module-personal-color: oklch(0.7 0.18 300); /* Purple */
```

### 2.2 사용 현황

| 색상 유형    | 사용 파일 수 | 사용 횟수 |
| ------------ | ------------ | --------- |
| Primary 색상 | 205개        | 493회     |
| 모듈 색상    | 20개         | 136회     |
| 그라디언트   | 5개          | 10회      |

### 2.3 현재 문제점

1. **쿨톤 배경(270도)**: 약간 차가운 느낌
2. **높은 채도 모듈 색상**: 통일감 부족
3. **블루 프라이머리**: 로고(민트→블랙)와 불일치

## 3. 변경 사항

### 3.1 배경색 변경

| 항목 | 현재                    | 변경                   | 효과          |
| ---- | ----------------------- | ---------------------- | ------------- |
| 배경 | `oklch(0.98 0.005 270)` | `oklch(0.99 0.005 90)` | 웜톤으로 전환 |
| 카드 | `oklch(0.98 0.005 270)` | `oklch(1.0 0 0)`       | 순백 카드     |

**Hue 변경**: 270 (쿨블루) → 90 (웜옐로우)

### 3.2 프라이머리 색상 변경

| 항목    | 현재                        | 변경                        | 근거                           |
| ------- | --------------------------- | --------------------------- | ------------------------------ |
| Primary | `oklch(0.53 0.23 262)` 블루 | `oklch(0.45 0.12 155)` 민트 | 로고 색상(#A8D5C2)과 동일 계열 |
| Ring    | `oklch(0.53 0.23 262)`      | `oklch(0.45 0.12 155)`      | 일관성                         |

**로고와의 관계**: 로고 민트(#A8D5C2, 155°)의 어두운 버전으로 브랜드 일관성 확보

### 3.3 모듈 색상 채도 조정 (뮤트화)

| 모듈           | 현재 채도 | 변경 채도 | 변화 |
| -------------- | --------- | --------- | ---- |
| Workout        | 0.15      | 0.12      | -20% |
| Nutrition      | 0.15      | 0.12      | -20% |
| Skin           | 0.12      | 0.10      | -17% |
| Body           | 0.15      | 0.12      | -20% |
| Personal-color | 0.18      | 0.14      | -22% |

### 3.4 그라디언트 업데이트

```css
/* 현재 */
--gradient-brand: linear-gradient(135deg, oklch(0.6 0.18 240), oklch(0.53 0.23 262));

/* 변경: 로고 민트(155°) 기반 그라디언트 */
--gradient-primary: linear-gradient(135deg, oklch(0.5 0.12 155), oklch(0.4 0.14 155));
--gradient-brand: linear-gradient(135deg, oklch(0.5 0.12 155), oklch(0.38 0.14 160));
```

### 3.5 보더/디바이더 톤 조정

```css
/* 현재: 쿨톤 보더 */
--border: oklch(0.87 0.03 270);

/* 변경: 웜톤 보더 */
--border: oklch(0.9 0.02 90);
```

## 4. 영향도 분석

### 4.1 영향받는 영역

| 영역      | 영향도       | 위험도  | 비고                   |
| --------- | ------------ | ------- | ---------------------- |
| 배경색    | 전역         | ⚪ 낮음 | CSS 변수만 변경        |
| 버튼/CTA  | 205개 파일   | ⚪ 낮음 | Primary 색상 자동 적용 |
| 모듈 카드 | 20개 파일    | ⚪ 낮음 | 채도만 조정            |
| 차트      | 5개 컴포넌트 | 🟡 중간 | chart-1 색상 확인 필요 |
| 다크모드  | 전역         | 🟡 중간 | 별도 색상 조정 필요    |

### 4.2 영향받지 않는 영역

| 영역          | 이유            |
| ------------- | --------------- |
| 레이아웃 구조 | CSS 변수만 변경 |
| 컴포넌트 로직 | TSX 파일 무변경 |
| API 호출      | 스타일만 변경   |
| 테스트        | 기능 무변경     |
| 라우팅        | 변경 없음       |

## 7. 구현 계획

### 7.1 변경 파일

```
apps/web/app/globals.css  (유일한 수정 파일)
```

### 7.2 변경 섹션

1. `:root` 라이트모드 색상 변수

변경 전 현재 `globals.css` 백업:

- 문제 발생 시 `git checkout` 으로 즉시 롤백

## 8. 검증 절차

### 8.1 자동 검증

```bash
npm run typecheck  # 타입 에러 없음 확인
npm run lint       # 린트 에러 없음 확인
npm run test       # 기존 테스트 통과 확인
```

### 8.2 수동 검증

1. 개발 서버 실행 (`npm run dev:web`)
2. 주요 페이지 순회 확인
   - 홈페이지
   - 대시보드
   - 분석 결과 페이지들
   - 제품 목록/상세
   - 프로필

### 8.3 시각적 검증

- 라이트모드 전체 확인
- 다크모드 전체 확인
- 모바일 뷰포트 확인

### 9.1 변경 요약

| ---------- | ---------------- |
| 배경       | 쿨톤 → 웜톤      |
