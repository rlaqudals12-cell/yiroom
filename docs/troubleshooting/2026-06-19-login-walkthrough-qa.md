# 로그인 워크스루 QA 체크리스트 (2026-06-19)

> 이번 세션 변경 중 **Clerk 보호 라우트**(curl로는 404 by design)는 코드/테스트로만 검증됨.
> 아래는 `yiroom.vercel.app` 로그인 후 **눈으로** 최종 확인할 항목. 각 항목 1~2분.

## 0. 사전

- `yiroom.vercel.app` 로그인 (Google 등). FORCE_MOCK_AI=false(실제 Gemini).
- ⚠️ **gap-apply-4 먼저 실행**: `docs/troubleshooting/2026-06-19-gap-apply-4-measurement-source.sql`을 Supabase SQL Editor에서 1회 Run (안 해도 깨지진 않음 — "측정 방식" 배지만 안 뜸).

## 3. 이상 시

- 체형이 "기타"/영문으로 보이면 → `getBodyShapeLabel` 미적용 경로 (보고).
- "측정 방식" 배지가 안 뜨면 → gap-apply-4 미적용(정상, 적용하면 노출) 또는 자가입력 모드.
- 통합 분석 저장 실패(체형) → measurement_source best-effort update라 본 저장엔 영향 없음. 로그 확인.
