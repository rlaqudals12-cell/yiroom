> **Version**: 1.0 | **Created**: 2026-01-21

---

## 공통 컨텍스트 (모든 요청에 포함)

```markdown
## 브랜드 컨텍스트

### 앱 정보
- 앱명: Yiroom (이룸)
- 헤더: "YIROOM IDENTITY ✨"
- 서브타이틀: "YIROOM INTELLIGENCE"
- 슬로건: "Absolute Singularity - 당신만의 아름다움을 발견하세요"
- 톤앤매너: 프리미엄, 신뢰감, 따뜻함, 모던

### 핵심 가치
- 통합 웰니스 AI 플랫폼
- 퍼스널컬러 + 피부 + 체형 + 영양 + 운동 분석
- 10만+ 사용자 신뢰

### 디자인 토큰 (필수 준수)

**컬러 시스템 (Dark Mode 기본)**
- Background: #0F0F0F
- Card: #1A1A1A
- Text Primary: #FFFFFF
- Text Secondary: #9CA3AF
- Primary Gradient: #F8C8DC → #FFB6C1 (핑크)
- Accent Blue: #60A5FA (피부)
- Accent Purple: #A78BFA (체형)
- Accent Pink: #F472B6 (퍼스널컬러)
- Accent Green: #4ADE80 (웰니스)

**필수 UI 패턴**
1. Progress Bar: "1/3 완료 (33%)" + 단계 아이콘 (⬡ ◯ ◯)
2. Trust Badge: 상단 우측 "10만+ 사용자 신뢰" | "AI 정확도 92%"
3. Sub-card: 컬러 아이콘 + 한글 설명 ("6존 AI 스캐닝")
4. CTA Button: 핑크 그라디언트, 둥근 모서리

**기술 제약**
- 모바일 퍼스트 (375px 기준)
- 터치 타겟 최소 44px
- shadcn/ui 호환 구조
- Tailwind CSS 클래스 사용
```

```markdown
# 요청: Yiroom Core Flow Part 1 - 온보딩 & 인증 UI 디자인

## 브랜드 컨텍스트
[공통 컨텍스트 삽입]

## 요청 화면 (10개)

### 1. Splash Screen
- 로고 "YIROOM" 중앙 배치
- 핑크 그라디언트 로고 또는 화이트 로고
- 배경: #0F0F0F
- 로딩 인디케이터 (subtle)

### 2. Onboarding 1
- "당신만의 아름다움을 발견하세요"
- AI 분석 아이콘 (퍼스널컬러/피부/체형)
- 페이지 인디케이터 (● ○ ○)
- "다음" 버튼

### 3. Onboarding 2
- "AI가 분석하고, 개인화된 추천을 드려요"
- 분석 예시 이미지 또는 일러스트
- 페이지 인디케이터 (○ ● ○)

### 4. Onboarding 3
- "10만+ 사용자가 이미 경험했어요"
- Trust badges + 사용자 후기
- 페이지 인디케이터 (○ ○ ●)
- "시작하기" CTA 버튼 (핑크 그라디언트)

### 5. Login Screen
- "YIROOM IDENTITY ✨" 헤더
- 소셜 로그인 버튼 (Google, Apple, Kakao)
- "이메일로 로그인" 옵션
- 하단 "계정이 없으신가요? 가입하기"

### 6. Sign Up Screen
- "YIROOM IDENTITY ✨" 헤더
- 이름, 이메일, 비밀번호 입력 필드
- 약관 동의 체크박스
- "가입하기" CTA 버튼
- 하단 "이미 계정이 있으신가요? 로그인"

### 7. Dashboard (Home) ⭐ 핵심 화면
- 헤더: "YIROOM IDENTITY ✨"
- Trust Badge: "10만+ 사용자 신뢰"
- Progress: "1/3 완료 (33%)" + 단계 아이콘
- Sub-cards:
  - 🔵 피부 분석 "6존 AI 스캐닝"
  - 🟣 체형 분석 "5-TYPE 체형 분석"
  - 💗 퍼스널컬러 "4계절 16타입"
- CTA: "무료로 시작하기" (핑크 그라디언트)
- 하단 네비게이션: 홈/분석/옷장/웰니스/마이

### 8. Analysis Hub
- "분석 허브" 타이틀
- 4개 분석 카드 그리드:
  - PC-1 퍼스널컬러 (💗 #F472B6)
  - S-1 피부 (🔵 #60A5FA)
  - C-1 체형 (🟣 #A78BFA)
  - IC-1 통합 (coming soon)
- 각 카드: 아이콘 + 이름 + 상태 (완료/미완료)

### 9. PC-1 Camera (퍼스널컬러 촬영)
- 카메라 프리뷰 영역
- 얼굴 가이드 오버레이 (타원형)
- "자연광에서 촬영해주세요" 안내 문구
- 촬영 버튼 (원형, 핑크)
- "갤러리에서 선택" 옵션

### 10. PC-1 Processing (분석 중)
- 로딩 애니메이션 (핑크 계열)
- "AI가 퍼스널컬러를 분석 중입니다..."
- Progress indicator (0% → 100%)
- 예상 소요 시간 표시

## 출력 형식

각 화면별:
1. **Layout**: 375px 너비 모바일 목업
2. **Components**: 사용된 컴포넌트 리스트
3. **States**: 기본/로딩/에러 상태
4. **Interactions**: 터치/스와이프 동작 설명

## 참조
[확정된 Dashboard 스크린샷 첨부 필요]
```

```markdown
- "정면을 바라봐주세요" 안내
- 촬영 버튼 (파란색 강조)
- 조명 품질 인디케이터

### 13. S-1 Processing
- 로딩 애니메이션 (파란색)
- "6존 피부 상태를 분석 중입니다..."
- 단계별 진행: "T존 분석 중..."

- FaceZoneMap: 6존 터치 가능한 얼굴 맵
- 피부 활력도 점수: "72점" 게이지
- 피부 타입: "복합성 (T존 지성/U존 건성)"
- 존별 점수 카드
- 스킨케어 솔루션 탭

### 15. S-1 Zone Detail (존 상세)
- 슬라이드업 모달
- 선택된 존 하이라이트
- 상세 점수 (수분/유분/민감도/주름)
- 해당 존 추천 제품

### 16. C-1 Camera (체형 촬영)
- 전신 가이드 오버레이
- "2m 거리에서 촬영해주세요"
- 정면/측면 가이드
- 촬영 버튼 (보라색 강조)

### 17. C-1 Processing
- 로딩 애니메이션 (보라색)
- "체형 비율을 분석 중입니다..."

### 18. C-1 Result (체형 분석 결과)
- 체형 타입: "웨이브 타입 💜"
- 체형 일러스트/다이어그램
- 특징 설명 (어깨/허리/힙 비율)
- 추천 패션 스타일
- "스타일링 보기" 버튼

### 19. IC-1 Integrated (통합 결과)
- "나의 종합 분석"
- 진행률: 원형 차트 "3/4 완료"
- 각 모듈 결과 요약 카드
- 크로스 인사이트 섹션
- "통합 리포트 다운로드" 버튼

### 20. Share Result (결과 공유)
- 공유 카드 프리뷰 (Instagram 사이즈)
- 공유 옵션: 카카오톡/인스타/링크 복사
- "이미지로 저장" 버튼
- 워터마크 옵션
```
