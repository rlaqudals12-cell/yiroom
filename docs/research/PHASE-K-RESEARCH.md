## 2. K-3: 아시아인 BMI 기준 및 자세 교정

### 2.1 한국 비만 기준 (대한비만학회, KSSO)

| 분류                 | BMI 범위    | 비고                       |
| -------------------- | ----------- | -------------------------- |
| 저체중               | < 18.5      | WHO 동일                   |
| 정상                 | 18.5 - 22.9 | WHO: 18.5-24.9             |
| 과체중 (비만 전단계) | 23.0 - 24.9 | **아시아 기준** (WHO: 25+) |
| 1단계 비만           | 25.0 - 29.9 | WHO: 과체중                |
| 2단계 비만           | 30.0 - 34.9 | WHO: 1단계 비만            |
| 3단계 비만 (고도)    | ≥ 35        | WHO: 2단계 비만            |

#### 복부비만 기준 (허리둘레)

| 성별 | 한국 기준 | WHO 아시아-태평양 |
| ---- | --------- | ----------------- |
| 남성 | ≥ 90cm    | ≥ 90cm            |
| 여성 | ≥ 85cm    | ≥ 80cm            |

> **중요**: 아시아인은 같은 BMI에서 더 많은 내장지방을 저장하므로, 22-25 사이에서 당뇨/심혈관 위험 증가

**Sources**:

- [대한비만학회](https://general.kosso.or.kr/html/?pmode=obesityDiagnosis)
- [서울대학교병원 - 고도비만](https://www.snuh.org/health/nMedInfo/nView.do?category=DIS&medid=AA001254)
- [KSSO 2024 Obesity Fact Sheet](https://general.kosso.or.kr/html/user/core/view/reaction/main/kosso/inc/data/2024_Obesity_Fact_sheet_web_eng.pdf)

### 2.2 BMI 계산 로직 수정

```typescript
// lib/body/bmi-calculator.ts (아시아 기준 적용)
export function calculateBMI(height: number, weight: number): BMIResult {
  const heightM = height / 100;
  const bmi = weight / (heightM * heightM);

  // 아시아인 기준 (대한비만학회)
  let category: BMIResult['category'];
  if (bmi < 18.5) category = 'underweight';
  else if (bmi < 23)
    category = 'normal'; // WHO는 25
  else if (bmi < 25)
    category = 'overweight'; // 비만 전단계
  else if (bmi < 30)
    category = 'obese1'; // 1단계 비만
  else if (bmi < 35)
    category = 'obese2'; // 2단계 비만
  else category = 'obese3'; // 3단계 비만 (고도)

  return {
    value: Math.round(bmi * 10) / 10,
    category,
    healthyRange: {
      min: Math.round(18.5 * heightM * heightM),
      max: Math.round(22.9 * heightM * heightM), // 아시아 기준
    },
    disclaimer:
      '아시아인 기준 적용 (대한비만학회). 의료 조언이 아니며, 정확한 진단은 전문의와 상담하세요.',
  };
}
```

### 2.3 체형별 자세 교정 운동 상세

#### 골반 전방경사 (Anterior Pelvic Tilt)

| 원인 근육        | 상태 | 교정 방향 |
| ---------------- | ---- | --------- |
| 대둔근, 햄스트링 | 약화 | 강화 운동 |
| 복근             | 약화 | 강화 운동 |
| 대퇴사두근       | 단축 | 스트레칭  |
| 장요근           | 단축 | 스트레칭  |

**교정 운동 프로그램** (연구 기반: 주 2회, 40분, 4주):

1. **다리 앞쪽 스트레칭**: 1-3분 유지, 양쪽 3-5회
2. **브릿지 + 한 다리 뻗기**: 엉덩이 들고 한 다리 일직선으로
3. **골반 중립 자세 유지**: 앉을 때, 서 있을 때, 걸을 때

#### 거북목 (Forward Head Posture)

| 문제 근육  | 상태      | 교정 방향             |
| ---------- | --------- | --------------------- |
| 흉쇄유돌근 | 단축/뭉침 | 이완                  |
| 사각근     | 단축/뭉침 | 이완                  |
| 후두하근   | 긴장      | 이완 (뇌가 위치 인식) |

**교정 운동 프로그램**:

1. **후두하근 마사지**: 마사지볼로 뒤통수 아래 5분 압박
2. **턱 당기기 (Chin Tuck)**: 턱을 뒤로 당기며 머리 바로 세우기
3. **벽 운동**: 뒤통수-어깨-골반 벽에 밀착, 팔 날갯짓 10회 (하루 2번, 1달)

#### 라운드 숄더 (Rounded Shoulders)

**교정 운동 프로그램**:

1. **가슴 스트레칭**: 어깨 앞쪽 늘어나는 느낌까지
2. **끈 운동**: 양손으로 끈 잡고 머리 위→뒤로 팔 뻗기
3. **Y-T-W 운동**: 엎드려서 팔을 Y, T, W 형태로 들기
