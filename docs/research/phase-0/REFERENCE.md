if (isMinor(birthDate)) {

## 🤖 Claude 4.x 프롬프트 패턴

### 핵심 특성
Claude 4.x는 지시사항을 **문자 그대로** 해석함
- 의도 추론 X → 명시된 것만 수행
- 모호하면 질문하거나 최소 해석

### 명시적 지시 필요
```
❌ "좋은 코드 작성해줘"
✅ "TypeScript strict, 에러 핸들링 포함, 한국어 주석"

❌ "개선해줘"
✅ "성능 최적화 위해 불필요한 리렌더링 제거, React.memo 적용"
```

### 긍정형 표현 (더 효과적)
```
❌ "신조어 쓰지 마"
✅ "일반적인 한국어로 작성 (예: GMG → 목표 달성)"

❌ "길게 쓰지 마"
✅ "3문장 이내로 요약해"
```

### XML 태그 구조
```xml
<role>역할 정의</role>
<context>프로젝트 현황</context>
<guidelines>작업 원칙</guidelines>
<constraints>제약사항</constraints>
<output_format>응답 형식</output_format>
<examples>예시</examples>
```

### 예시의 힘
Claude 4.x는 예시의 **세부 패턴까지 학습**
- 코드 스타일, 응답 길이, 포맷팅, 어조 모두 반영
- 의도하지 않은 패턴 없는지 확인 필수

### 트러블슈팅 체크리스트
```
□ 지시가 충분히 구체적인가?
□ 컨텍스트가 신선한가? (오래된 대화 아닌가?)
□ 예시가 원하는 행동과 일치하는가?
□ 출력 형식이 명시되어 있는가?
□ "~하지 마" 대신 "~해라"로 표현했는가?
```
