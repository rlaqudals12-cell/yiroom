# 이룸(Yiroom) AI 분석 결과 바이럴 공유 메커니즘 종합 전략

이룸 플랫폼의 바이럴 성장을 위해서는 **결과 카드의 시각적 임팩트**, **카카오톡/인스타그램 네이티브 공유 통합**, 그리고 **양면 보상 기반 친구 초대 시스템**이 핵심입니다. K-factor **0.5~0.7** 달성을 목표로, 분석 완료 직후 공유 트리거와 프라이버시를 보호하는 아바타 기반 결과 카드를 통해 사용자의 공유 심리적 장벽을 낮춰야 합니다. 한국 시장에서는 카카오톡이 70% 이상의 공유 트래픽을 차지하므로 Kakao Share API 최적화가 최우선이며, 토스와 마켓컬리 사례처럼 **추천인 3,000원 + 피추천인 5,000원** 구조의 레퍼럴 프로그램이 검증된 효과를 보입니다.

## 공유 가능한 결과 카드 설계 전략

AI 분석 결과를 바이럴 가능한 콘텐츠로 만들기 위해서는 **핵심 정보만 압축한 시각적 카드**가 필수입니다. 컬러그램, YouCam Makeup, Gradient 등 성공한 뷰티 앱들의 공통점은 결과를 **한 장의 공유 이미지**로 즉시 생성하고, SNS에 최적화된 포맷을 제공한다는 점입니다.

### 카카오톡 Share API 구현 (JavaScript)

카카오톡은 한국 시장에서 가장 높은 공유 전환율을 보이므로 **Feed 템플릿**을 활용한 구현이 핵심입니다.

```javascript
// Kakao SDK 초기화
Kakao.init('YOUR_JAVASCRIPT_KEY');

// Feed 템플릿 공유 함수
function shareToKakao(analysisResult) {
  Kakao.Share.sendDefault({
    objectType: 'feed',
    content: {
      title: `나의 퍼스널컬러는 ${analysisResult.seasonType}!`,
      description: '이룸에서 AI 퍼스널컬러 진단 받아보세요',
      imageUrl: `https://api.yiroom.io/og-image/${analysisResult.id}`,
      link: {
        mobileWebUrl: `https://yiroom.io/result/${analysisResult.id}`,
        webUrl: `https://yiroom.io/result/${analysisResult.id}`,
      },
    },
    social: {
      likeCount: analysisResult.likeCount,
      sharedCount: analysisResult.shareCount,
    },
    buttons: [
      {
        title: '결과 보기',
        link: {
          mobileWebUrl: `https://yiroom.io/result/${analysisResult.id}`,
          webUrl: `https://yiroom.io/result/${analysisResult.id}`,
        },
      },
      {
        title: '나도 진단받기',
        link: {
          mobileWebUrl: 'https://yiroom.io/diagnose',
          webUrl: 'https://yiroom.io/diagnose',
        },
      },
    ],
  });
}
```

### 인스타그램 스토리 공유 (iOS Swift)

```swift
func shareToInstagramStories(resultCard: UIImage) {
    guard let stickerData = resultCard.pngData() else { return }
    
    let urlScheme = URL(string: "instagram-stories://share?source_application=\(Bundle.main.bundleIdentifier ?? "")")
    
    if let url = urlScheme, UIApplication.shared.canOpenURL(url) {
        let pasteboardItems: [[String: Any]] = [[
            "com.instagram.sharedSticker.stickerImage": stickerData,
            "com.instagram.sharedSticker.backgroundTopColor": "#FFE4E1",
            "com.instagram.sharedSticker.backgroundBottomColor": "#E6E6FA"
        ]]
        
        UIPasteboard.general.setItems(pasteboardItems, 
            options: [.expirationDate: Date().addingTimeInterval(300)])
        UIApplication.shared.open(url)
    }
}
```

## K-factor 바이럴 계수 최적화 전략

### K-factor 계산 공식과 목표

**K = i × c** (초대 수 × 전환율)

| 단계 | 목표 K-factor | 달성 기간 | 전략 |
|-----|--------------|----------|------|
| **런칭** | 0.3 | 1~3개월 | 기본 공유 기능 |
| **성장** | 0.5 | 3~6개월 | 인센티브 프로그램 |
| **최적화** | 0.7 | 6~12개월 | A/B 테스트 기반 개선 |
| **바이럴** | 1.0+ | 12개월+ | 네이티브 바이럴 기능 |

뷰티/웰니스 앱의 평균 K-factor는 **0.3~0.5** 수준이며, Dropbox는 K=0.7로 15개월 만에 10만→400만 사용자를 달성했습니다. **K < 1이어도 유의미한 성장이 가능**하며, K=0.2만 달성해도 5명당 1명을 무료 획득하는 효과가 있습니다.

```
        ↓

```

### 바이럴 사이클 타임 단축 전략

사이클 타임(가입→첫 공유)은 **3일 이하**가 목표입니다. 온보딩 최적화가 핵심이며:

1. **Time-to-Value 최소화**: 첫 분석 30초 이내 완료
2. **즉각적 공유 유도**: 분석 완료 직후 공유 팝업 (전환율 25~35%)
3. **감정적 트리거 활용**: "상위 10% 피부" 등 특별한 결과 발견 시 공유 유도

### K-factor 측정 대시보드 지표

| 메트릭 | 계산 방법 | 목표 |
|-------|----------|-----|
| **공유율** | 공유 클릭 ÷ 결과 확인 | 25%+ |
| **초대당 클릭률** | 링크 클릭 ÷ 초대 발송 | 30%+ |
| **클릭→가입 전환율** | 가입 ÷ 링크 클릭 | 50%+ |
| **바이럴 사이클 타임** | 평균(가입→첫 공유) | 3일 이하 |

## 인플루언서 협업 전략

### 마이크로 인플루언서 활용 (팔로워 1만~10만)

마이크로 인플루언서는 메가 인플루언서 대비 **60% 높은 인게이지먼트**와 **6.7배 비용 효율**을 보입니다. 티르티르는 마이크로 인플루언서 의견을 제품 개발에 반영해 미국 아마존 파운데이션 1위를 달성했습니다.

| 티어 | 팔로워 | 협업 비용 | 활용 목적 |
|------|--------|----------|----------|
| **나노** | 1천~1만 | 제품 무료 제공 | 대량 시딩, UGC 생성 |
| **마이크로** | 1만~10만 | $100~500/게시물 | 신뢰도 구축, 타겟 도달 |
| **매크로** | 10만~100만 | $1,000~10,000 | 인지도 확대 |
| **메가** | 100만+ | $10,000+ | 대규모 론칭 |

**인플루언서 전용 혜택**
- 전용 초대 코드 (팔로워 할인 + 인플루언서 수익 배분)
- 신기능 선론칭 체험 기회
- 맞춤 상세 분석 리포트 제공

## 결론: 핵심 성공 지표

이룸의 바이럴 성장을 위한 핵심 KPI는 다음과 같습니다:

| 지표 | 목표치 | 측정 방법 |
|------|-------|----------|
| **K-factor** | 0.5~0.7 | (초대 수 × 전환율) 주간 측정 |
| **공유율** | 25%+ | 분석 완료 후 공유 클릭 비율 |
| **레퍼럴 전환율** | 50%+ | 초대 링크 클릭 → 가입 |
| **바이럴 사이클 타임** | 3일 이하 | 가입 → 첫 공유 평균 시간 |
| **CAC 절감률** | 20%+ | 레퍼럴 vs 유료 마케팅 비용 |

가장 중요한 원칙은 **분석 완료 직후의 감정적 고조 순간**에 공유를 유도하고, **프라이버시를 보호하는 아바타 옵션**으로 공유 심리적 장벽을 낮추며, **명확한 양면 보상**으로 추천인과 피추천인 모두에게 가치를 제공하는 것입니다. 한국 시장에서는 카카오톡이 압도적인 공유 채널이므로 Kakao Share API 최적화에 집중하고, 시즌별 해시태그 챌린지로 지속적인 UGC 생성을 유도해야 합니다.
