# AI 분석 결과의 바이럴 공유 메커니즘: 이룸(Yiroom) 성장 전략 가이드

퍼스널컬러 분석 결과는 본질적으로 높은 바이럴리티를 갖는다. **K-factor 0.5 이상을 달성하면 CAC를 50% 절감**할 수 있으며, 한국 MZ세대의 SNS 활용 특성과 뷰티 콘텐츠 소비 패턴을 활용한 전략적 접근이 필수다. 이 보고서는 이룸 플랫폼의 AI 분석 결과를 바이럴 성장 엔진으로 전환하기 위한 구체적인 UI/UX 가이드, 기술 구현 방법, 보상 체계, 그리고 마케팅 캘린더를 제시한다.

---

## 공유 가능한 결과 카드 설계 가이드라인

퍼스널컬러 분석 결과의 공유율을 극대화하려면 **시각적 완성도와 개인화 요소의 균형**이 핵심이다. 인스타그램 스토리 최적 사이즈는 **1080×1920px(9:16)**, 피드는 **1080×1350px(4:5)**가 표준이며, 카카오톡 링크 미리보기는 **800×400px**를 권장한다.

### 프라이버시 보호 공유 옵션

개인 사진 노출 우려를 해소하면서 공유를 유도하는 방법론이 필수다:

| 옵션 | 구현 방법 | 적합 상황 |
|------|----------|----------|
| **팔레트 전용 카드** | 분석 결과 컬러만 표시, 얼굴 제외 | 프라이버시 민감 사용자 |
| **스타일화된 아바타** | 사용자 컬러 적용된 일러스트 | 젊은 층 선호 |
| **블러 처리 옵션** | 사용자 선택 시 얼굴 가우시안 블러 | 중간 수준 공유 |
| **데이터 시각화** | 퍼센트/차트로 결과 표현 | 정보 중심 사용자 |

아이컬러(iColor)와 컬러와이즈(Colorwise)는 "사진 데이터는 어디에도 전송되지 않습니다"라는 명확한 프라이버시 성명을 결과 화면에 표시하여 사용자 신뢰를 확보했다.

### 인스타그램 스토리 공유 (React Native)

```javascript
import Share from 'react-native-share';
import { captureRef } from 'react-native-view-shot';

const shareToInstagramStory = async (resultCardRef) => {
  try {
    // 결과 카드 이미지 캡처 (1080x1920px)
    const uri = await captureRef(resultCardRef, {
      format: 'png',
      quality: 1,
      width: 1080,
      height: 1920,
    });

    await Share.shareSingle({
      appId: 'YOUR_FACEBOOK_APP_ID', // 2023년 1월부터 필수
      stickerImage: uri,
      social: Share.Social.INSTAGRAM_STORIES,
      backgroundBottomColor: '#FFFFFF',
      backgroundTopColor: '#FFF5F5', // 봄 웜 배경색 예시
    });
  } catch (error) {
    // 인스타그램 미설치 시 일반 공유 시트로 폴백
    await Share.open({ url: uri });
  }
};
```

## 친구 초대 보상 체계 설계

토스와 뱅크샐러드 사례 분석 결과, **양방향 보상(Two-Sided Reward)**과 **지연 지급 방식**이 어뷰징을 방지하면서 전환율을 극대화한다.

### 이룸 권장 보상 구조

| 구분 | 보상 내용 | 지급 조건 | 금액/가치 |
|------|----------|----------|----------|
| **초대자** | 이룸 크레딧 | 피초대자 첫 분석 완료 시 | **5,000원** |
| **피초대자** | 첫 분석 할인 + 프리미엄 체험 | 가입 완료 시 | **3,000원 할인 + 7일 무료** |
| **보너스 티어** | 추가 크레딧 | 5명 초대 달성 | **10,000원 보너스** |
| **골드 티어** | 1개월 프리미엄 | 10명 초대 달성 | **약 30,000원 가치** |

### 초대코드 vs 초대링크 하이브리드 전략

```

└── 온라인 공유 최적화

초대코드: YROOM7823
├── 구두/오프라인 공유 가능
├── 인플루언서 언급용

```

### 어뷰징 방지 체크리스트

토스는 "만 19세 이상, 24시간 내 코드 입력, 재가입 불가" 조건으로 어뷰징을 효과적으로 차단한다:

- [ ] **휴대폰 번호 인증**: 1인 1계정 원칙 (필수)
- [ ] **기기 핑거프린팅**: 동일 기기 중복 가입 탐지
- [ ] **가입 후 활동 조건**: 첫 분석 완료 또는 7일 활성 사용 후 보상 지급
- [ ] **일/월 초대 한도**: 일 10명, 월 50명 제한
- [ ] **보상 지급 지연**: 7~14일 검토 기간 후 자동 지급
- [ ] **탈퇴 후 재가입 제한**: 동일 휴대폰/기기 재가입 차단
- [ ] **쿠폰 사이트 모니터링**: 외부 노출 코드 비활성화

## 바이럴 계수(K-factor) 최적화 전략

K-factor는 **K = i × c** (i=사용자당 초대 수, c=초대 전환율)로 계산된다. **K > 1.0이면 자생적 성장**, K = 0.5면 유료 획득 비용을 50% 절감한다.

### 업계 벤치마크 및 목표

| 앱 카테고리 | 일반적 K-factor | 최고 사례 |
|------------|----------------|----------|
| 소셜 네트워크 | 0.5 - 1.0+ | Facebook 초기 K>1.0 |
| 소비자 앱 | 0.15 - 0.70 | Dropbox 0.7+, WhatsApp 0.4 |
| 뷰티/웰니스 | 0.2 - 0.5 | 퍼스널컬러 특성상 상향 가능 |
| B2B SaaS | 0.10 - 0.30 | Slack 최대 8.5 (예외적) |

**이룸 목표치:**
- **런칭 단계**: K = 0.3 ~ 0.5 (업계 평균 상회)
- **성장 단계 (6-12개월)**: K = 0.5 ~ 0.7 (상위 10%)
- **스트레치 목표**: K = 0.8+ (Dropbox 수준)

### 바이럴 사이클 타임 최적화

David Skok의 분석에 따르면, **사이클 타임 단축이 K-factor 증가보다 더 큰 영향**을 미친다:

```
K=2.0, 사이클타임=2일, 20일 후 → 20,470 사용자
K=2.0, 사이클타임=1일, 20일 후 → 20,000,000+ 사용자
```

이룸의 퍼스널컬러 분석은 **즉각적 만족(Instant Gratification)** 특성이 있어 동일 세션 내 공유가 가능하다. 목표 사이클 타임은 **24시간 이내**다.

### 이룸 바이럴 루프 설계

```
[셀피 업로드] → [AI 퍼스널컬러 분석 (30초)] → [결과 카드 생성]
      ↓                                              ↓
[Peak Delight 순간] ←←←←←←←←←←←←←←←←←← [공유 프롬프트 노출]
      ↓                                              ↓
[인스타/카카오 공유] → [친구가 결과 확인] → [호기심 유발]
      ↓                                              ↓
[친구 앱 다운로드] → [친구 분석 완료] → [친구도 공유] → (루프 반복)
```

### K-factor 향상 전략

| 전략 | 예상 효과 | 구현 우선순위 |
|------|----------|-------------|
| 결과 화면에 공유 버튼 눈에 띄게 배치 | 공유율 +30~50% | 🔴 높음 |
| 자동 생성 공유용 이미지 | 공유율 +100% | 🔴 높음 |
| 다중 플랫폼 공유 (카카오+인스타+X) | 도달 +40% | 🔴 높음 |
| Peak Delight 순간 공유 유도 | 타이밍 최적화 +50% | 🟡 중간 |
| 양방향 인센티브 | 전환율 2~3배 | 🟡 중간 |
| 마일스톤 보상 (3명, 5명, 10명) | 참여율 +20% | 🟢 낮음 |

## FOMO 및 소셜 프루프 UI 컴포넌트

한국 MZ세대의 **SNS 기반 소비 문화**와 **뒤처지지 않으려는 심리**를 활용한 FOMO 전략이 효과적이다.

## 바이럴 성과 측정 KPI 대시보드

### 핵심 지표

| 지표 | 계산식 | 이룸 목표치 |
|------|--------|------------|
| **K-factor** | 초대 수 × 전환율 | 0.5+ (런칭), 0.7+ (12개월) |
| **바이럴 사이클 타임** | 가입→추천 평균 일수 | < 3일 (목표: < 24시간) |
| **공유율** | 분석 완료자 중 공유자 비율 | > 15% |
| **초대 전환율** | 초대링크 클릭 → 가입 | > 15% (벤치마크: 15-25%) |
| **추천 CAC** | 리퍼럴 1인당 비용 | < ₩500 |
| **추천 유저 LTV** | 추천받은 유저 생애가치 | 일반 대비 +16-25% |
| **어뷰징 비율** | 무효 처리된 리퍼럴 비율 | < 5% |

### K-factor 계산 예시

```
시나리오: 이룸 런칭 3개월 후

분석 완료 사용자: 10,000명
공유한 사용자: 2,000명 (공유율 20%)
총 초대 발송: 6,000건 (1인당 평균 3건)
초대 통한 가입: 900명 (전환율 15%)

i = 6,000 / 10,000 = 0.6 (사용자당 초대 수)
c = 900 / 6,000 = 0.15 (전환율)
K = 0.6 × 0.15 = 0.09

목표 달성을 위한 개선:
- 공유율 20% → 40%로 개선 (i = 1.2)
- 전환율 15% → 25%로 개선 (양방향 인센티브)
- 개선된 K = 1.2 × 0.25 = 0.30 ✓
```

## 결론: 핵심 실행 우선순위

이룸의 퍼스널컬러 분석은 **본질적으로 공유하고 싶은 콘텐츠**다. 자기 발견의 순간, 시각적으로 아름다운 결과, 친구와 비교하고 싶은 호기심—이 세 가지가 바이럴 성장의 핵심 동력이다.

**즉시 실행 (Week 1-2):**
1. 1080×1920px 인스타 스토리 최적화 결과 카드 템플릿 구축
2. 카카오톡 공유 SDK 통합 완료
3. 분석 결과 reveal 순간에 공유 프롬프트 배치

**단기 실행 (Month 1):**
4. 양방향 보상 친구 초대 시스템 런칭 (초대자 5,000원/피초대자 3,000원)
5. 어뷰징 방지 기본 체계 구축 (휴대폰 인증 + 7일 지연 지급)
6. K-factor 트래킹 대시보드 구축

**중기 실행 (Month 2-3):**
7. #나의봄웜찾기 챌린지 런칭 (마이크로 인플루언서 50명 시딩)
8. FOMO UI 컴포넌트 (실시간 카운터, 친구 활동) A/B 테스트
9. UGC 인센티브 프로그램 정식 오픈

**K-factor 0.5 달성 시 유료 광고 CAC가 절반으로 감소**하며, 0.7 이상이면 업계 상위 10% 바이럴 성과다. 퍼스널컬러 분석의 고유한 "자기 발견" 특성과 한국 MZ세대의 SNS 활용 문화를 결합하면, 이룸은 뷰티/웰니스 앱 중 최고 수준의 바이럴 계수를 달성할 잠재력을 갖고 있다.
