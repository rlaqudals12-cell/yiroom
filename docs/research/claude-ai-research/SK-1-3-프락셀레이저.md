## 1. 시술 기본 정보

### 1.1 정식 명칭

| 구분 | 명칭 |
|------|------|
| **한글** | 프락셀 레이저, 프랙셔널 레이저 |
| **영문** | Fraxel Laser, Fractional Laser |
| **브랜드** | Fraxel (Solta Medical 고유 브랜드명) |

> "프락셀(Fraxel)"은 Solta Medical의 고유 브랜드명이며, 일반적인 기술 명칭은 "프랙셔널 레이저(Fractional Laser)"입니다.

### 1.2 시술 원리

**프랙셔널(Fractional) 기술의 핵심:**

1. **미세 열손상 치유 반응 (Fractional Photothermolysis)**
   - 피부 전체에 레이저를 조사하는 것이 아니라, 아주 미세한 점 형태로 에너지를 분할하여 피부 깊숙이 침투
   - 수천 개의 미세한 구역(Microthermal Treatment Zones, MTZ)을 피부 표면에 생성
   - 레이저가 조사된 범위에만 작은 상처가 생겨 조직 반응 유발

2. **빠른 회복의 원리**
   - 레이저가 조사되지 않은 주변 피부 조직은 손상 없이 건강한 상태로 유지
   - 건강한 조직이 손상된 조직의 치유를 촉진
   - 기존 전체 박피 레이저 대비 회복 기간 대폭 단축

3. **피부 재생 메커니즘**
   - 표피의 재생 주기를 리셋
   - 진피의 탄력 섬유 지지층에 자극을 주어 피부 DNA에서 엘라스틴, 콜라겐 등의 밀도 증진
   - 멜라닌 세포도 리셋되어 미백 효과 발생

### 1.3 프락셀 레이저 종류

| 종류 | 파장 | 특징 | 적응증 |
|------|------|------|--------|
| **CO2 프락셔널** | 10,600nm | 박피성, 강한 효과, 긴 회복기간 | 심한 여드름 흉터, 깊은 주름 |
| **어븀야그(Er:YAG)** | 2,940nm | 박피성, 중간 강도 | 모공, 흉터, 피부결 |
| **프락셀 듀얼** | 1,550nm + 1,927nm | 비박피성, 듀얼 파장 | 기미, 색소침착, 흉터 복합 |
| **프락셀 제나** | 1,550nm | 비박피성, 부드러운 시술 | 잔주름, 모공, 피부결 |
| **피코 프락셀** | 1,064nm | 초단 펄스, 열손상 최소화 | 모공, 피부결, 색소 |

**박피성 vs 비박피성:**

| 구분 | 박피성 (Ablative) | 비박피성 (Non-ablative) |
|------|-------------------|------------------------|
| **원리** | 표피부터 제거하며 들어감 | 표피는 보호, 진피층만 자극 |
| **효과** | 강력함 | 완만함 |
| **다운타임** | 7-14일 | 3-5일 |
| **부작용 위험** | 상대적으로 높음 | 상대적으로 낮음 |
| **대표 장비** | CO2 프락셀, 에코투 | 프락셀 듀얼, 제나, 모자이크 |

### 1.4 시술 부위

- 얼굴 전체 (Full Face)
- 특정 부위: 나비존(코주변), T존, U존, 볼, 이마, 턱, 관자놀이
- 목
- 데콜테 (가슴 상부)
- 손등
- 튼살 부위 (복부, 허벅지 등)

### 1.5 시술 시간

| 단계 | 소요 시간 |
|------|----------|
| 마취 (연고마취) | 20-30분 |

| 진정 관리 | 10-15분 |

### 1.6 마취 필요 여부

- **마취 시간**: 20-30분 도포

- **추가 마취**: 통증에 예민한 환자의 경우 추가적인 국소 마취 가능

### 2.2 FDA 승인 적응증

- Skin resurfacing (피부 재생)
- 잔주름 (Fine lines)
- 검버섯 및 Dyschromia (색소 불균형)
- 기미 (Melasma)

### 2.3 기대 효과

**단기 효과 (1-2주):**
- 각질 탈락 후 매끈한 새 피부
- 피부결 개선
- 즉각적인 피부 탄력감

**중기 효과 (1-3개월):**
- 콜라겐 재생으로 피부 탄력 증가
- 모공 축소
- 색소침착 완화

**장기 효과 (3-6개월):**
- 여드름 흉터 눈에 띄게 개선
- 잔주름 감소
- 피부 톤 균일화

### 2.4 효과 지속 기간

## 3. 시술 정보

### 3.1 시술 횟수 (권장 세션)

| 목적 | 권장 횟수 | 비고 |
|------|----------|------|

### 3.2 시술 간격

- **일반적**: 3-4주 간격
- **비박피성 프락셀**: 2-4주 간격
- **박피성 프락셀**: 4-6주 간격 (회복 기간 고려)

### 3.3 다운타임

| 시기 | 증상 |
|------|------|
| **시술 직후** | 화끈거림, 얼얼한 느낌 (1-2시간 내 소실) |
| **당일-3일** | 홍조, 붓기 (1-2일 후 감소) |
| **2-3일차** | 딱지 형성 시작, 미세 각질 |
| **3-7일차** | 각질 탈락, 피부 거칠게 느껴짐 |
| **7-10일차** | 딱지 탈락, 새 피부 노출 |
| **2-3주** | 기초적 재생 완료, 붉은기 잔존 가능 |

**다운타임 요약:**
- 비박피성: 3-5일
- 박피성: 7-14일

### 3.4 시술 후 주의사항

**필수 관리 (3대 원칙):**

1. **자외선 차단 (가장 중요)**
   - 시술 후 최소 2-4주간 철저한 자외선 차단
   - SPF 50+ PA++++ 자외선 차단제 수시 덧바름
   - 외출 시 모자, 양산 병행 권장
   - 게을리 시 시술 전보다 피부 악화 가능

2. **보습 관리**
   - 피부가 매우 건조해지므로 수시로 보습
   - 재생크림, 보습제 적극 사용
   - 수분 섭취 충분히

3. **자극 최소화**
   - 세안은 시술 다음날부터 부드럽게
   - 약산성 클렌저로 거품 세안
   - 손으로 세게 문지르지 않기
   - 각질 억지로 제거 금지 (색소침착 유발)

**금지 사항:**

| 기간 | 금지 항목 |
|------|----------|
| 3-7일 | 과음, 사우나/찜질방, 열탕 목욕, 격한 운동 |
| 1주 | 미백/주름개선 등 기능성 제품 사용 |
| 2주 | 스크럽, 필링 제품 사용 |
| 1개월 | 다른 레이저 시술 |

**피부 타입별 주의:**

- **지성 피부**: 유분 많은 재생크림 과다 사용 주의, 모공 막힘으로 좁쌀여드름 유발 가능
- **민감성 피부**: 시술 강도 조절 필요, 부작용 위험 높음
- **건성 피부**: 보습 관리 특히 중요

### 5.2 AI 추천 시 고려사항

**추천 조건 (Inclusion Criteria):**

```typescript
interface FraxelRecommendationCriteria {
  // 피부 고민 기반
  skinConcerns: {
    acneScars: boolean;      // 여드름 흉터
    enlargedPores: boolean;  // 넓은 모공
    fineWrinkles: boolean;   // 잔주름
    pigmentation: boolean;   // 색소침착/기미
    unevenTexture: boolean;  // 피부결 불균일
  };

  // 피부 상태 기반
  skinCondition: {
    sensitivityLevel: 'low' | 'medium' | 'high';
    activeAcne: boolean;
    inflammationPresent: boolean;
  };

  // 사용자 조건
  userProfile: {
    age: number;            // 20-60세 적합
    hasDowntimeCapacity: boolean;  // 다운타임 감당 가능 여부
    sunExposureRisk: 'low' | 'medium' | 'high';  // 자외선 노출 위험
  };
}
```

**비추천 조건 (Exclusion Criteria):**

- 활성 여드름/염증이 심한 경우
- 민감성 피부로 피부 반응이 심한 경우
- 켈로이드 체질
- 임산부/수유부
- 광과민성 질환 보유
- 최근 다른 레이저 시술 이력 (1개월 이내)
- 자외선 차단 관리가 어려운 환경

**AI 추천 로직:**

```typescript
function calculateFraxelScore(userProfile: UserProfile): number {
  let score = 0;

  // 피부 고민 점수 (최대 50점)
  if (userProfile.concerns.acneScars) score += 15;
  if (userProfile.concerns.enlargedPores) score += 10;
  if (userProfile.concerns.fineWrinkles) score += 10;
  if (userProfile.concerns.pigmentation) score += 10;
  if (userProfile.concerns.unevenTexture) score += 5;

  // 피부 타입 점수 (최대 30점)
  if (userProfile.skinType === 'normal') score += 30;
  if (userProfile.skinType === 'oily') score += 25;
  if (userProfile.skinType === 'dry') score += 25;
  if (userProfile.skinType === 'combination') score += 25;
  if (userProfile.skinType === 'sensitive') score += 15;

  // 관리 가능성 점수 (최대 20점)
  if (userProfile.canManageDowntime) score += 10;
  if (userProfile.canProtectFromSun) score += 10;

  // 제외 조건 (감점)
  if (userProfile.hasActiveAcne) score -= 30;
  if (userProfile.hasKeloidTendency) score -= 50;

  return Math.max(0, Math.min(100, score));
}
```

| **피코 레이저** | 색소+모공 동시 개선 | 프락셀 후 2-4주 |
| **울쎄라/써마지** | 탄력+모공 동시 개선 | 프락셀 후 4-6주 |
| **보톡스** | 잔주름 추가 개선 | 동시 또는 별도 |
| **필러** | 깊은 흉터 볼륨 보충 | 프락셀 완료 후 |
| **스킨부스터** | 수분+재생 촉진 | 프락셀 후 1-2주 |
| **재생 레이저 (LDM/힐라이트)** | 회복 촉진 | 시술 직후 |

**대체 시술 (프락셀이 부적합한 경우):**

| 상황 | 대체 시술 |
|------|----------|
| 민감성 피부 | 피코 레이저 (저강도) |
| 다운타임 불가 | 제네시스, IPL |
| 색소만 고민 | 피코토닝, 레이저토닝 |
| 모공만 고민 | 클라리티, 젠틀맥스 |
| 깊은 흉터 | 서브시전 + 프락셀 병합 |
