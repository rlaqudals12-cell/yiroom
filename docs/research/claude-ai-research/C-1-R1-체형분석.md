## 핵심 요약

체형 분석(C-1)은 골격 구조, 근육량, 체지방 분포를 종합적으로 평가하여 사용자의 체형 유형을 분류하고 맞춤형 스타일링/운동 추천을 제공하는 모듈이다. 핵심 기술로 **MediaPipe Pose 33 랜드마크**를 활용하며, 체형 분류 체계는 **과일형(5종)**과 **Kibbe 시스템(5대 유형)**을 병행 적용한다. 한국인 체형은 서양인 대비 **상체가 길고 하체가 짧은** 특성이 있어 Size Korea 8차 조사 데이터 기반 정규화가 필수적이다. 어깨:허리:엉덩이 비율 분석을 통해 체형을 자동 분류하고, 체형별 최적 스타일링 원리를 적용한다.

## 1. 체형 분류 체계

### 1.1 체형 결정 3대 요소

체형은 다음 세 가지 요소의 상호작용으로 결정된다.

| 요소 | 설명 | 측정 방법 | 변화 가능성 |
|------|------|----------|------------|
| **골격 구조** | 뼈대의 크기와 비율 | 어깨너비, 골반너비, 늑골 크기 | 불변 (성인 기준) |
| **근육량** | 근육의 분포와 발달 정도 | 둘레 측정, 체성분 분석 | 운동으로 변화 가능 |
| **체지방 분포** | 지방이 축적되는 부위 | 허리둘레, WHR, 체지방률 | 식이/운동으로 변화 가능 |

### 1.2 골격 구조 분류

골격 크기(Frame Size)는 체형 분류의 기초가 된다.

#### 손목 둘레 기준 (Metropolitan Life Insurance 방식)

**여성**:
| 신장 | 소골격 | 중골격 | 대골격 |
|------|--------|--------|--------|
| < 157cm | < 14.0cm | 14.0-14.6cm | > 14.6cm |
| 157-165cm | < 15.2cm | 15.2-15.9cm | > 15.9cm |
| > 165cm | < 16.5cm | 16.5-16.5cm | > 16.5cm |

**남성**:
| 신장 | 소골격 | 중골격 | 대골격 |
|------|--------|--------|--------|
| > 165cm | < 16.5cm | 16.5-19.0cm | > 19.0cm |

#### 팔꿈치 너비 기준

더 정확한 골격 측정법으로, 팔꿈치 폭(elbow breadth)을 사용한다.

```
골격 크기 = 팔꿈치 폭 / 신장 × 100

남성: 6.5-8.0 = 중골격
여성: 5.5-7.0 = 중골격
```

### 1.3 체지방 분포 유형

체지방 분포 패턴은 유전적으로 결정되며, 건강 위험도와 밀접한 관련이 있다.

| 분포 유형 | 특징 | 건강 위험 | 대표 체형 |
|----------|------|----------|----------|
| **Android (남성형)** | 복부 중심 축적 | 높음 (심혈관) | 사과형 |
| **Gynoid (여성형)** | 엉덩이/허벅지 중심 | 낮음 | 배형 |
| **균등 분포** | 전신 균등 | 중간 | 직사각형 |

### 1.4 체성분 비율

| 지표 | 남성 정상 | 여성 정상 | 측정 방법 |
|------|----------|----------|----------|

## 2. Kibbe 체형 시스템

### 2.1 Kibbe 시스템 개요

David Kibbe가 1987년 저서 "Metamorphosis"에서 제안한 체형 분류 시스템으로, **Yin(여성적/부드러움)**과 **Yang(남성적/날카로움)**의 균형을 기반으로 13가지 유형을 분류한다.

### 2.2 5대 주요 체형 (Family)

실용적 적용을 위해 13가지를 5대 유형으로 단순화한다.

| 유형 | Yin/Yang | 골격 특징 | 체형 특징 | 대표 연예인 |
|------|----------|----------|----------|------------|
| **Dramatic (D)** | Pure Yang | 길고 날카로움, 각진 어깨 | 직선적, 좁은 골반 | 틸다 스윈튼 |
| **Natural (N)** | Yang + 약간 Yin | 넓고 무딘 뼈대, 넓은 어깨 | 운동선수 체형, 직선적 | 제니퍼 애니스톤 |
| **Classic (C)** | 균형 | 중간 크기, 대칭적 | 균형 잡힌 비율 | 그레이스 켈리 |
| **Gamine (G)** | 혼합 | 작고 날카로움, 각진 | 짧은 사지, 소년 같은 | 오드리 헵번 |
| **Romantic (R)** | Pure Yin | 작고 둥근, 좁은 어깨 | 곡선적, 넓은 골반 | 마릴린 먼로 |

### 2.3 Kibbe 분류 기준

| 평가 영역 | Yang (A) | 균형 (B-C) | Yin (D-E) |
|----------|----------|-----------|-----------|
| **뼈대 크기** | 크고 길다 | 중간 | 작고 섬세 |
| **어깨 형태** | 넓고 각짐 | 약간 각짐/대칭 | 좁고 경사짐 |
| **사지 길이** | 길다 | 중간 | 짧다 |
| **손/발 크기** | 크다 | 중간 | 작다 |
| **체형 윤곽** | 직선적/각진 | 약간 곡선 | 곡선적/부드러움 |
| **허리 정의** | 불분명 | 중간 | 명확한 곡선 |
| **엉덩이/허벅지** | 평평/직선 | 중간 | 둥글고 풍만 |

### 2.4 Kibbe 점수 계산

```typescript
interface KibbeAssessment {
  bone: 'A' | 'B' | 'C' | 'D' | 'E';
  body: 'A' | 'B' | 'C' | 'D' | 'E';
  face: 'A' | 'B' | 'C' | 'D' | 'E';
}

// 점수 변환: A=1, B=2, C=3, D=4, E=5
function calculateKibbeType(assessment: KibbeAssessment): string {
  const boneScore = letterToNumber(assessment.bone);
  const bodyScore = letterToNumber(assessment.body);
  const faceScore = letterToNumber(assessment.face);

  const totalScore = boneScore + bodyScore + faceScore;
  const avgScore = totalScore / 3;

  // Yin/Yang 혼합도 분석
  const yangCount = [boneScore, bodyScore, faceScore].filter(s => s <= 2).length;
  const yinCount = [boneScore, bodyScore, faceScore].filter(s => s >= 4).length;

  if (avgScore <= 1.5) return 'Dramatic';
  if (avgScore <= 2.3 && yangCount >= 2) return 'Natural';
  if (avgScore >= 2.5 && avgScore <= 3.5) return 'Classic';
  if (yangCount >= 1 && yinCount >= 1) return 'Gamine';
  if (avgScore >= 4.0) return 'Romantic';

  return 'Classic'; // 기본값
}
```

### 2.5 Kibbe vs 과일형 매핑

| Kibbe 유형 | 유사 과일형 | 핵심 차이 |
|------------|-----------|----------|
| Dramatic | 역삼각형 | 뼈대 강조 vs 어깨너비 강조 |
| Natural | 직사각형 | 골격 크기 포함 vs 비율만 |
| Classic | 모래시계 | Yin/Yang 균형 vs 허리 곡선 |
| Gamine | 직사각형(소형) | 얼굴 포함 vs 체형만 |
| Romantic | 배형/모래시계 | 전체 부드러움 vs 하체 강조 |

### 2.6 Kibbe 시스템의 한계

| 한계 | 설명 | 이룸 대응 |
|------|------|----------|
| 주관성 | 자가 평가 시 오류 높음 | AI 이미지 분석 보완 |
| 복잡성 | 13유형 구분 어려움 | 5대 유형으로 단순화 |
| 서양 중심 | 아시아 체형 고려 부족 | 한국인 기준 보정 적용 |
| 얼굴 포함 | 체형만으로 판단 어려움 | 체형+얼굴 분리 분석 |

## 3. 체형 측정 비율

### 3.1 어깨:허리:엉덩이 비율 (SWH Ratio)

체형 분류의 핵심 측정값으로, 세 부위의 상대적 비율을 분석한다.

#### 측정 위치

| 부위 | 측정 위치 | MediaPipe 랜드마크 |
|------|----------|-------------------|
| **어깨** | 삼각근 최외측점 간 수평 거리 | 11, 12 (shoulder) |
| **허리** | 배꼽 높이 최소 둘레 | 추정값 (어깨-엉덩이 60%) |
| **엉덩이** | 둔부 최대 둘레 | 23, 24 (hip) |

#### 비율 계산

```typescript
interface SWHRatio {
  shoulder: number;  // cm
  waist: number;     // cm
  hip: number;       // cm
  shoulderToWaist: number;  // S:W
  waistToHip: number;       // W:H (WHR)
  shoulderToHip: number;    // S:H (SHR)
}

function calculateSWHRatio(measurements: {
  shoulder: number;
  waist: number;
  hip: number;
}): SWHRatio {
  return {
    ...measurements,
    shoulderToWaist: measurements.shoulder / measurements.waist,
    waistToHip: measurements.waist / measurements.hip,
    shoulderToHip: measurements.shoulder / measurements.hip,
  };
}
```

### 3.2 이상적 비율 기준

#### 여성 이상적 비율

| 비율 | 이상 값 | 범위 | 의미 |
|------|---------|------|------|
| WHR | 0.70 | 0.67-0.80 | 허리/엉덩이 비율 |
| SHR | 0.95-1.00 | 0.90-1.05 | 어깨/엉덩이 비율 |
| 어깨:허리:엉덩이 | 1:0.7:1 | - | 모래시계 기준 |

#### 남성 이상적 비율

| 비율 | 이상 값 | 범위 | 의미 |
|------|---------|------|------|
| WHR | 0.85 | 0.80-0.90 | 허리/엉덩이 비율 |
| SHR | 1.20-1.25 | 1.10-1.30 | 어깨/엉덩이 비율 |
| Adonis Index | 1.618 | 1.55-1.65 | 어깨/허리 황금비 |

### 3.3 체형 분류 알고리즘

```typescript
type BodyShape = 'hourglass' | 'pear' | 'apple' | 'rectangle' | 'invertedTriangle';

interface BodyShapeClassification {
  type: BodyShape;
  confidence: number;
  koreanName: string;
  description: string;
}

function classifyBodyShape(
  bust: number,   // 가슴둘레 (cm)
  waist: number,  // 허리둘레 (cm)
  hip: number,    // 엉덩이둘레 (cm)
  gender: 'male' | 'female'
): BodyShapeClassification {
  const bustHipDiff = Math.abs(bust - hip);
  const bustWaistDiff = bust - waist;
  const hipWaistDiff = hip - waist;
  const whr = waist / hip;

  if (gender === 'female') {
    // 모래시계형: 가슴≈엉덩이, 허리 명확히 작음
    if (bustHipDiff <= 2.5 && (bustWaistDiff >= 23 || hipWaistDiff >= 25)) {
      return {
        type: 'hourglass',
        confidence: 0.90,
        koreanName: '모래시계형',
        description: '어깨와 엉덩이가 비슷하고 허리가 잘록한 체형'
      };
    }

    // 배형: 엉덩이 > 가슴
    if (hip - bust >= 9 && hipWaistDiff < 23) {
      return {
        type: 'pear',
        confidence: 0.85,
        koreanName: '배형',
        description: '하체가 발달하고 엉덩이가 넓은 체형'
      };
    }

    // 역삼각형: 가슴 > 엉덩이
    if (bust - hip >= 9 && bustWaistDiff < 23) {
      return {
        type: 'invertedTriangle',
        confidence: 0.85,
        koreanName: '역삼각형',
        description: '어깨가 넓고 상체가 발달한 체형'
      };
    }

    // 사과형: 허리가 가장 큼
    if (whr >= 0.85 || waist >= bust || waist >= hip) {
      return {
        type: 'apple',
        confidence: 0.80,
        koreanName: '사과형',
        description: '복부 중심으로 둥근 체형'
      };
    }

    // 직사각형: 전체적으로 일자
    return {
      type: 'rectangle',
      confidence: 0.75,
      koreanName: '직사각형',
      description: '전체적으로 일자형으로 굴곡이 적은 체형'
    };
  }

  // 남성 체형 분류
  const shr = bust / hip;

  if (waist >= bust && waist >= hip) {
    return {
      type: 'apple',
      confidence: 0.80,
      koreanName: '타원형',
      description: '복부가 발달한 체형'
    };
  }

  if (hip > bust) {
    return {
      type: 'pear',
      confidence: 0.75,
      koreanName: '삼각형',
      description: '하체가 상체보다 넓은 체형'
    };
  }

  if (shr > 1.2) {
    return {
      type: 'invertedTriangle',
      confidence: 0.85,
      koreanName: '역삼각형',
      description: '넓은 어깨와 좁은 허리의 V자 체형'
    };
  }

  return {
    type: 'rectangle',
    confidence: 0.70,
    koreanName: '직사각형',
    description: '어깨, 허리, 엉덩이가 비슷한 체형'
  };
}
```

### 4.3 랜드마크 신뢰도 필터링

```typescript
interface PoseLandmark {
  x: number;           // 0.0 ~ 1.0 정규화
  y: number;           // 0.0 ~ 1.0 정규화
  z: number;           // 깊이 (카메라 기준)
  visibility: number;  // 가시성 (0.0 ~ 1.0)
  presence: number;    // 존재 확신도 (0.0 ~ 1.0)
}

const RELIABILITY_THRESHOLD = 0.5;

function isLandmarkReliable(landmark: PoseLandmark): boolean {
  return landmark.visibility >= RELIABILITY_THRESHOLD &&
         landmark.presence >= RELIABILITY_THRESHOLD;
}

function getReliableLandmarks(
  landmarks: PoseLandmark[],
  requiredIndices: number[]
): { valid: boolean; landmarks: PoseLandmark[] } {
  const result: PoseLandmark[] = [];

  for (const idx of requiredIndices) {
    const landmark = landmarks[idx];
    if (!isLandmarkReliable(landmark)) {
      return { valid: false, landmarks: [] };
    }
    result.push(landmark);
  }

  return { valid: true, landmarks: result };
}
```

### 4.4 픽셀-실제 치수 변환

```typescript
interface CalibrationData {
  referenceHeightPx: number;   // 참조 객체 픽셀 높이
  referenceHeightCm: number;   // 참조 객체 실제 높이
  focalLength?: number;        // 카메라 초점 거리 (선택)
}

function convertPixelToCm(
  pixelValue: number,
  calibration: CalibrationData
): number {
  const scale = calibration.referenceHeightCm / calibration.referenceHeightPx;
  return pixelValue * scale;
}

// 신장 기반 캘리브레이션 (참조 객체 없을 때)
function calibrateByHeight(
  userHeightCm: number,
  noseToAnklePx: number,
  headRatio: number = 0.13  // 머리 비율 약 13%
): number {
  // 전체 신장 = 코-발목 거리 / (1 - 머리비율)
  const estimatedFullHeightPx = noseToAnklePx / (1 - headRatio);
  return userHeightCm / estimatedFullHeightPx;
}
```

### 5.5 한국인 기준 정규화 함수

```typescript
const KOREAN_BODY_STANDARDS = {
  male: {
    '20s': { height: 174.4, shoulder: 401, waist: 78.4, hip: 94.3 },
    '30s': { height: 174.9, shoulder: 402, waist: 84.5, hip: 97.0 },
    '40s': { height: 172.5, shoulder: 397, waist: 87.0, hip: 97.5 },
    '50s': { height: 170.5, shoulder: 392, waist: 88.0, hip: 96.0 },
    '60s': { height: 168.2, shoulder: 387, waist: 87.5, hip: 94.5 },
  },
  female: {
    '20s': { height: 161.1, shoulder: 357, waist: 68.0, hip: 91.0 },
    '30s': { height: 162.0, shoulder: 362, waist: 72.0, hip: 93.5 },
    '40s': { height: 160.4, shoulder: 362, waist: 76.0, hip: 94.5 },
    '50s': { height: 157.5, shoulder: 357, waist: 79.0, hip: 95.0 },
    '60s': { height: 154.8, shoulder: 352, waist: 80.5, hip: 94.0 },
  },
};

const KOREAN_STD_DEVS = {
  male: { height: 5.8, shoulder: 21, waist: 8.5, hip: 5.2 },
  female: { height: 5.2, shoulder: 18, waist: 7.0, hip: 5.0 },
};

type AgeGroup = '20s' | '30s' | '40s' | '50s' | '60s';
type Measurement = 'height' | 'shoulder' | 'waist' | 'hip';

interface NormalizationResult {
  zScore: number;      // 표준점수
  percentile: number;  // 백분위
  category: string;    // 분류 (매우 작음/작음/보통/큼/매우 큼)
}

function normalizeToKorean(
  value: number,
  measurement: Measurement,
  gender: 'male' | 'female',
  ageGroup: AgeGroup
): NormalizationResult {
  const mean = KOREAN_BODY_STANDARDS[gender][ageGroup][measurement];
  const stdDev = KOREAN_STD_DEVS[gender][measurement];

  const zScore = (value - mean) / stdDev;

  // 정규분포 CDF 근사 (Abramowitz & Stegun)
  const t = 1 / (1 + 0.2316419 * Math.abs(zScore));
  const d = 0.3989423 * Math.exp(-zScore * zScore / 2);
  const p = d * t * (0.3193815 + t * (-0.3565638 + t * (1.781478 + t * (-1.821256 + t * 1.330274))));
  const percentile = zScore >= 0 ? (1 - p) * 100 : p * 100;

  // 카테고리 분류
  let category: string;
  if (percentile <= 10) category = '매우 작음';
  else if (percentile <= 30) category = '작음';
  else if (percentile <= 70) category = '보통';
  else if (percentile <= 90) category = '큼';
  else category = '매우 큼';

  return {
    zScore: Math.round(zScore * 100) / 100,
    percentile: Math.round(percentile),
    category,
  };
}
```

## 6. 체형별 스타일링 원리

### 6.1 스타일링 기본 원칙

| 원칙 | 설명 | 적용 |
|------|------|------|
| **비율 조정** | 시각적으로 이상적 비율에 가깝게 | 라인, 색상 배치 |
| **강조/분산** | 장점 강조, 콤플렉스 분산 | 디테일 위치 |
| **착시 활용** | 수직선=길어 보임, 수평선=넓어 보임 | 패턴, 커팅 |
| **색상 대비** | 밝은 색=확대, 어두운 색=축소 | 상하의 색상 |

### 6.2 체형별 스타일링 가이드

#### 모래시계형 (Hourglass)

```
특징: 어깨 ≈ 엉덩이, 허리 잘록함
목표: 곡선 강조, 허리 정의

✅ 권장
- 허리 벨트, 랩 스타일
- 바디콘 드레스, 피티드 재킷
- X-라인 실루엣
- 허리 절개 디테일

❌ 피할 것
- 박시 실루엣
- 허리 안 보이는 옷
- 일자 커팅
```

#### 배형 (Pear)

```
특징: 하체 > 상체, 엉덩이 넓음
목표: 상체 볼륨 추가, 시선 분산

✅ 권장
- 보트넥, 오프숄더 (어깨 강조)
- 밝은 색 상의 + 어두운 색 하의
- 러플, 패드 어깨
- A라인 스커트/원피스

❌ 피할 것
- 스키니진, 밝은 색 하의
- 타이트한 힙라인
- 큰 포켓이 달린 하의
```

#### 역삼각형 (Inverted Triangle)

```
특징: 어깨 > 엉덩이, 넓은 상체
목표: 하체 볼륨 추가, 상체 축소

✅ 권장
- V넥, 깊은 넥라인
- 어두운 색 상의 + 밝은 색 하의
- 와이드 팬츠, 플레어 스커트
- 힙 디테일 (포켓, 주름)

❌ 피할 것
- 패드 어깨, 넓은 넥라인
- 스트라이프 상의
- 너무 타이트한 하의
```

#### 사과형 (Apple)

```
특징: 복부 중심, 허리 불분명
목표: 세로 라인 강조, 복부 분산

✅ 권장
- V넥, 세로 스트라이프
- 엠파이어 라인, A라인
- 하이웨이스트 (가슴 아래)
- 긴 카디건, 롱 재킷

❌ 피할 것
- 타이트한 상의, 밸트
- 허리 강조 디테일
- 가로 스트라이프
- 짧은 상의
```

#### 직사각형 (Rectangle)

```
특징: 어깨 ≈ 허리 ≈ 엉덩이, 일자형
목표: 굴곡 생성, 허리 정의

✅ 권장
- 펠플럼, 페플럼 탑
- 벨트로 허리 정의
- 레이어링
- 러플, 드레이핑

❌ 피할 것
- 완전한 일자 커팅
- 굴곡 없는 실루엣
- 너무 박시한 옷
```

### 6.3 남성 체형별 스타일링

| 체형 | 권장 스타일 | 피할 스타일 |
|------|------------|------------|
| **역삼각형** | 슬림핏, V넥, 다크 상의 | 패드 어깨, 가로 스트라이프 |
| **사다리꼴** | 클래식 핏, 대부분 가능 | 극단적 실루엣 |
| **직사각형** | 레이어링, 구조적 재킷 | 너무 슬림/박시 |
| **삼각형** | 어깨 패드, 구조적 상의 | 타이트한 하의 |
| **타원형** | 세로 스트라이프, 다크 색상 | 타이트한 옷 전체 |

### 6.4 Kibbe 시스템 스타일링 원칙

| Kibbe 유형 | 라인 | 패브릭 | 디테일 |
|------------|------|--------|--------|
| **Dramatic** | 날카로운 직선, 길고 좁음 | 뻣뻣함, 광택 | 기하학적, 미니멀 |
| **Natural** | 부드러운 직선, 넓음 | 자연스러운 텍스처 | 캐주얼, 레이어드 |
| **Classic** | 대칭, 균형 | 중간 두께, 매끈함 | 절제된, 고전적 |
| **Gamine** | 직선+곡선 혼합, 짧음 | 가벼움, 선명한 패턴 | 장난스러운, 대비 |
| **Romantic** | 부드러운 곡선 | 드레이피, 부드러움 | 러플, 레이스 |
