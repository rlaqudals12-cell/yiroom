## 1. 시술 유형 체계 정리

### 1.1 에너지 기반 시술 (레이저/RF/HIFU) - 6종

| 시술명 | 에너지 원리 | 타겟층 | 주요 효과 | 지속 기간 |
|--------|------------|--------|----------|----------|
| **클라리티** | 레이저 (755nm/1064nm) | 표피-진피 | 제모, 색소 치료 | 영구 감모 |
| **인모드** | RF (바이폴라) | 진피-피하 | 타이트닝, 흉터, 컨투어링 | 6개월-4년 |
| **울쎄라** | HIFU (MFU-V) | SMAS (4.5mm) | 리프팅 | 12-24개월 |
| **써마지** | RF (모노폴라, 6.78MHz) | 진피 | 탄력, 피부결 | 1-2년 |
| **슈링크** | HIFU (MMFU) | SMAS (최대 13mm) | 리프팅 | 4-6개월 |
| **레이저토닝** | 레이저 (1064nm) | 표피-진피 | 색소(기미) 치료 | 유지 필요 |

### 1.2 주사 시술 - 8종

| 시술명 | 작용 메커니즘 | 주요 목적 | 효과 지속 |
|--------|-------------|----------|----------|
| **보톡스** | SNARE 복합체 차단 → 근육 이완 | 표정 주름(이마/미간/눈가) | 3-6개월 |
| **필러** | 히알루론산 볼륨 충전 | 정적 주름, 볼륨 손실 | 6-24개월 |
| **물광주사** | 비가교 HA + 영양 성분 주입 | 수분, 광채, 피부결 | 2-4개월 |
| **스킨보톡스** | 진피 내 저용량 보툴리눔 | 모공, 피지, 땀 억제 | 3-4개월 |
| **PRP** | 자가 혈소판 성장인자 | 피부 재생, 탈모 치료 | 6-12개월 |
| **줄기세포** | 파라크린 효과 (성장인자 분비) | 피부 재생, 흉터 | 연구 단계 |
| **리쥬란** | PDRN/PN → A2 Receptor 활성화 | 피부 재생, 콜라겐 합성 | 6-12개월 |
| **엑소좀** | miRNA/성장인자 전달 | 세포 재생 | 연구 단계 |

## 2. 피부 상태 → 시술 매핑 매트릭스

### 2.1 고민별 1순위/2순위 추천

| 피부 고민 | 1순위 추천 | 2순위 추천 | 주의사항 |
|-----------|-----------|-----------|---------|
| **처진 피부/리프팅** | 울쎄라 | 슈링크 | 얼굴 지방 적으면 볼패임 주의 |
| **피부 탄력 저하** | 써마지 | 인모드(Forma) | 얇은 피부에 써마지 우선 |
| **기미/색소** | 레이저토닝 | 클라리티 | 과도한 시술 시 저색소증 위험 |
| **여드름 흉터** | 인모드(Morpheus8) | MTS+PRP | 켈로이드 체질 금기 |
| **제모** | 클라리티 | - | Type IV 이상은 1064nm 필수 |
| **표정 주름(이마/미간/눈가)** | 보톡스 | - | 근본적 해결 |
| **정적 주름(팔자/입가)** | 필러 | 리쥬란 | 볼륨 충전 |
| **사각턱/이갈이** | 보톡스 | - | 교근 축소 |
| **피부 수분/광채** | 물광주사 | 리쥬란 | 즉각 효과 |
| **모공/피지** | 스킨보톡스 | 리쥬란 S | 피지선 억제 |
| **다크서클** | 카복시테라피 | 리쥬란 i | 혈관/색소형 구분 |
| **피지 과다/블랙헤드** | 아쿠아필 | 살리실산 필링 | 청색 LED 병합 |
| **칙칙함/각질** | 화학필링(AHA) | 스케일링 | 워밍업 후 시술 |
| **복합 고민** | 울써마지(병합) | 인모드 | 예산 고려 |

### 2.2 복합 고민 시 우선순위 결정 로직

```
1단계: 구조적 문제 (리프팅/처짐)
  → 울쎄라/슈링크/써마지 선 시술

2단계: 피부 톤 문제 (색소/기미)
  → 리프팅 회복 후 2-4주 뒤 레이저토닝/클라리티

3단계: 피부 질감 문제 (흉터/모공)
  → 인모드(Morpheus8)/MTS/포텐자

4단계: 유지 관리 (수분/재생)
  → 물광주사/리쥬란/PRP 정기 시술
```

## 3. 금기사항 및 리스크 통합

### 3.1 절대 금기 사항

|------|----------|

| **슈링크** | 페이스메이커, 시술 부위 금속, 얼굴 지방 매우 적은 경우 |

| 시술 | 절대 금기 |

|------|----------|

## 5. 시술 조합 및 병합 전략

### 5.1 효과적인 시술 조합

| 조합 | 목적 | 시너지 효과 | 간격 |
|------|------|-----------|------|
| **보톡스 + 필러** | 종합 안티에이징 | 동적+정적 주름 동시 해결 | 동시 가능 |
| **물광 + 리쥬란** | 피부 재생 | 수분+재생 시너지 | 2주 간격 |
| **울쎄라 + 써마지** | 프리미엄 리프팅 | SMAS+진피 이중 타겟 | 동시 또는 2주 |
| **MTS + PRP** | 프리미엄 재생 | 흡수율 극대화, 콜라겐 1000%↑ | 동시 |
| **스킨보톡스 + 리쥬란S** | 모공 집중 | 피지억제+재생 | 2주 간격 |
| **아쿠아필 + LED** | 피지/여드름 | 클렌징+살균 | 동시 |

interface SkinProcedureMatchInput {

### 6.2 추천 로직 플로우

```
1. 금기사항 필터링
   └─ 절대 금기 해당 시술 제외

2. 피부 타입 기반 1차 필터링
   └─ Type III-IV: RF/HIFU 우선
   └─ Type I-II: 모든 옵션

3. 주요 고민 기반 2차 필터링
   └─ 리프팅: 울쎄라/슈링크/써마지
   └─ 색소: 레이저토닝/클라리티
   └─ 재생: 리쥬란/PRP/물광

4. 예산 기반 3차 필터링
   └─ 예산 범위 내 최적 시술 선택

5. 부가 조건 반영
   └─ 지방 적음 + HIFU = 볼패임 경고
   └─ 통증 민감 = 슈링크/인모드 우선
   └─ 바쁜 라이프스타일 = 다운타임 최소 시술

6. 조합 추천
   └─ 시너지 조합 제안
```
