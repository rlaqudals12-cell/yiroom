# C-2-R1: 의료기기법 경계

> AI 피부분석 앱의 의료기기 해당 여부 및 법적 리스크 분석

## 1. 리서치 배경

### 1.1 핵심 질문

이룸(Yiroom)의 AI 피부분석 서비스가 의료기기법 적용 대상인가?

### 1.2 관련 법령

- 디지털의료제품법 (2024년 제정, 2025년 시행)
- 의료기기법
- AI기본법 (2026년 1월 22일 시행)

## 2. 의료기기 분류 체계 (2025년)

### 2.1 디지털의료제품법 시행

2025년 디지털의료제품법이 본격 시행되면서 AI 기반 제품의 규제 체계가 명확해졌습니다.

| 분류 | 정의 | 규제 수준 |
|------|------|----------|
| **SaMD** | Software as a Medical Device | 의료기기 허가 필요 |
| **SiMD** | Software in Medical Device | 의료기기 허가 필요 |
| **디지털 건강지원기기** | 비의료 목적 건강관리 | 자율신고제 |

### 2.2 의료기기 등급 분류

식약처 '디지털 의료기기 분류 및 등급 지정 가이드라인'에 따른 등급:

| 등급 | 위험도 | 예시 |
|------|--------|------|
| 1등급 | 최저 | 건강정보 기록/조회 |
| 2등급 | 낮음 | 비침습적 측정 보조 |
| 3등급 | 중간 | 진단 보조 소프트웨어 |
| 4등급 | 높음 | 치료 결정 지원 |

### 2.3 자체 위험도 평가 (Self-Assessment)

기업은 다음 기준으로 자사 제품의 등급을 결정해야 합니다:

1. **제품의 목적** - 의료 목적 vs 일반 건강관리
2. **알고리즘 개입 수준** - 정보 제공 vs 의사결정 지원
3. **환자 안전 영향** - 직접 영향 vs 간접 영향

## 3. 이룸 서비스 분석

### 3.1 이룸의 AI 분석 기능

| 기능 | 목적 | 출력 |
|------|------|------|
| 퍼스널컬러 분석 | 색상 추천 | 시즌 타입 |
| 피부 타입 분석 | 스킨케어 추천 | 건성/지성/복합성 |
| 피부 상태 분석 | 제품 추천 | 수분도, 유분도 점수 |
| 체형 분석 | 스타일 추천 | 체형 타입 |

### 3.2 의료기기 해당 여부 판단

```
이룸 서비스 분석:

✅ 비의료 목적 요소:
├── 미용/뷰티 목적
├── 제품 추천 목적
├── 스타일 가이드 목적
└── 진단/치료 의도 없음

❌ 의료기기 해당 요소 (없음):
├── 질병 진단 기능 없음
├── 치료 방향 결정 없음
├── 의료진 의사결정 지원 없음
└── 처방/투약 연계 없음
```

### 3.3 결론: 비의료기기 (디지털 건강지원기기)

이룸의 AI 피부분석 서비스는 **의료기기에 해당하지 않습니다**.

| 판단 기준 | 이룸 서비스 | 의료기기 요건 |
|----------|------------|--------------|
| 목적 | 뷰티/스타일 추천 | 질병 진단/치료 |
| 출력 | 제품/색상 추천 | 진단 결과/치료 방향 |
| 사용자 | 일반 소비자 | 환자/의료진 |
| 결과 활용 | 구매 참고 | 의료 의사결정 |

## 4. 법적 안전 가이드라인

### 4.1 금지 표현 (의료기기 오인 방지)

```
❌ 사용 금지:
- "피부 질환 진단"
- "피부암 조기 발견"
- "의료급 분석"
- "피부과 수준 진단"
- "병원용 피부 분석"
- "의사 대체 가능"

✅ 권장 표현:
- "피부 타입 분석"
- "피부 상태 체크"
- "스킨케어 추천을 위한 분석"
- "뷰티 컨설팅 보조"
- "참고용 피부 정보"
```

### 4.2 면책 조항 (Disclaimer)

모든 분석 결과 화면에 다음 문구 표시:

```
"본 분석 결과는 뷰티 제품 추천을 위한 참고 정보이며,
의학적 진단이나 치료를 대체하지 않습니다.
피부 건강에 대한 우려가 있으시면 전문의와 상담하시기 바랍니다."
```

### 4.3 기능 범위 제한

| 허용 | 금지 |
|------|------|
| 피부 타입 분류 (건성/지성) | 피부 질환 진단 |
| 수분도/유분도 점수 | 특정 질병 가능성 |
| 제품 성분 추천 | 치료제 추천 |
| 스타일 가이드 | 의료 조언 |
