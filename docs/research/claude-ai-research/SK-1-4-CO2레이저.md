# SK-1-4: CO2 레이저 (Carbon Dioxide Laser) 피부과 시술 리서치

## 1. 핵심 요약

**CO2 레이저**(이산화탄소 레이저, Carbon Dioxide Laser)는 **10,600nm 파장**의 원적외선을 방출하여 피부 조직 내 수분에 흡수되어 조직을 기화(증발)시키는 절제(ablative) 레이저입니다. 점, 사마귀, 검버섯, 쥐젖 등 다양한 양성 피부 병변 제거에 **골드 스탠다드**로 사용되며, 프락셔널 모드로는 흉터 치료와 피부 재생에도 활용됩니다. **Fitzpatrick 피부 타입 I-III**에 가장 적합하며, 타입 IV 이상에서는 색소 침착 위험이 증가합니다. 시술 후 **1-2주**의 다운타임이 필요하며, 한국 기준 시술 비용은 부위와 범위에 따라 **1만원~100만원** 이상까지 다양합니다.

## 2. 시술 기본 정보

### 2.1 정식 명칭

| 언어 | 명칭 |

| **영문** | Carbon Dioxide Laser, CO2 Laser |
| **의학 용어** | Ablative CO2 Laser, Fractional CO2 Laser (프락셔널 모드) |

### 2.2 시술 원리

#### 물리적 원리

CO2 레이저는 이산화탄소(CO2)를 매질로 하여 **10,600nm 파장**의 원적외선을 발생시킵니다. 이 파장은 피부 조직 내 **수분(H2O)**에 선택적으로 흡수되어 다음과 같은 과정을 거칩니다:

```
레이저 에너지 조사
       ↓
조직 내 수분이 레이저 흡수
       ↓
순간적 열 발생 (100°C 이상)
       ↓
세포 내 수분 증발 → 조직 기화(vaporization)
       ↓
병변 조직 제거
```

#### 작용 메커니즘

1. **광열 효과 (Photothermal Effect)**
   - 레이저 에너지가 조직 내 수분에 흡수
   - 수분의 급격한 온도 상승 → 조직 증발
   - 주변 조직에 열 확산 → 지혈 효과

2. **조직 제거 방식**
   - **Non-fractional (전통적)**: 연속 조사로 병변 전체 제거
   - **Fractional (프락셔널)**: 미세 레이저 빔을 점 형태로 조사하여 마이크로 열 손상 존(MTZ) 생성

3. **콜라겐 리모델링**
   - 열 자극에 의한 진피층 콜라겐 수축
   - 새로운 콜라겐 및 엘라스틴 섬유 생성 촉진
   - 피부 탄력 및 질감 개선

#### 레이저 유형 비교

| 특성 | CO2 레이저 | 어븀야그 레이저 (Er:YAG) |
|------|-----------|------------------------|
| **파장** | 10,600nm | 2,940nm |
| **흡수 계수** | 중간 | 높음 (수분에 10-15배 더 흡수) |
| **조직 침투 깊이** | 상대적으로 깊음 | 얕음 |
| **열 손상 범위** | 넓음 (20-150μm) | 좁음 (5-20μm) |
| **지혈 효과** | 우수 | 미흡 (출혈 있을 수 있음) |
| **회복 기간** | 다소 김 | 빠름 |
| **흉터 위험** | 시술자 경험 중요 | 상대적으로 적음 |

### 2.3 시술 부위

CO2 레이저는 전신의 다양한 부위에 적용 가능합니다:

| 부위 | 주요 적응증 |
|------|------------|
| **얼굴** | 점, 검버섯, 사마귀, 비립종, 여드름 흉터, 주름 |
| **목** | 검버섯, 쥐젖(연성섬유종), 사마귀 |
| **몸통** | 검버섯, 쥐젖, 지루각화증 |
| **손/발** | 티눈, 사마귀(심상성, 발바닥) |
| **생식기** | 곤지름(첨규콘딜로마) |
| **눈꺼풀** | 안검황색종, 한관종, 눈꺼풀 성형 (특수) |

### 2.4 시술 시간

| 시술 범위 | 소요 시간 |
|----------|----------|
| 단일 점 제거 (1-2mm) | 1-5분 |
| 다발성 점 (5-10개) | 10-20분 |
| 검버섯 다수 | 15-30분 |
| 프락셔널 CO2 (전체 얼굴) | 30-60분 |
| 흉터 치료 (부분) | 20-40분 |

### 2.5 마취 필요 여부

| 시술 유형 | 마취 방법 |
|----------|----------|
| **작은 점 (1-2mm)** | 마취 없이 또는 국소 마취 크림 |
| **중간 크기 점 (3-5mm)** | 국소 마취 주사 권장 |
| **큰 점/융기 병변** | 국소 마취 주사 필수 |
| **프락셔널 CO2 (전체 얼굴)** | 국소 마취 크림 (30-60분) + 냉각 |
| **사마귀/티눈** | 국소 마취 주사 |

**마취 유형 상세:**
- **국소 마취 크림 (EMLA 등)**: 시술 30-60분 전 도포
- **국소 마취 주사 (리도카인)**: 시술 직전, 병변 주위에 주사하여 통증 완전 차단
- **신경 차단 마취**: 넓은 범위 시술 시

#### 금기증 (Contraindications)

**절대적 금기:**
- 활동성 감염 (피부 감염, 헤르페스 활성기)
- 악성 피부 병변 (확진 전)
- 켈로이드 체질 (주의 필요)
- 임신/수유 중

**상대적 금기:**
- Fitzpatrick 피부 타입 V-VI (고위험)
- 최근 일광 화상 또는 태닝
- 광과민성 약물 복용 중
- 당뇨병 등 상처 치유 지연 질환
- 최근 피부 박피 시술 이력

### 4.4 시술 후 주의사항

#### 필수 관리

1. **재생테이프 관리**
   - 시술 후 1-2주간 재생테이프 부착
   - 1-2일마다 교체
   - 테이프 부착 상태로 세안 가능

2. **세안 및 위생**
   - 시술 당일 세안 가능 (테이프 부착 상태)
   - 토너/알코올 성분 자극제 피하기
   - 딱지는 절대 손으로 떼지 않기

3. **자외선 차단**
   - **필수**: SPF 50+ 자외선 차단제 사용
   - 직사광선 노출 최소화
   - 모자/양산 사용 권장
   - 최소 3-6개월간 철저한 자외선 관리

4. **일상 활동 제한**
   - 시술 후 7일간 격한 운동, 땀 나는 활동 자제
   - 찜질방, 사우나, 수영장 2주간 금지
   - 음주 1주일간 자제

#### 금지 사항

- 딱지 억지로 떼기 (흉터 위험)
- 스크럽/필링 (회복 전)
- 과도한 화장 (초기 1주일)
- 태닝 및 직사광선 노출

#### 합병증 및 대처

| 합병증 | 발생률 | 대처 방법 |
|--------|--------|----------|
| **일시적 홍조** | 매우 흔함 | 정상 경과, 시간이 지나면 호전 |
| **색소 침착 (PIH)** | 10-30% (아시아인) | 미백 연고, 자외선 차단 철저 |
| **색소 저하** | 드묾 (< 5%) | 시간 경과로 일부 회복 |
| **흉터** | 드묾 (< 2%) | 시술자 경험이 중요, 조기 개입 |
| **감염** | 매우 드묾 | 항생제 연고/복용 |
| **재발** | 병변에 따라 다양 | 재시술 |

## 6. 이룸 앱 연동

### 6.1 피부 타입별 적합도

#### Fitzpatrick 피부 타입별 권장도

| 타입 | 피부색 | CO2 레이저 적합도 | 주의사항 |
|------|--------|------------------|----------|
| **I** | 매우 밝은 백색 | ★★★★★ | 최적 |
| **II** | 밝은 백색 | ★★★★★ | 매우 적합 |
| **III** | 밝은 갈색 (유럽/아시아) | ★★★★☆ | 적합, 색소 침착 주의 |
| **IV** | 중간 갈색 (동아시아, 지중해) | ★★★☆☆ | 주의 필요, 보수적 설정 |
| **V** | 어두운 갈색 | ★★☆☆☆ | 높은 위험, 대안 권장 |
| **VI** | 매우 어두운 갈색/흑색 | ★☆☆☆☆ | 비권장, 다른 방법 고려 |

**한국인 대부분은 Fitzpatrick III-IV에 해당하며, 적절한 시술 강도와 사후 관리가 중요합니다.**

#### 이룸 피부 분석 연동

| 이룸 피부 타입 | CO2 레이저 권장 | 대안 시술 |
|---------------|----------------|----------|
| **민감성 피부** | 주의 필요 | 어븀야그 레이저 |
| **건성 피부** | 적합 | - |
| **지성 피부** | 적합 | - |
| **복합성 피부** | 적합 | - |
| **트러블성 피부** | 염증 완화 후 가능 | 여드름 치료 선행 |

function evaluateCO2Laser(input: CO2LaserRecommendation): CO2LaserResult {

  const riskLevel = calculateOverallRisk(input, skinRisk);

### 6.3 연관 시술

#### 대체 시술

| 시술 | 적응증 | CO2 대비 장점 | CO2 대비 단점 |
|------|--------|--------------|--------------|
| **어븀야그 레이저** | 점, 사마귀, 흉터 | 열 손상 적음, 회복 빠름 | 지혈 약함, 깊은 병변 제한 |
| **외과적 절제** | 큰 점, 의심 병변 | 조직 검사 가능, 완전 제거 | 흉터, 봉합 필요 |
| **냉동 치료** | 사마귀, 검버섯 | 간단, 저렴 | 정밀도 낮음, 여러 번 필요 |
| **고주파 치료** | 쥐젖, 사마귀 | 빠름, 저렴 | 흉터 가능성 |

#### 병행 시술

| 시술 | 병행 목적 | 시기 |
|------|----------|------|
| **미백 레이저/필링** | PIH 예방/치료 | 회복 후 4-8주 |
| **재생 레이저 (프락셔널)** | 흉터 개선 | 회복 후 6-8주 |
| **PRP/줄기세포** | 재생 촉진 | 동시 또는 직후 |
| **보습 관리** | 회복 촉진 | 시술 직후부터 |

## 8. 결론

CO2 레이저는 **10,600nm 파장**을 이용한 절제 레이저로, 점, 사마귀, 검버섯 등 다양한 양성 피부 병변 제거에 **골드 스탠다드**로 사용됩니다. 한국인(Fitzpatrick III-IV)에서는 적절히 사용 가능하나, **색소 침착(PIH)** 위험이 있어 철저한 자외선 차단과 사후 관리가 필수입니다.

이룸 앱에서는 사용자의 **피부 타입**, **병변 특성**, **생활 패턴**을 종합 분석하여 CO2 레이저 적합도를 평가하고, 필요시 대안 시술을 추천하는 것이 바람직합니다. 특히 **Fitzpatrick V-VI** 또는 **켈로이드 병력**이 있는 사용자에게는 주의 메시지와 함께 다른 치료 옵션을 제시해야 합니다.
