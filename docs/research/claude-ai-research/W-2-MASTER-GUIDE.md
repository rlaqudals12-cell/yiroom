## 1. 스트레칭 과학적 기초

### 1.1 근육 생리학 기초

#### 근섬유 유형
| 유형 | 특성 | 스트레칭 반응 |
|------|------|--------------|
| Type I (지근) | 지구력, 저강도 | 느린 이완, 지속적 스트레칭 효과적 |
| Type IIa (속근) | 중간 특성 | 동적 스트레칭에 반응 |
| Type IIb/IIx (속근) | 순발력, 고강도 | 빠른 동적 스트레칭 필요 |

#### 근육-건 단위 (Muscle-Tendon Unit)
- **직렬 탄성 요소 (SEC)**: 힘줄, 아포네우로시스 - 즉각적 장력 전달
- **병렬 탄성 요소 (PEC)**: 근막, 에피미시움 - 수동적 장력 유지
- **수축 요소 (CE)**: 근섬유 - 능동적 장력 생성

#### 점탄성 특성 (Viscoelasticity)
```
스트레칭 → 크리프 현상 → 응력 이완 → 히스테리시스
                ↓
        연조직 길이 증가 (일시적)
                ↓
        반복 적용 시 영구적 변화
```

### 1.2 신경생리학적 메커니즘

#### 근방추 (Muscle Spindle)
| 구성요소 | 기능 | 스트레칭 반응 |
|---------|------|--------------|
| Ia 구심성 섬유 | 빠른 길이 변화 감지 | 동적 스트레칭 시 활성화 |
| II 구심성 섬유 | 정적 길이 유지 감지 | 정적 스트레칭 시 활성화 |
| γ-운동뉴런 | 민감도 조절 | 훈련으로 조절 가능 |

#### 골지건기관 (GTO)
- **위치**: 근육-건 접합부
- **기능**: 장력 감지 → 과도한 힘 시 보호적 억제
- **스트레칭 활용**: PNF 기법의 자가억제 원리

#### 핵심 반사 메커니즘
| 반사 | 메커니즘 | 스트레칭 적용 |
|------|---------|-------------|
| 신장반사 (Stretch Reflex) | 갑작스러운 신장 → 반사적 수축 | 느린 스트레칭으로 회피 |
| 자가억제 (Autogenic Inhibition) | GTO 자극 → 동일 근육 이완 | PNF Contract-Relax |
| 상호억제 (Reciprocal Inhibition) | 주동근 수축 → 길항근 이완 | 능동적 스트레칭 |

### 1.3 관절 가동범위 (ROM) 표준

#### 주요 관절 ROM (한국인 기준)
| 관절 | 운동 | 정상 ROM | 기능적 ROM |
|------|------|---------|-----------|
| 목 (경추) | 굴곡/신전 | 45°/45° | 35°/35° |
| 목 (경추) | 회전 | 60°/60° | 50°/50° |
| 어깨 | 굴곡/신전 | 180°/60° | 160°/40° |
| 어깨 | 외전/내전 | 180°/45° | 160°/40° |
| 고관절 | 굴곡/신전 | 120°/30° | 100°/20° |
| 고관절 | 외회전/내회전 | 45°/35° | 40°/30° |
| 무릎 | 굴곡/신전 | 135°/0° | 120°/0° |
| 발목 | 배굴/저굴 | 20°/50° | 15°/40° |

## 2. 스트레칭 유형 분류 체계

### 2.1 정적 스트레칭 (Static Stretching)

#### 정의 및 특성
- 근육을 늘린 상태에서 15-60초 유지
- 신장반사 회피, 점탄성 변화 유도
- 유연성 증가에 가장 효과적

#### 프로토콜
```
단계 1: 준비 자세
단계 2: 천천히 신장 (2-4초에 걸쳐)
단계 3: 불편감 느껴지는 지점에서 정지
단계 4: 15-60초 유지
단계 5: 천천히 원위치
반복: 2-4세트
```

#### 적용 시기
| 시기 | 권장 | 이유 |
|------|------|------|
| 운동 전 | ⚠️ 제한적 | 근력/파워 일시적 감소 |
| 운동 후 | ✅ 권장 | 쿨다운, 유연성 향상 |
| 별도 세션 | ✅ 권장 | 최적의 ROM 증가 |

### 2.2 동적 스트레칭 (Dynamic Stretching)

#### 정의 및 특성
- 조절된 움직임으로 ROM 점진적 확장
- 신경근 활성화, 체온 상승
- 운동 전 워밍업에 최적

#### 유형
| 유형 | 설명 | 예시 |
|------|------|------|
| 리드미컬 동적 | 일정한 리듬으로 반복 | 레그스윙, 암서클 |
| 기능적 동적 | 운동 동작 모방 | 런지워크, 하이니즈 |
| 점진적 동적 | 강도/범위 점차 증가 | A-스킵, B-스킵 |

#### 프로토콜
```
강도: 낮은 → 중간 강도
속도: 조절된 속도 (반동 금지)
반복: 10-15회/세트
세트: 1-2세트/동작
총 시간: 5-10분
```

### 2.3 PNF 스트레칭 (고유수용성 신경근 촉진법)

#### 기법 분류
| 기법 | 절차 | 메커니즘 |
|------|------|---------|
| Contract-Relax (CR) | 수축 → 이완 → 스트레칭 | 자가억제 |
| Hold-Relax (HR) | 등척성 수축 → 이완 → 스트레칭 | 자가억제 |
| Contract-Relax-Agonist-Contract (CRAC) | CR + 길항근 수축 | 자가억제 + 상호억제 |
| Hold-Relax-Swing (HRS) | HR + 탄도적 움직임 | 복합 |

#### Contract-Relax 프로토콜
```
1. 초기 스트레칭 위치 설정 (10초)
2. 등척성 수축 6-10초 (50-75% 최대 강도)
3. 2-3초 이완
4. 수동적 스트레칭 10-30초
5. 2-4세트 반복
```

#### 효과
- ROM 증가: 정적 스트레칭 대비 10-15% 추가 개선
- 신경근 적응: 스트레칭 내성 증가
- 주의: 숙련된 파트너 또는 적절한 도구 필요

### 2.4 탄도적 스트레칭 (Ballistic Stretching)

#### 특성
- 반동을 이용한 빠른 움직임
- 신장반사 유발 가능 → 부상 위험
- 일반인 권장하지 않음

#### 제한적 적용
| 대상 | 조건 | 예시 |
|------|------|------|
| 엘리트 운동선수 | 충분한 기초 유연성 확보 후 | 체조, 무용 |
| 스포츠 특이적 | 해당 동작 요구 시 | 발차기 동작 |

### 2.5 스트레칭 유형 비교표

| 기준 | 정적 | 동적 | PNF | 탄도적 |
|------|------|------|-----|-------|
| ROM 증가 | ★★★★ | ★★★ | ★★★★★ | ★★★ |
| 부상 위험 | 낮음 | 낮음 | 중간 | 높음 |
| 운동 전 적합 | ⚠️ | ✅ | ⚠️ | ❌ |
| 운동 후 적합 | ✅ | ⚠️ | ✅ | ❌ |
| 난이도 | 쉬움 | 쉬움 | 어려움 | 중간 |
| 파트너 필요 | 불필요 | 불필요 | 권장 | 불필요 |

### 4.2 골프

#### 한국 골프 트렌드 (2025)
- 인구 600만+ (스크린골프 포함)
- 2030 여성 골퍼 급증
- 주요 부상: 허리 (40%), 어깨 (25%), 손목/팔꿈치 (20%)

#### 바이오메카닉스

## 5. 자세교정 원칙 및 안전

### 5.1 자세 불균형 유형

#### Janda Crossed Syndrome

**상부교차증후군 (Upper Crossed Syndrome)**
```
단축 근육:
- 상부승모근, 견갑거근
- 대흉근, 소흉근
- 후두하근

약화/신장 근육:
- 심부 경추 굴근
- 하부승모근, 전거근
- 능형근

결과: 전방두위, 라운드숄더
```

**하부교차증후군 (Lower Crossed Syndrome)**
```
단축 근육:
- 장요근
- 척추기립근
- 대퇴직근

약화/신장 근육:
- 복근
- 대둔근
- 햄스트링

결과: 골반 전방경사, 요추 전만 증가
```

### 5.2 교정 원리

#### Thomas Myers - Anatomy Trains
주요 근막 라인:
- **Superficial Back Line**: 발바닥 → 두피
- **Superficial Front Line**: 발등 → 흉쇄유돌근
- **Lateral Line**: 외측 발 → 두개골
- **Spiral Line**: 전신 나선형 연결

**적용 원리**
```
단일 근육 아닌 근막 라인 전체 고려
→ 통증 부위 아닌 원인 부위 교정
→ 전신적 접근
```

### 6.4 C-1 (체형분석) 연동

```typescript
// C-1 분석 결과에 따른 추가 스트레칭
function getPostureBasedStretches(postureData: PostureData): StretchItem[] {
  const additionalStretches: StretchItem[] = [];

  // 상부교차증후군 패턴
  if (postureData.upperCrossed) {
    additionalStretches.push(
      { name: 'Doorway Pec Stretch', ... },
      { name: 'Chin Tuck', ... },
      { name: 'Upper Trap Stretch', ... }
    );
  }

  // 하부교차증후군 패턴
  if (postureData.lowerCrossed) {
    additionalStretches.push(
      { name: 'Hip Flexor Stretch', ... },
      { name: 'Thomas Stretch', ... },
      { name: 'Cat-Cow', ... }
    );
  }

  // 측면 불균형
  if (postureData.lateralImbalance) {
    // 짧은 쪽 집중 스트레칭
    additionalStretches.push(
      { name: 'Side Bend Stretch', side: postureData.shortSide, ... }
    );
  }

  return additionalStretches;
}
```

### A. ACSM/NSCA 유연성 가이드라인 요약

**ACSM 권장사항 (2021)**
- 빈도: 주 2-3회 이상
- 강도: 약간의 불편감 수준
- 시간: 정적 스트레칭 15-60초/근육군
- 유형: 정적, PNF, 동적 스트레칭

**NSCA 권장사항**
- 운동 전: 동적 스트레칭 5-10분
- 운동 후: 정적 스트레칭 10-15분
- 개별 스트레칭: 2-4세트
