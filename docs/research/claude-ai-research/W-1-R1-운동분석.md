## 2. MET (Metabolic Equivalent of Task) 시스템

### 2.1 MET 정의

```
MET (Metabolic Equivalent of Task):
────────────────────────────────────
1 MET = 안정 시 산소 소비량 = 3.5 mL O₂/kg/min

물리적 정의:
- 1 MET = 1 kcal/kg/h ≈ 안정 시 대사율
- 활동의 에너지 비용을 안정 시 대비 배수로 표현

예시:
- 앉아있기: 1.0 MET
- 걷기 (4.8 km/h): 3.0 MET → 안정 시의 3배 에너지 소비
- 달리기 (9.7 km/h): 9.8 MET → 안정 시의 약 10배 에너지 소비
```

### 2.3 칼로리 소모 계산 공식

```
기본 공식 (ACSM):
────────────────────────────────────────────────
Calories (kcal) = MET × 체중(kg) × 시간(h)

상세 공식 (산소 소비량 기반):
────────────────────────────────────────────────
VO₂ (mL/min) = MET × 3.5 × 체중(kg)
Calories (kcal) = VO₂ (L/min) × 시간(min) × 5.0

예시: 70kg 사람이 5.0 MET 운동을 30분 수행
────────────────────────────────────────────────
방법 1: 5.0 × 70 × 0.5h = 175 kcal
방법 2: (5.0 × 3.5 × 70 / 1000) × 30 × 5.0 = 183.75 kcal

※ 방법 1이 실용적으로 많이 사용됨
※ 실제 소모량은 개인 차이로 ±15% 변동 가능
```

```typescript
// lib/workout/calorie-calculator.ts
interface CalorieParams {
  met: number;           // MET 값 (1.0-20.0)
  weightKg: number;      // 체중 (kg)
  durationMinutes: number; // 운동 시간 (분)
}

interface CalorieResult {
  calories: number;           // 소모 칼로리 (kcal)
  intensity: IntensityLevel;  // 강도 분류
  metCategory: string;        // MET 카테고리명
  confidenceRange: {          // 신뢰 구간 (±15%)
    min: number;
    max: number;
  };
}

type IntensityLevel = 'light' | 'moderate' | 'vigorous' | 'near_maximal';

export function calculateCalories(params: CalorieParams): CalorieResult {
  const { met, weightKg, durationMinutes } = params;

  // 입력 검증
  if (met < 1.0 || met > 20.0) {
    throw new Error('Invalid MET value: must be 1.0-20.0');
  }
  if (weightKg <= 0 || weightKg > 300) {
    throw new Error('Invalid weight: must be 1-300 kg');
  }
  if (durationMinutes <= 0 || durationMinutes > 600) {
    throw new Error('Invalid duration: must be 1-600 minutes');
  }

  const durationHours = durationMinutes / 60;
  const calories = Math.round(met * weightKg * durationHours);

  // 강도 분류 (ACSM 기준)
  const intensity = classifyIntensity(met);

  // 신뢰 구간 (개인 차이 ±15%)
  const confidenceRange = {
    min: Math.round(calories * 0.85),
    max: Math.round(calories * 1.15),
  };

  return {
    calories,
    intensity,
    metCategory: getMETCategory(met),
    confidenceRange,
  };
}

function classifyIntensity(met: number): IntensityLevel {
  if (met < 3.0) return 'light';
  if (met < 6.0) return 'moderate';
  if (met < 9.0) return 'vigorous';
  return 'near_maximal';
}

function getMETCategory(met: number): string {
  if (met < 2.0) return 'Very Light (아주 가벼움)';
  if (met < 3.0) return 'Light (가벼움)';
  if (met < 6.0) return 'Moderate (중강도)';
  if (met < 9.0) return 'Vigorous (고강도)';
  return 'Near-Maximal (최대 근접)';
}
```

```
┌─────────────────────────────────────────────────────────────────┐
│                    ACSM 운동 유형 분류 (FITT-VP)                 │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  1. 유산소 운동 (Aerobic/Cardiorespiratory)                     │
│     ├── 정의: 심폐 지구력 향상을 위한 지속적 대근육 활동         │

│                                                                  │
│  2. 저항 운동 (Resistance/Strength)                             │
│     ├── 정의: 근력, 근지구력, 근비대를 위한 저항 기반 운동       │

│                                                                  │
│  3. 유연성 운동 (Flexibility)                                    │
│     ├── 정의: 관절 가동 범위(ROM) 유지/향상을 위한 운동          │
│     ├── 예시: 정적 스트레칭, 동적 스트레칭, PNF, 요가           │
│     ├── 목표 근육: 단축된 근육, 긴장된 근막                      │
│     └── 효과: 유연성 향상, 부상 예방, 자세 개선                 │
│                                                                  │
│  4. 신경근 운동 (Neuromotor/Functional Fitness)                  │
│     ├── 정의: 균형, 민첩성, 협응력 향상을 위한 운동              │
│     ├── 예시: 태극권, 요가, 균형 훈련, 기능성 운동              │
│     ├── 대상: 특히 고령자에게 중요                              │
│     └── 효과: 낙상 예방, 일상 기능 유지, 자세 안정성            │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### 4.2 심박수 기반 강도 측정

#### 4.2.1 최대 심박수(HRmax) 추정 공식

```
┌─────────────────────────────────────────────────────────────────┐
│              나이별 최대심박수(HRmax) 추정 공식                   │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  1. Fox 공식 (1971) - 가장 널리 알려짐, 오차 큼                 │
│     HRmax = 220 - 나이                                          │
│     예: 30세 → 220 - 30 = 190 bpm                              │
│     오차: ±10-12 bpm (표준편차)                                 │
│                                                                  │
│  2. Tanaka 공식 (2001) - 메타분석 기반, 권장 ✅                 │
│     HRmax = 208 - (0.7 × 나이)                                  │
│     예: 30세 → 208 - 21 = 187 bpm                              │
│     오차: ±7-10 bpm (더 정확)                                   │
│                                                                  │
│  3. Gellish 공식 (2007) - 건강한 성인용                         │
│     HRmax = 207 - (0.7 × 나이)                                  │
│                                                                  │
│  4. Inbar 공식 (1994) - 운동선수용                              │
│     HRmax = 205.8 - (0.685 × 나이)                              │
│                                                                  │
│  ※ 이룸은 Tanaka 공식을 기본으로 사용                           │
│  ※ 실제 측정값이 있으면 측정값 우선                             │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

#### 4.2.2 Karvonen 공식 (목표 심박수)

```
Karvonen 공식 (Heart Rate Reserve 기반):
────────────────────────────────────────────────────────────────

THR = ((HRmax - HRrest) × 강도%) + HRrest

여기서:
- THR: Target Heart Rate (목표 심박수)
- HRmax: 최대 심박수
- HRrest: 안정 시 심박수
- HRR: Heart Rate Reserve = HRmax - HRrest

예시: 30세, 안정시 심박수 60bpm, Zone 3 (70%) 계산
────────────────────────────────────────────────────────────────
Step 1: HRmax = 208 - (0.7 × 30) = 187 bpm
Step 2: HRR = 187 - 60 = 127 bpm
Step 3: THR (70%) = (127 × 0.70) + 60 = 149 bpm
```

```typescript
// lib/workout/heart-rate-zones.ts
interface HeartRateZone {
  zone: number;
  name: string;
  minPercent: number;
  maxPercent: number;
  minHR: number;
  maxHR: number;
  purpose: string;
  duration: string;
  fuel: string;
}

export function calculateHRmax(age: number): number {
  // Tanaka 공식 (2001)
  return Math.round(208 - (0.7 * age));
}

export function calculateKarvonenZones(
  age: number,
  restingHR: number = 70  // 기본값
): HeartRateZone[] {
  const hrMax = calculateHRmax(age);
  const hrReserve = hrMax - restingHR;

  const calcTHR = (percent: number): number => {
    return Math.round(hrReserve * (percent / 100) + restingHR);
  };

  return [
    {
      zone: 1,
      name: '회복 (Recovery)',
      minPercent: 50,
      maxPercent: 60,
      minHR: calcTHR(50),
      maxHR: calcTHR(60),
      purpose: '워밍업, 쿨다운, 적극적 회복',
      duration: '5-10분 (전후)',
      fuel: '지방 (주로)',
    },
    {
      zone: 2,
      name: '지방 연소 (Fat Burn)',
      minPercent: 60,
      maxPercent: 70,
      minHR: calcTHR(60),
      maxHR: calcTHR(70),
      purpose: '체지방 연소, 기초 지구력',
      duration: '30-60분',
      fuel: '지방 (주로)',
    },
    {
      zone: 3,
      name: '유산소 (Aerobic)',
      minPercent: 70,
      maxPercent: 80,
      minHR: calcTHR(70),
      maxHR: calcTHR(80),
      purpose: '심폐 지구력 향상, VO₂max 개선',
      duration: '20-40분',
      fuel: '지방 + 탄수화물',
    },
    {
      zone: 4,
      name: '무산소 역치 (Anaerobic Threshold)',
      minPercent: 80,
      maxPercent: 90,
      minHR: calcTHR(80),
      maxHR: calcTHR(90),
      purpose: '젖산 역치 향상, 속도 지구력',
      duration: '10-20분 (인터벌)',
      fuel: '탄수화물 (주로)',
    },
    {
      zone: 5,
      name: '최대 (VO₂max)',
      minPercent: 90,
      maxPercent: 100,
      minHR: calcTHR(90),
      maxHR: hrMax,
      purpose: '최대 심폐 능력, 스프린트',
      duration: '30초-2분 (인터벌)',
      fuel: '탄수화물 + 무산소',
    },
  ];
}
```
