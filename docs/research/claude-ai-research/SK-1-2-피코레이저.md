# SK-1-2: 피코레이저 (Pico Laser) 시술 리서치

## 1. 핵심 요약

피코레이저는 **피코초(10^-12초, 1조분의 1초)** 단위로 레이저를 조사하는 최신 피부과 레이저 기술입니다. 기존 나노초(10^-9초) 레이저보다 **1,000배 빠른 조사 속도**로 주변 조직 손상을 최소화하면서 색소를 효과적으로 파괴합니다. 주요 적응증은 **기미, 주근깨, 잡티, 오타모반, 문신 제거, 모공/흉터 개선, 피부 리쥬브네이션**입니다. 이룸 앱에서는 피부 분석 결과와 연계하여 사용자에게 적합한 시술 정보를 제공할 수 있습니다.

### 2.1 정식 명칭

| 구분 | 명칭 |
|------|------|
| **한글** | 피코레이저, 피코토닝, 피코초 레이저 |

| **주요 브랜드** | 피코슈어(PicoSure), 피코웨이(PicoWay), 피코플러스(PicoPlus), 피코케어(PicoCare), 인라이튼(enLIGHTen) |

### 2.2 시술 원리 (작용 메커니즘)

#### 피코초(Picosecond)란?

- **1 피코초 = 10^-12초 = 1조분의 1초**
- 기존 Q-스위치 레이저: 나노초(10^-9초) 단위
- 피코레이저: 나노초의 **1/1000** 속도

#### 핵심 메커니즘

```
1. 광열효과 (Photothermal) → 광음향/광기계효과 (Photoacoustic/Photomechanical)로 전환

2. 작동 원리:
   ┌─────────────────────────────────────────────────────────────┐
   │ 피코초 펄스 → 초고속 온도 상승 → 강력한 음향 충격파 발생     │
   │                    ↓                                        │
   │ 색소 입자에 장력(tensile stress) 가해짐                      │
   │                    ↓                                        │
   │ 조직 파괴 임계값 초과 → 색소 미세 분해                       │
   └─────────────────────────────────────────────────────────────┘

3. 비유:
   - 나노초 레이저: 돌 → 자갈로 분해
   - 피코초 레이저: 돌 → 모래로 분해
```

#### 스트레스 이완 시간(SRT) vs 열 이완 시간(TRT)

| 메커니즘 | 레이저 유형 | 원리 |
|----------|-------------|------|
| **TRT 기반** | 나노초 레이저 | 열 이완 시간보다 짧은 펄스로 선택적 광열분해 |
| **SRT 기반** | 피코초 레이저 | 스트레스 이완 시간보다 짧은 펄스로 광기계적 파괴 |

#### 프락셔널 피코레이저

```
프락셔널 전달 시스템 부착 시:
- LIOB (Laser-Induced Optical Breakdown): 표피층
- LIC (Laser-Induced Cavitation): 진피층
→ 피부 재생 및 진피 리모델링 촉진
```

#### 파장별 특성

| 파장 | 타겟 색소 | 용도 |
|------|----------|------|
| **532nm** | 멜라닌(표피), 빨간색/주황색 잉크 | 표피 색소, 문신 |
| **595nm** | 헤모글로빈 | 혈관 병변 |
| **660nm** | 녹색/청색 잉크 | 문신 제거 |
| **755nm** | 멜라닌 | 색소 병변, 문신 |
| **1064nm** | 멜라닌(진피), 검정/파란색 잉크 | 심부 색소, 문신 |

### 2.3 시술 부위

- **얼굴 전체**: 기미, 주근깨, 잡티, 피부톤 개선
- **부분 얼굴**: 광대, T존, 눈가 등 특정 부위
- **목 및 데콜테**: 색소침착, 광노화
- **손등**: 검버섯, 잡티
- **전신**: 문신 제거 (팔, 다리, 등 등)

### 2.4 시술 시간

| 시술 범위 | 소요 시간 |
|-----------|----------|
| **부분 시술** | 10-15분 |
| **얼굴 전체** | 15-30분 |
| **문신 제거** | 15-60분 (크기에 따라) |

### 2.5 마취 필요 여부

| 시술 유형 | 마취 |
|-----------|------|
| **피코토닝** | 불필요 또는 쿨링 |
| **피코프락셀** | 국소 마취 크림 (30분 전) |
| **문신 제거** | 국소 마취 크림 권장 |

## 3. 적응증 및 효과

### 3.1 적응증

#### 색소 질환

| 질환 | 설명 | 효과 |
|------|------|------|
| **기미 (Melasma)** | 진피형/혼합형 기미 | ★★★★☆ |
| **주근깨 (Freckles)** | 표피성 색소 | ★★★★★ |
| **잡티/검버섯** | 광노화 색소침착 | ★★★★★ |
| **오타모반** | 진피 멜라닌세포 증가 | ★★★★☆ |
| **ABNOM** | 후천성 양측성 오타양모반 | ★★★★☆ |
| **염증 후 색소침착** | PIH | ★★★☆☆ |

#### 문신 제거

| 문신 색상 | 권장 파장 | 효과 |
|-----------|----------|------|
| **검정/회색** | 1064nm | ★★★★★ |
| **파란색** | 1064nm, 755nm | ★★★★★ |
| **빨간색** | 532nm | ★★★★☆ |
| **노란색** | 532nm | ★★★☆☆ |
| **녹색** | 660nm, 755nm | ★★★☆☆ |

#### 피부 리쥬브네이션

- **모공 축소**: 프락셔널 모드 활용
- **여드름 흉터**: 진피 콜라겐 리모델링
- **잔주름 개선**: 탄력 증가
- **피부 결 개선**: 전반적 피부톤 균일화

- 활성 피부 감염이나 염증 시 시술 연기

### 6.3 AI 추천 시 고려사항

```typescript
// 피코레이저 추천 로직 예시
interface SkinAnalysisResult {
  pigmentationScore: number;  // 0-100
  elasticityScore: number;    // 0-100
  poreScore: number;          // 0-100
  sensitivityScore: number;   // 0-100
  moistureScore: number;      // 0-100
  skinType: 'dry' | 'oily' | 'combination' | 'sensitive' | 'acne';
}

function getPicoLaserRecommendation(analysis: SkinAnalysisResult) {
  const recommendations = {
    isRecommended: false,
    priority: 'low' as 'high' | 'medium' | 'low',
    treatmentType: [] as string[],
    cautions: [] as string[],
    alternativeSuggestions: [] as string[],
  };

  // 색소침착 점수 높으면 적극 추천
  if (analysis.pigmentationScore > 60) {
    recommendations.isRecommended = true;
    recommendations.priority = 'high';
    recommendations.treatmentType.push('피코토닝');
  }

  // 모공/탄력 문제시 프락셀 추천
  if (analysis.poreScore > 50 || analysis.elasticityScore < 50) {
    recommendations.isRecommended = true;
    recommendations.treatmentType.push('피코프락셀');
  }

  // 민감성 피부 주의
  if (analysis.sensitivityScore > 70) {
    recommendations.priority = 'low';
    recommendations.cautions.push('민감성 피부로 전문의 상담 필수');
    recommendations.cautions.push('테스트 시술 권장');
  }

  // 건조한 피부 주의
  if (analysis.moistureScore < 40) {
    recommendations.cautions.push('시술 전후 충분한 보습 필요');
  }

  return recommendations;
}
```
