## 1. 시술 기본 정보

### 1.1 명칭

| 구분 | 명칭 |
|------|------|
| **한글 정식명** | 제네시스 레이저, 제네시스 토닝 |
| **영문 정식명** | Laser Genesis, Genesis Laser |
| **장비명** | Cutera XEO (제오 레이저) 제네시스 모드 |
| **레이저 종류** | 롱펄스 Nd:YAG (Long Pulse Nd:YAG) |
| **파장** | 1064nm |

### 1.2 시술 원리

제네시스 레이저는 **1064nm 파장의 롱펄스 Nd:YAG 레이저**를 이용한 **논어블레이티브(Non-ablative) 스킨 리쥬브네이션** 시술이다.

#### 작용 메커니즘

```
레이저 에너지 조사
        ↓
표피 통과 (손상 없음)
        ↓
진피 상부 가열 (40~50°C)
        ↓
┌─────────────────────────────────────┐
│  3가지 동시 작용                      │
├─────────────────────────────────────┤
│ 1. 미세혈관 응고 (Micro-vascular     │
│    Coagulation)                      │
│    → 홍조, 모세혈관 확장증 개선       │
│                                      │
│ 2. 콜라겐 자극 (Collagen             │
│    Stimulation)                      │
│    → 섬유아세포 활성화               │
│    → 새로운 콜라겐 생성              │
│                                      │
│ 3. 피지선 조절 (Sebaceous Gland      │
│    Modulation)                       │
│    → 피지 분비 감소                  │
│    → 모공 개선                       │
└─────────────────────────────────────┘
```

#### 핵심 원리

1. **마이크로펄스 전달**: 수천 개의 마이크로 펄스로 에너지를 전달하여 표피(피부 외층)를 손상으로부터 보호
2. **열 확산**: 피부 표면에 손상을 주지 않고 진피층까지 레이저 에너지 깊숙이 전달
3. **멜라닌/헤모글로빈 작용**: 표피를 통과한 레이저가 멜라닌 색소와 옥시헤모글로빈에 선택적으로 작용
4. **섬유아세포 활성화**: 진피층의 섬유아세포를 자극하여 콜라겐 섬유 재생 유도

## 2. 적응증 및 효과

### 2.1 주요 적응증

| 적응증 | 설명 | 효과 정도 |
|--------|------|----------|
| **안면 홍조** | 확산성 홍조, 얼굴 붉어짐 | ★★★★★ |
| **주사(Rosacea)** | 모세혈관 확장증 | ★★★★★ |
| **넓은 모공** | 모공 축소 | ★★★★☆ |
| **잔주름** | 미세 주름 개선 | ★★★★☆ |
| **피부 결** | 피부 질감 개선 | ★★★★★ |
| **피부 탄력** | 콜라겐 재생으로 탄력 증가 | ★★★★☆ |
| **여드름 홍반** | 여드름 붉은 자국 | ★★★★☆ |
| **여드름 흉터** | 얕은 흉터 개선 | ★★★☆☆ |
| **기미** | 혈관 주변 기미 유발요인 제거 | ★★★☆☆ |
| **피부 톤 불균일** | 전체적인 톤 정리 | ★★★★☆ |

### 2.2 기대 효과

#### 단기 효과 (시술 직후 ~ 1주)

- 피부 윤기 증가
- 피부 톤 개선
- 일시적인 홍조 감소

#### 중기 효과 (3~6주)

- 모공 축소
- 잔주름 완화
- 피부 결 개선
- 콜라겐 재생 시작

#### 장기 효과 (3~6개월)

- 지속적인 콜라겐 리모델링
- 피부 탄력 증가
- 전반적인 피부 재생

### 2.3 효과 지속 기간

| 항목 | 기간 |
|------|------|
| **효과 발현** | 시술 후 1주부터 점진적 개선 |
| **최적 효과** | 3~6회 시술 후 |
| **효과 지속** | 6~12개월 |
| **유지 관리** | 6개월~1년 간격 1회 유지 시술 권장 |

#### 추천 조건 (Positive Triggers)

```typescript
// AI 추천 로직 예시
const shouldRecommendGenesis = (userProfile: SkinProfile): boolean => {
  const positiveIndicators = [
    userProfile.concerns.includes('redness'),      // 홍조
```
