## 1. 시술 기본 정보

### 1.1 정식 명칭

| 구분 | 명칭 |
|------|------|
| **한글** | 인텐스 펄스 라이트, 아이피엘 |
| **영문** | Intense Pulsed Light (IPL) |
| **별칭** | 광선 치료, 플래시램프 치료, 포토페이셜 |

### 1.2 시술 원리 (작용 메커니즘)

IPL은 **선택적 광열분해(Selective Photothermolysis)** 원리를 기반으로 작동한다.

#### 핵심 메커니즘

```
빛 에너지 조사 → 특정 발색단(Chromophore) 흡수 → 열 에너지 변환 → 표적 조직 파괴
```

1. **다중 파장 방출**: 500~1200nm의 다양한 파장을 포함한 광대역 빛을 피부에 조사
2. **선택적 흡수**: 빛 에너지가 특정 발색단(멜라닌, 헤모글로빈)에 선택적으로 흡수
3. **열 변환**: 흡수된 빛 에너지가 열로 변환되어 표적 조직 가열
4. **조직 파괴**: 열에 의해 색소 세포, 혈관, 모낭 등 표적 조직 선택적 파괴
5. **콜라겐 자극**: 진피층 열 자극으로 콜라겐/엘라스틴 합성 촉진

#### 레이저와의 차이점

| 구분 | IPL | 레이저 |
|------|-----|--------|
| 파장 | 다중 파장 (500~1200nm) | 단일 파장 |
| 치료 범위 | 복합적 피부 문제 동시 개선 | 특정 문제에 집중 |
| 조사 면적 | 넓음 (플래시램프) | 좁음 (집중 빔) |
| 에너지 강도 | 상대적으로 낮음 | 높음 |
| 다운타임 | 적음 | 시술에 따라 다양 |

### 1.3 시술 부위

| 부위 | 적용 가능성 | 주요 적응증 |
|------|------------|------------|
| **얼굴 전체** | ✅ 주요 부위 | 색소침착, 홍조, 잔주름 |
| **목** | ✅ 가능 | 색소, 광노화 |
| **손등** | ✅ 가능 | 검버섯, 잡티 |
| **데콜테** | ✅ 가능 | 색소, 혈관 |
| **팔/다리** | ✅ 가능 | 제모, 모세혈관 |

### 1.4 시술 시간

| 부위 | 소요 시간 |
|------|----------|
| 얼굴 전체 | 20~30분 |
| 부분 시술 | 10~15분 |
| 전신 제모 | 60~90분 |

### 1.5 마취 필요 여부

| 구분 | 필요 여부 | 비고 |
|------|----------|------|
| **마취** | 불필요 | 대부분 무마취 진행 |
| **쿨링 젤** | 필수 | 시술 전 도포, 피부 보호 |
| **국소 마취크림** | 선택적 | 민감한 환자의 경우 |

## 2. 적응증 및 효과

### 2.1 적응증 (어떤 피부 문제에 효과적인가)

#### 색소성 병변 (멜라닌 타겟)

| 적응증 | 효과 수준 | 권장 횟수 |
|--------|----------|----------|
| 주근깨 | ⭐⭐⭐⭐⭐ | 1~3회 |
| 잡티 | ⭐⭐⭐⭐⭐ | 2~3회 |
| 검버섯 | ⭐⭐⭐⭐ | 2~4회 |
| 기미 (표피성) | ⭐⭐⭐ | 3~5회 |
| 칙칙한 피부톤 | ⭐⭐⭐⭐ | 3~5회 |

#### 혈관성 병변 (헤모글로빈 타겟)

| 적응증 | 효과 수준 | 권장 횟수 |
|--------|----------|----------|
| 안면 홍조 | ⭐⭐⭐⭐ | 3~5회 |
| 모세혈관 확장 | ⭐⭐⭐⭐ | 2~4회 |
| 주사(로사시아) | ⭐⭐⭐ | 5회+ |
| 붉은 여드름 자국 | ⭐⭐⭐⭐ | 3~5회 |

#### 피부 재생 (콜라겐 자극)

| 적응증 | 효과 수준 | 권장 횟수 |
|--------|----------|----------|
| 잔주름 | ⭐⭐⭐ | 5회+ |
| 모공 개선 | ⭐⭐⭐ | 5회+ |
| 피부 탄력 저하 | ⭐⭐⭐ | 5회+ |
| 광노화 | ⭐⭐⭐⭐ | 5회+ |

#### 기타

| 적응증 | 효과 수준 | 비고 |
|--------|----------|------|
| 영구 제모 | ⭐⭐⭐ | 레이저 대비 효과 낮음 |
| 여드름 (활성) | ⭐⭐⭐ | P.acnes 살균 |
| 안구건조증 (MGD) | ⭐⭐⭐⭐ | 안과 적응증 |

### 2.2 기대 효과

```
1회 시술 후:
├── 주근깨, 잡티: 즉시 개선 (일시적으로 더 진해진 후 1주 내 탈락)
├── 피부톤: 즉시 환해짐
└── 혈관: 일시적 개선

3~5회 시술 후:
├── 색소 병변: 80%+ 개선
├── 홍조: 50~70% 개선
├── 모공/탄력: 점진적 개선 시작
└── 전반적 피부결 개선
```

### 2.3 효과 지속 기간

| 적응증 | 효과 지속 | 유지 권장 |
|--------|----------|----------|
| 색소 (주근깨, 잡티) | 6개월~2년 | 3~6개월마다 유지 시술 |
| 홍조/혈관 | 1~2년 | 필요시 추가 시술 |
| 탄력/모공 | 6개월~1년 | 지속적 유지 시술 |
| 제모 | 영구적 (일부 재생) | 1~2년 후 터치업 |

> **주의**: 자외선 노출, 생활 습관에 따라 효과 지속 기간이 달라질 수 있다.

## 3. 시술 정보

### 3.1 시술 횟수 (권장 세션)

| 목적 | 기본 코스 | 유지 시술 |
|------|----------|----------|
| 색소 개선 | 3~5회 | 3~6개월마다 1회 |
| 홍조 개선 | 5~7회 | 필요시 |
| 피부 재생 | 5~10회 | 2~3개월마다 1회 |
| 제모 | 6~8회 | 1~2년 후 터치업 |

### 3.2 시술 간격

| 일반적 간격 | 3~4주 |
|------------|-------|
| 최소 간격 | 2주 |
| 최대 간격 | 6주 |

### 3.3 다운타임

| 증상 | 기간 | 관리 |
|------|------|------|
| 즉시 홍반 | 3~4시간 | 쿨링 |
| 일시적 홍조 | 24~48시간 | 진정 관리 |
| 색소 진해짐 | 3~7일 | 자연 탈락 대기 |
| 미세 각질 | 5~7일 | 자연 탈락 |

> **일상생활**: 시술 직후 가능 (화장, 세안 OK)

### 3.4 주의사항

#### 시술 전 주의사항

```
□ 2~4주 전부터 자외선 노출 자제 (선탠 금지)
□ 2~4주 전부터 셀프 태닝 제품 사용 금지
□ 광과민성 유발 약물 복용 확인 (항생제, 레티노이드 등)
□ 최근 필러, 보톡스 시술 여부 확인
□ 피부염, 상처 유무 확인
```

#### 시술 후 주의사항

```
□ 2주간 철저한 자외선 차단 (SPF30+ 필수)
□ 뜨거운 사우나, 찜질방 5일간 금지
□ 격렬한 운동 5일간 자제
□ 스크럽, 필링 제품 2주간 금지
□ 충분한 보습 관리
□ 알코올, 레티놀 제품 1주간 자제
```

#### 금기사항 (Contraindications)

| 절대적 금기 | 상대적 금기 |
|------------|------------|
| 피부암/전암 병변 | 어두운 피부 (피츠패트릭 V-VI) |
| 임신/수유 중 | 피츠패트릭 IV (주의) |
| 광과민성 질환 | 아토피, 습진 활성기 |
| 간질 병력 | 켈로이드 체질 |
| 최근 태닝/선번 | 최근 필러/보톡스 |
| 아큐탄 복용 중 | 특정 항생제 복용 |
| 출혈성 질환 | 자가면역 질환 |
| 헤르페스 활성기 | 당뇨 (조절 안 됨) |

## 5. 이룸 앱 연동

### 5.1 피부 타입별 적합도

#### 피츠패트릭 스케일 기준

| 피부 타입 | 적합도 | 권장 여부 | 주의사항 |
|----------|--------|----------|----------|
| Type I-II (밝은 피부) | ⭐⭐⭐⭐⭐ | 강력 추천 | - |
| Type III (보통) | ⭐⭐⭐⭐ | 추천 | 에너지 조절 필요 |
| Type IV (약간 어두움) | ⭐⭐⭐ | 조건부 추천 | 패치 테스트 필수 |
| Type V (어두운 피부) | ⭐⭐ | 주의 필요 | 화상/색소침착 위험 |
| Type VI (매우 어두움) | ⭐ | 비추천 | 대체 시술 권장 |

#### 피부 상태별 적합도

| 피부 상태 | 적합도 | 비고 |
|----------|--------|------|
| **건성 피부** | ⭐⭐⭐⭐ | 시술 후 보습 강화 필요 |
| **지성 피부** | ⭐⭐⭐⭐⭐ | 모공, 피지 개선에 효과적 |
| **복합성 피부** | ⭐⭐⭐⭐ | 부위별 에너지 조절 |
| **민감성 피부** | ⭐⭐⭐ | 저출력으로 시작, 패치 테스트 |
| **여드름성 피부** | ⭐⭐⭐⭐ | 염증 진정 후 시술 권장 |

### 5.2 AI 추천 시 고려사항

#### 추천 알고리즘 가중치

```typescript
interface IPLRecommendation {
  // 필수 고려 요소
  skinType: FitzpatrickType;          // 피츠패트릭 타입 (1-6)
  primaryConcern: SkinConcern[];      // 주요 피부 고민
  recentTanning: boolean;             // 최근 태닝 여부

  // 적합도 점수 계산
  suitabilityScore: number;           // 0-100

  // 금기 사항 체크
  contraindications: string[];        // 금기 사항 목록

  // 대안 시술
  alternatives: Treatment[];          // 부적합 시 대안
}

// 추천 로직 예시
function calculateIPLSuitability(profile: UserProfile): IPLRecommendation {
  let score = 100;

  // 피츠패트릭 타입에 따른 감점
  if (profile.fitzpatrick >= 4) score -= 20;
  if (profile.fitzpatrick >= 5) score -= 30;
  if (profile.fitzpatrick === 6) score -= 50;

  // 최근 태닝 시 대폭 감점
  if (profile.recentTanning) score -= 40;

  // 민감성 피부 감점
  if (profile.skinSensitivity === 'high') score -= 15;

  // 적응증 매칭 시 가점
  if (profile.concerns.includes('pigmentation')) score += 10;
  if (profile.concerns.includes('redness')) score += 10;

  return {
    suitabilityScore: Math.max(0, Math.min(100, score)),
    // ...
  };
}
```

#### 추천 조건

| 조건 | 점수 가중치 | 설명 |
|------|------------|------|
| 주근깨/잡티 고민 | +15 | 높은 효과 기대 |
| 홍조/혈관 고민 | +12 | 효과적 적응증 |
| 밝은 피부톤 | +10 | 안전성 높음 |
| 톤 개선 원함 | +8 | 부가 효과 |
| 어두운 피부톤 | -20 | 위험 증가 |
| 최근 야외활동 多 | -15 | 태닝 우려 |
| 민감성 피부 | -10 | 주의 필요 |

#### 비추천 조건 (하드 필터)

```
- 피츠패트릭 VI → IPL 추천 제외
- 최근 2주 내 강한 자외선 노출 → 추천 보류
- 광과민성 약물 복용 중 → 추천 제외
- 활성 피부염/상처 → 추천 제외
```

### 5.3 연관 시술

#### 시너지 효과 있는 병합 시술

| 연관 시술 | 시너지 효과 | 병합 순서 |
|----------|------------|----------|
| **레이저토닝** | 진피성 색소 동시 개선 | 토닝 → IPL 또는 동일 세션 |
| **제네시스** | 콜라겐 자극 시너지 | 동일 세션 가능 |
| **아쿠아필** | 노폐물 제거 후 흡수율 증가 | 아쿠아필 → IPL |
| **물광주사** | 보습+재생 시너지 | IPL → 물광 (2주 후) |

#### 순차 시술 권장 조합

| 조합 | 목적 | 간격 |
|------|------|------|
| IPL + 레이저토닝 | 색소 종합 개선 | 동시 or 2주 간격 |
| IPL + 프락셀 | 피부결+색소 개선 | 2~4주 간격 |
| IPL + 리프팅 (슈링크 등) | 톤+탄력 개선 | 2~4주 간격 |

#### 주의가 필요한 조합

| 조합 | 주의사항 |
|------|----------|
| IPL + 화학 필링 | 2주 이상 간격 필요 |
| IPL + 레티놀 | 시술 전후 1주 중단 |
| IPL + 필러/보톡스 | 2주 이상 간격 권장 |
