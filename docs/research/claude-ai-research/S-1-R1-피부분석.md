# S-1 피부 분석 종합 리서치

> 이룸 S-1 (피부 분석) 모듈의 과학적 기반 및 구현 원리

---

## 요약

피부 분석(S-1)은 **이미지 기반 AI 분석**을 통해 피부 타입, 수분/유분 밸런스, 모공 상태, 붉은기/트러블 등을 종합 평가한다. 본 리서치는 피부의 **해부학적 구조**, **생리학적 메커니즘**, **측정 지표**, **한국인 특화 데이터**, **이미지 처리 기법**을 체계적으로 정리하여 원리 문서(skin-physiology.md)와 ADR, 스펙의 기반을 제공한다.

**핵심 결론**:
- 피부 타입 분류는 **T존/U존 피지 분비량 비교**가 핵심 (복합성 판정)
- 한국인은 **가장 작은 모공 면적**(0.03-0.06 mm²)과 **높은 피부 장벽 기능**(낮은 TEWL)
- **Lab 색공간 a* 값**이 붉은기 정량화의 표준
- 측정값은 **화장품 선택 참고용**이며 의료 진단이 아님

## 1. 피부 구조

### 1.1 피부 층 구조 (Skin Layers)

피부는 **3개의 주요 층**으로 구성되며, 각 층은 서로 다른 기능을 담당한다.

```
┌─────────────────────────────────────────────────────────────────┐
│                          피부 층 구조                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  표피 (Epidermis) - 두께: 0.05-1.5mm                            │
│  ├── 각질층 (Stratum Corneum)      ← 수분 장벽, 방어막           │
│  │   └── 10-30개 세포층, 두께 10-15μm                           │
│  │   └── 각질세포(Corneocyte) + 지질 매트릭스                   │
│  ├── 투명층 (Stratum Lucidum)      ← 손바닥/발바닥에만 존재      │
│  ├── 과립층 (Stratum Granulosum)   ← 지질 합성, 2-3개 세포층     │
│  ├── 유극층 (Stratum Spinosum)     ← 면역세포 분포, 8-10개 층    │
│  └── 기저층 (Stratum Basale)       ← 세포 분열, 멜라닌 생성      │
│      └── 기저 세포 + 멜라노사이트(Melanocyte)                    │
│                                                                  │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  진피 (Dermis) - 두께: 0.5-5mm                                   │
│  ├── 유두층 (Papillary Dermis)     ← 모세혈관, 감각 수용체       │
│  │   └── 느슨한 결합조직, 표피 영양 공급                        │
│  └── 망상층 (Reticular Dermis)     ← 콜라겐/엘라스틴 네트워크    │
│      └── 치밀한 결합조직, 피부 탄력 담당                        │
│      └── 콜라겐 70%, 엘라스틴 2-4%, 프로테오글리칸              │
│                                                                  │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  피하조직 (Hypodermis/Subcutis)                                 │
│  └── 지방세포 + 결합조직                                        │
│  └── 단열, 충격 흡수, 에너지 저장                               │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 주요 부속기관 (Skin Appendages)

| 부속기관 | 위치 | 기능 | 분석 관련성 |
|---------|------|------|------------|
| **피지선 (Sebaceous Gland)** | 진피 내, 모낭 연결 | 피지 분비, 피부 보호 | 피부 타입 결정 핵심 |
| **땀샘 - 에크린 (Eccrine)** | 전신 분포 | 체온 조절 | 수분 밸런스 |
| **땀샘 - 아포크린 (Apocrine)** | 겨드랑이, 사타구니 | 냄새 분비 | 분석 대상 아님 |
| **모낭 (Hair Follicle)** | 진피~피하조직 | 털 성장 | 모공 크기 측정 |
| **모공 (Pore)** | 표피 표면 | 피지/땀 배출 | 텍스처 분석 핵심 |

### 1.3 피지선 분포 밀도

피지선 밀도는 **얼굴 부위별로 크게 다르며**, 이것이 T존/U존 개념의 생리학적 기반이다.

| 부위 | 피지선 밀도 (glands/cm²) | 모낭피지단위 (/cm²) | 특징 |
|------|-------------------------|-------------------|------|
| **이마** | 400-900 | 200-300 | 최다 밀집, 번들거림 |
| **코** | 높음 (변동적) | 높음 | 블랙헤드 호발부위 |
| **턱** | 중간 | 중간 | 호르몬 영향 큼 |
| **볼** | 낮음 | 적음 | 건조 경향 |
| **눈 주위** | 매우 낮음 | 최소 | 민감, 주름 발생 |

### 1.4 표피 턴오버 (Epidermal Turnover)

```
기저층 세포 분열
       ↓
   14-21일 (이동)
       ↓
각질층 도달 (각화 완료)
       ↓
   14일 (탈락 대기)
       ↓
자연 탈락 (Desquamation)

총 턴오버 주기: 28-40일 (연령에 따라 증가)
```

| 연령대 | 턴오버 주기 | 특징 |
|--------|-----------|------|
| 10-20대 | 28일 | 빠른 재생, 맑은 피부 |
| 30-40대 | 35-40일 | 점진적 둔화 |
| 50대+ | 45-60일 | 각질 축적, 칙칙함 |

## 2. 피부 타입 분류

### 2.1 기본 피부 타입 (4분류)

피부 타입은 **피지 분비량**과 **수분 함량**의 조합으로 분류된다.

| 타입 | 영문 | 피지 | 수분 | 모공 | 특징 |
|------|------|------|------|------|------|
| **건성** | Dry | 낮음 | 낮음 | 작음 | 당김, 각질, 잔주름 |
| **지성** | Oily | 높음 | 다양 | 큼 | 번들거림, 여드름 경향 |
| **복합성** | Combination | T존 높음, U존 낮음 | 불균형 | T존 큼 | 부위별 차이 |
| **중성** | Normal | 적정 | 적정 | 보통 | 밸런스 양호 |

### 2.2 추가 분류: 민감성 피부

민감성(Sensitive)은 **피부 타입 위에 중첩**되는 특성이다.

| 지표 | 민감성 기준 | 측정 방법 |
|------|-----------|----------|
| **TEWL** | > 18.0 g/m²/h | 경피수분손실 측정 |
| **pH** | > 5.45 | pH 미터 |
| **반응성** | 자극 물질 반응 | 패치 테스트 |
| **붉은기** | a* > 15 | Lab 색공간 분석 |

## 3. T-zone/U-zone 분석 원리

### 3.1 영역 정의

T존과 U존은 **피지선 밀도와 분비 패턴**에 따라 구분된다.

```
┌─────────────────────────────────────────────────────────────────┐
│                      얼굴 영역 구분도                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│           ┌─────────────────────────────┐                       │
│           │       이마 (Forehead)        │  ← T존 (상부)         │
│           │     피지선 400-900/cm²       │                       │
│           └─────────────────────────────┘                       │
│                        │                                         │
│        ┌───────────────┼───────────────┐                        │
│        │               │               │                        │
│        │   볼 (왼쪽)    │   코 (Nose)   │   볼 (오른쪽)          │
│        │    U존        │    T존        │    U존                 │
│        │   건조 경향    │ 번들거림 경향  │   건조 경향            │
│        │               │               │                        │
│        └───────────────┼───────────────┘                        │
│                        │                                         │
│           ┌─────────────────────────────┐                       │
│           │        턱 (Chin)            │  ← T존 (하부)         │
│           │      호르몬 영향 큼         │                       │
│           └─────────────────────────────┘                       │
│                                                                  │
│   T존 = 이마 + 코 + 턱 (알파벳 T 형태)                          │
│   U존 = 양쪽 볼 (알파벳 U 형태로 감싸는 영역)                    │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### 3.2 T존 vs U존 상세 비교

| 특성 | T존 (이마/코/턱) | U존 (양 볼) |
|------|-----------------|------------|
| **피지선 밀도** | 400-900 glands/cm² | 낮음 (정확한 데이터 제한적) |
| **모낭피지단위** | 200-300/cm² | 적음 |
| **피지 분비량** | 높음, 변동성 큼 | 낮음, 안정적 |
| **모공 크기** | 크고 가시적 | 작고 덜 가시적 |
| **TEWL** | 상대적으로 낮음 (피지막) | 상대적으로 높음 |
| **주요 문제** | 번들거림, 여드름, 블랙헤드 | 건조, 당김, 민감 |
| **케어 방향** | 피지 조절, 클렌징 | 보습, 진정 |

### 3.3 피지 분비량 기준 (Sebumeter, μg/cm²)

**Sebumeter**는 피부 표면 피지량을 측정하는 표준 장비다.

| 부위 | 건성 | 정상 | 지성 |
|------|------|------|------|
| **이마** | < 70 | 70-150 | > 150 |
| **볼** | < 30 | 30-70 | > 70 |
| **턱** | < 40 | 40-100 | > 100 |
| **코** | 변동적 | 변동적 | 변동적 |

**피지 분비 속도 기준**:
- 건성: < 0.5-1 μg/cm²/min
- 정상: 1-2 μg/cm²/min
- 지성: > 2-4 μg/cm²/min

### 3.4 복합성 피부 판정 알고리즘

```typescript
/**
 * 복합성 피부 판정 원리
 *
 * 핵심: T존과 U존의 피지 분비량 차이가 일정 수준 이상이면 복합성
 *
 * 1. T존 피지 측정 → T존 타입 분류 (건성/정상/지성)
 * 2. U존 피지 측정 → U존 타입 분류 (건성/정상/지성)
 * 3. T존 타입 ≠ U존 타입 → 복합성
 * 4. T존 타입 = U존 타입 → 해당 타입
 */

type SkinType = 'dry' | 'normal' | 'oily' | 'combination';

interface ZoneAnalysis {
  zone: 'T' | 'U';
  sebumLevel: number;  // μg/cm²
  classification: 'dry' | 'normal' | 'oily';
}

function classifyZone(sebum: number, zone: 'T' | 'U'): 'dry' | 'normal' | 'oily' {
  // 부위별 다른 기준 적용
  const thresholds = zone === 'T'
    ? { low: 70, high: 150 }   // T존: 피지 분비 많음
    : { low: 30, high: 70 };   // U존: 피지 분비 적음

  if (sebum < thresholds.low) return 'dry';
  if (sebum > thresholds.high) return 'oily';
  return 'normal';
}

function determineSkinType(tZone: ZoneAnalysis, uZone: ZoneAnalysis): SkinType {
  const tType = classifyZone(tZone.sebumLevel, 'T');
  const uType = classifyZone(uZone.sebumLevel, 'U');

  // T존과 U존 분류가 다르면 복합성
  if (tType !== uType) {
    return 'combination';
  }

  // 같으면 해당 타입
  return tType;
}

// 예시:
// T존 120 μg/cm² (정상), U존 25 μg/cm² (건성) → 복합성
// T존 180 μg/cm² (지성), U존 80 μg/cm² (지성) → 지성
```

## 4. 피부 측정 지표

### 4.1 수분도 (Hydration)

#### 4.1.1 측정 원리

**Corneometer** 방식: 각질층의 **유전 상수(Dielectric Constant)** 측정
- 물의 유전 상수: ~80
- 건조 각질층: ~2-5
- 전극 간 정전용량 변화 → 수분량 추정

| 측정값 (A.U.) | 수분 상태 | 피부 상태 |
|--------------|----------|----------|
| < 30 | 매우 건조 | 각질, 당김, 가려움 |
| 30-50 | 건조 | 당김감, 칙칙함 |
| 50-70 | 정상 | 촉촉함 유지 |
| > 70 | 충분 | 윤기, 탄력 |

#### 4.1.2 TEWL (경피수분손실)

**TEWL**(Trans-Epidermal Water Loss)은 **피부 장벽 기능**의 핵심 지표다.

```
TEWL 측정 원리:
피부 표면 → 증발 수분 → 센서 감지

TEWL = 수분 flux (g/m²/h)
```

| TEWL (g/m²/h) | 장벽 기능 | 의미 |
|---------------|----------|------|
| < 10 | 우수 | 건강한 장벽 |
| 10-18 | 정상 | 양호 |
| 18-25 | 저하 | 민감성 경향 |
| > 25 | 손상 | 장벽 손상, 민감성 |

### 4.2 유분도 (Sebum Level)

#### 4.2.1 측정 원리

**Sebumeter** 방식: 특수 테이프의 **광투과율 변화** 측정
- 피지가 테이프에 흡착 → 투명도 증가 → 광투과율 변화

| 측정값 (μg/cm²) | 유분 상태 | 피부 상태 |
|----------------|----------|----------|
| < 50 | 건조 | 당김, 각질 |
| 50-100 | 정상 | 밸런스 양호 |
| 100-150 | 약간 지성 | 낮 동안 번들거림 |
| > 150 | 지성 | 번들거림, 여드름 경향 |

### 4.3 탄력 (Elasticity)

#### 4.3.1 Cutometer 측정

**Cutometer**는 **음압 흡입**으로 피부 탄력을 측정한다.

```
측정 과정:
1. 음압 적용 (450 mbar)
2. 피부 흡입 (변형)
3. 음압 해제
4. 피부 복원 측정

주요 파라미터:
- Ue: 즉각적 변형량 (표피)
- Uf: 최종 변형량 (표피+진피)
- Ur: 즉각적 회복량
- Ua: 총 회복량
```

| 파라미터 | 공식 | 의미 | 정상 범위 |
|---------|------|------|----------|
| **R2** (Ua/Uf) | 총회복/총변형 | 전체 탄력 | 0.70-0.90 |
| **R5** (Ur/Ue) | 순탄성 | 즉각 회복력 | 0.80-1.00 |
| **R7** (Ur/Uf) | 회복 비율 | 탄성 비율 | 0.40-0.65 |

| R2 값 | 탄력 상태 | 연령대 |
|-------|----------|-------|
| > 0.85 | 매우 좋음 | 20대 |
| 0.70-0.85 | 양호 | 30대 |
| 0.55-0.70 | 보통 | 40대 |
| < 0.55 | 저하 | 50대+ |

### 4.4 모공 (Pore)

#### 4.4.1 모공 측정 메트릭

| 메트릭 | 공식 | 의미 |
|--------|------|------|
| **면적 (Area)** | Σ pixels in contour | 개별 모공 크기 |
| **등가 직경** | √(4 × Area / π) | 원형 환산 직경 |
| **원형도** | 4π × Area / Perimeter² | 1.0=완벽한 원 |
| **밀도** | N / ROI Area | 단위 면적당 개수 |
| **가시성 지수 (PVI)** | Σ(Area × Contrast) / Total × 100 | 눈에 띄는 정도 |

#### 4.4.2 모공 크기 분류

| 분류 | 직경 (μm) | 면적 (mm²) | 특징 |
|------|----------|-----------|------|
| 작은 모공 | < 40 | < 0.05 | 피부 결 고움 |
| 보통 모공 | 40-70 | 0.05-0.10 | 정상 범위 |
| 큰 모공 | > 70 | 0.10-0.20 | 가시적 |
| 확대된 모공 | - | > 0.20 | 눈에 띔 |

### 4.5 표면 거칠기 (Roughness)

#### 4.5.1 ISO 4287 표준 파라미터

| 파라미터 | 공식 | 정상값 | 의미 |
|---------|------|--------|------|
| **Ra** | (1/n)Σ\|Zi\| | 13-35 μm | 산술 평균 거칠기 |
| **Rq** | √[(1/n)ΣZi²] | Ra×1.11 | RMS 거칠기 |
| **Rz** | (1/5)Σ(Rpi+Rvi) | 85-100 μm | 최대 높이 |

#### 4.5.2 거칠기와 피부 상태

| Ra 값 (μm) | 상태 | 관련 요인 |
|-----------|------|----------|
| < 20 | 매끄러움 | 충분한 수분, 젊은 피부 |
| 20-35 | 정상 | 건강한 피부 |
| 35-50 | 거침 | 건조, 경미한 노화 |
| > 50 | 매우 거침 | 심한 건조, 노화 |

### 4.6 붉은기 (Redness)

#### 4.6.1 Lab 색공간 a* 값

**Lab 색공간**의 **a* 축**은 Green(-) ↔ Red(+)를 나타낸다.

| a* 값 | 붉은기 수준 | 상태 |
|-------|-----------|------|
| < 8 | 없음/낮음 | 정상 |
| 8-12 | 경미 | 민감 징후 |
| 12-18 | 중간 | 눈에 띄는 붉음 |
| 18-25 | 높음 | 명백한 홍조 |
| > 25 | 심각 | 강한 염증 |

#### 4.6.2 영역별 정상 a* 값

| 영역 | 정상 a* | 주의 a* |
|------|---------|---------|

  // 1. RGB → Lab 변환
  const labImage = rgbToLab(rgbImage);

  // 2. a* 채널 추출 (인덱스 1)
  const aStarChannel = labImage.map(pixel => pixel[1]);

  // 3. 영역별 평균 계산
  const avgAStar = average(aStarChannel);

  // 4. 붉은기 수준 판정
  let rednessLevel: string;
  if (avgAStar < 8) rednessLevel = 'none';
  else if (avgAStar < 12) rednessLevel = 'mild';
  else if (avgAStar < 18) rednessLevel = 'moderate';
  else if (avgAStar < 25) rednessLevel = 'high';
  else rednessLevel = 'severe';

  return { avgAStar, rednessLevel };
}
