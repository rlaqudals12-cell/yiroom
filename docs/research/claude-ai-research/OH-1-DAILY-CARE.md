# 파트 C: 일상 관리 (★★★ 최심층)

## C-1. 올바른 칫솔질

칫솔질은 구강건강의 기본이지만, **회전법(롤링법)의 플라크 제거 효과가 가장 낮다**는 최신 연구 결과가 주목됩니다. 서울대 조현재 교수팀(2019)에 따르면 회전법의 세균막 지수는 2.44로 칫솔질을 하지 않은 경우(2.52)와 큰 차이가 없었으며, **변형 바스법이 특히 설면(혀쪽)에서 일반 칫솔질 대비 2.9배 더 효과적**이었습니다.

### 칫솔질 기법별 상세 비교

| 기법 | 각도/동작 | 적합 대상 | 효과 근거 |
|------|----------|----------|----------|
| **바스법** | 45도 각도, 치은열구 진입, 진동 동작 | 치주질환자, 잇몸 염증 | Gold standard |
| **변형 바스법** | 바스법 + 롤링 추가 | 일반 성인 | 치경부 플라크 제거 최고 |
| **롤링법** | 잇몸→치아 방향 회전 | 교육용(초보자) | 효과 낮음 (연구 근거) |
| **스틸만법** | 치아-잇몸 경계면 문지르기 | 특수 상황 | 잇몸 손상 위험 |
| **차터스법** | 교합면 방향 45도 진동 | 치주수술 후 | 치간 유두 마사지 |
| **폰즈법** | 원 그리기 동작 | 어린이 | 습득 용이, 효과 중간 |

### 전동칫솔 vs 수동칫솔 (Cochrane Review 2014, 56개 RCT)

전동칫솔의 **회전-진동(Oscillating-Rotating)** 방식만 통계적으로 유의미한 우위가 입증되었습니다.

| 지표 | 단기(1-3개월) | 장기(3개월+) |
|------|-------------|-------------|
| **플라크 감소** | 11% | **21%** |
| **치은염 감소** | 6% | **11%** |

교정 환자의 경우 전동칫솔이 **51.6% 더 우수한 플라크 제거** 효과를 보였습니다.

### 칫솔 선택 기준 및 권장사항

**강모 경도**: ADA는 **Soft 권장**을 공식 입장으로 하고 있습니다. Hard 칫솔은 치은 병변(gingival lesions)을 유의미하게 증가시키며, 과도한 압력 시 법랑질 마모와 잇몸 퇴축 위험이 있습니다.

| 항목 | 권장 수치 | 근거 |
|------|----------|------|
| 칫솔질 시간 | **2분** | 45초 대비 26% 더 제거 |
| 칫솔질 횟수 | 하루 2회 | ADA 권장 |
| 칫솔 교체 주기 | **3개월** | 40일 후 효율 감소 시작 |
| 식후 대기 | 산성 음식 후 **30분** | 법랑질 연화 보호 |

### Yiroom 칫솔 추천 알고리즘 로직

```
입력: 잇몸 상태, 손 기능, 연령대, 치아 배열

IF 잇몸상태 == "치주염/수술 후":
    칫솔 = "Ultra-Soft" + 전동칫솔(회전-진동형)
    기법 = "바스법"
    
ELIF 잇몸상태 == "치은염":
    칫솔 = "Soft" + 전동칫솔 권장
    기법 = "변형 바스법"
    
ELIF 손기능 == "감소" (노인/관절염):
    칫솔 = "전동칫솔 필수"
    
ELIF 연령대 == "소아":
    칫솔 = "Soft 소형 헤드"
    기법 = "폰즈법 → 점진적 전환"
    
ELSE: // 건강한 성인
    칫솔 = "Soft"
    기법 = "변형 바스법"
```

**UI용 한 줄 요약**: "45도 각도, 진동으로 잇몸 청결" (14자)

## C-2. 치실 사용법

치실은 **치간칫솔보다 모든 치간 공간에 사용 가능**하다는 장점이 있지만, Cochrane Review에 따르면 **플라크 감소 근거는 약하고 신뢰도가 낮습니다**. 다만 치은염 감소에는 중간 수준의 근거가 있습니다.

### 치실 종류별 특성

| 종류 | 특징 | 적합 대상 |
|------|------|----------|
| **왁스 치실** | 두꺼움, 잘 끊어지지 않음 | 밀집 치아, 교정 환자 |
| **무왁스 치실** | 좁은 공간 진입 용이 | 촘촘한 치아 |
| **테이프형** | 넓고 평평함 | 넓은 치간 공간 |
| **PTFE 치실** | 가장 매끄러움 | 밀집 치아, 보철물 |
| **슈퍼플로스** | 3부분 구조 | 교정기, 브릿지, 임플란트 |

### 올바른 C자형 사용법

1. 치실 45cm 준비, 양 손 중지에 감기
2. 2-3cm 간격 유지, 치아 사이로 부드럽게 삽입
3. **C자로 치아 측면 감싸기**
4. 잇몸선 아래까지 내려간 후 **위아래 5-6회** 긁어 올림
5. 한 치간 공간에서 **양쪽 치아 모두** 청소

**사용 빈도**: 하루 1회 (ADA 권장), 칫솔질 전 사용이 불소 침투에 유리

### Yiroom 치간 제품 추천 알고리즘

```
IF 치간공간 == "좁음"(<0.6mm) AND 보철물 없음:
    추천 = "무왁스/PTFE 치실"
    
ELIF 치간공간 == "넓음"(>1.0mm):
    추천 = "치간칫솔" (사이즈 매칭)
    
IF 보철물 == "교정기/브릿지":
    필수 = "슈퍼플로스 + 워터픽"
    
IF 보철물 == "임플란트":
    추천 = "워터픽 or 치간칫솔(코팅 와이어)"
    
IF 손기능 == "감소":
    추천 = "워터픽" or "치실 홀더"
```

**UI용 한 줄 요약**: "C자로 감싸고 상하로 긁기" (13자)

## C-3. 치간칫솔

치간칫솔은 **치실보다 플라크 제거에 더 효과적**입니다. 체계적 문헌고찰에서 플라크 감소 WMD 0.13, 출혈 감소 MD 0.08로 치간칫솔이 우위를 보였으며, 환자 선호도도 치간칫솔이 높았습니다. 유럽치주학회(EFP) 2015 합의문에서 "치간칫솔이 가장 효과적인 치간 플라크 제거 방법"으로 인정되었습니다.

### ISO 사이즈 분류표 (ISO 16409:2016)

| ISO | PHD(mm) | 컬러코드 | 권장 치간 |
|-----|---------|---------|----------|
| 0 | ≤0.6 | 핑크 | 매우 좁음 |
| 1 | 0.7-0.8 | 오렌지 | 좁음 |
| 2 | 0.9-1.0 | 레드 | 좁음~보통 |
| 3 | 1.1-1.2 | 블루 | 보통 |
| 4 | 1.3-1.5 | 옐로우 | 보통~넓음 |
| 5 | 1.6-1.8 | 그린 | 넓음 |
| 6-8 | 1.9+ | 퍼플/그레이/블랙 | 매우 넓음 |

### 사용법 핵심

- **삽입 각도**: 전치부 90°, 구치부 45-90°
- **왕복 횟수**: 각 치간당 2-3회 부드럽게
- **교체 주기**: 1-2주마다 또는 모가 벌어지면
- **칫솔질만 대비 추가 플라크 제거율**: **40%**

### Yiroom 사이즈 추천 알고리즘

```
IF 치주질환 OR 치은퇴축:
    시작 = ISO 3-5 (넓은 공간)
ELIF 교정 치료 중:
    시작 = ISO 2-4 (다양하게)
ELIF 건강한 치주, 젊은 성인:
    시작 = ISO 0-2 (좁은 공간)
    
→ "가장 작은 사이즈부터 시작, 저항 없이 들어가는 가장 큰 사이즈 선택"
```

**UI용 한 줄 요약**: "치간 공간에 맞는 사이즈로 매일" (15자)

## C-4. 구강청결제 종류

구강청결제는 성분에 따라 효과와 부작용이 크게 다릅니다. **클로르헥시딘(CHX)이 Gold Standard**이지만 착색과 미각 변화로 2-4주 단기 사용이 권장됩니다.

### 성분별 상세 분류

| 성분 | 농도 | 플라크 감소 | 치은염 감소 | 권장 사용기간 | 주요 부작용 |
|------|------|------------|------------|-------------|-------------|
| **CHX** | 0.12% | 15-20% | 15-20% | **2-4주** | 착색, 미각변화 |
| **CHX** | 0.2% | 20-30% | 20-25% | **2-4주** | 착색 심함 |
| **CPC** | 0.05-0.07% | 10-15% | 10-15% | 장기 가능 | 경미 |
| **에센셜 오일** | (리스테린) | **37%** | **27%** | 장기 가능 | 작열감(알코올) |
| **불소** | 0.05% | 간접 | - | 장기 가능 | 거의 없음 |
| **자일리톨** | 5-10% | 간접 | - | 장기 가능 | 과량 시 설사 |

### 알코올 vs 알코올프리 영향

2024년 Journal of Medical Microbiology 연구에 따르면, 알코올 기반 리스테린 3개월 사용 후 **Fusobacterium nucleatum**(치주병, 대장암 연관)이 증가하고 **Actinobacteria**(혈압 조절 관련 유익균)가 감소했습니다. 구강건조증 환자는 **알코올프리 제품** 필수입니다.

### 사용 타이밍 권장

**핵심 원칙**: 불소 치약 효과를 씻어내지 않도록 **칫솔질 직후 사용 피하기** (NHS 영국 권장)

### Yiroom 구강청결제 추천 알고리즘

```
IF 치주수술 후 OR 급성 치은염:
    추천 = "CHX 0.12%" + 기간 = "2-4주"
    
ELIF 만성 치은염 OR 일상 관리:
    IF 구강건조증 없음:
        추천 = "CPC 0.05%" OR "에센셜 오일"
    ELSE:
        추천 = "CPC 알코올프리"
        
ELIF 충치 고위험 OR 교정 치료:
    추천 = "불소 0.05%" + "자일리톨"
    
ELIF 구취 관리:
    추천 = "CPC" OR "에센셜 오일" (아연 함유 우선)
    
[제외 조건]
- 구강건조증 → 알코올 제품 제외
- 장기 사용 → CHX 제외
- 착색 민감 → CHX 제외
```

**UI용 한 줄 요약**: "목적에 맞는 성분, 칫솔질 30분 후" (15자)

## C-5. 워터픽 (구강세정기)

워터픽은 **치실보다 잇몸 출혈 감소에 26-145% 더 효과적**이며, 교정/임플란트 환자에게 특히 권장됩니다.

### 학술적 근거 (RCT 연구)

| 연구 | 대상 | 결과 |
|------|------|------|
| Sharma et al. 2008 | 교정환자 105명 | 플라크 감소 치실 대비 **3배**, 칫솔 단독 대비 **5배** |
| Magnuson et al. 2013 | 임플란트 | BOP 감소 워터픽 **81.8%** vs 치실 **33.3%** |
| 2023 RCT (12주) | 일반 성인 | 잇몸지수, 출혈지수 유의미 감소, 구취 감소 |

### 작동 원리 및 권장 설정

- **수압**: 10-160 PSI (민감한 잇몸 10-40, 일반 40-70, 숙련자 70-100)
- **맥동 주파수**: 1,200-1,800회/분 (권장 최소 1,200회)
- **사용 각도**: 잇몸선에 **90도**
- **사용 시간**: 약 1분

### 제품 비교 (한국 시장)

| 유형 | 수조 용량 | 수압 단계 | 가격대 |
|------|----------|----------|--------|
| **고정형** | 600-1,000ml | 3-10단계 | 8-15만원 |
| **휴대형** | 150-350ml | 2-3단계 | 5-10만원 |

### Yiroom 워터픽 추천 알고리즘

```
IF 교정 중 OR 임플란트 OR 브릿지:
    추천등급 = "강력추천"
    메시지 = "보철물 관리에 워터픽 필수!"
    
ELIF 치실사용빈도 < 주3회 OR 손기능제한:
    추천등급 = "추천"
    메시지 = "치실 대안으로 워터픽"
    
ELIF 잇몸출혈 OR 치주질환:
    추천등급 = "추천"
    메시지 = "잇몸 건강 개선에 효과적"
```

**UI용 한 줄 요약**: "치실보다 3배 효과, 교정·임플란트 필수" (17자)

## C-6. 구취 관리

구취의 **약 90%는 구강 내 원인**이며, 그 중 **설태(혀 백태)가 VSC의 67-76%를 생성**합니다.

### 휘발성 황화합물(VSC) 상세

| VSC | 화학식 | 냄새 | 주요 발생원 | 기여도 |
|-----|--------|------|------------|--------|
| **황화수소** | H₂S | 썩은 달걀 | 설태 | **76%** |
| **메틸 머캅탄** | CH₃SH | 양배추 썩은 냄새 | 치주낭 | 32% |
| **디메틸 설파이드** | (CH₃)₂S | 양파 냄새 | 혈액/구강외 | 가장 적음 |

### 전신질환별 특징적 구취

| 질환 | 냄새 특성 |
|------|----------|
| 당뇨병(케톤산증) | 과일향/아세톤 |
| 신부전 | 비린내/암모니아 |
| 간부전 | 곰팡이 냄새 |
| 위식도역류 | 신맛 |

### 자가 진단법

| 방법 | 정확도 | 설명 |
|------|--------|------|
| 스푼 테스트 | 높음 | 혀 뒷부분 긁어 냄새 확인 |
| 치실 테스트 | 높음 | 뒷니 치실질 후 냄새 확인 |
| 손목 핥기 | 중간 | 혀 앞쪽만 반영 |

### 효과적인 구취 관리 성분

| 성분 | VSC 감소 | 작용 |
|------|---------|------|
| **아연(Zn)** | **45%** | Zn²⁺가 S²⁻와 결합 |
| CHX | 장기 효과 | 항균 |
| 이산화염소 | 중화 | 산화 반응 |

**최적 조합**: CHX 0.05% + CPC 0.05% + Zn lactate 0.14%

### Yiroom 구취 자가 체크리스트

```
□ 혀에 하얀/노란 코팅이 있다 → 혀 클리너 추천
□ 잇몸에서 피가 난다 → 치주 관리 제품 추천
□ 입이 자주 마른다 → 인공타액, 자일리톨 껌
□ 위장 불편감이 있다 → 전문의 상담 권장
```
