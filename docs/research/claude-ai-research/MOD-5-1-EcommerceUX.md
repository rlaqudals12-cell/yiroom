# MOD-5-1: E-commerce UX 트렌드

> **리서치 목적**: 이룸 앱 제품 추천 및 커머스 기능에 적용할 최신 UX 트렌드 조사
> **조사 일자**: 2026-01-16
> **조사 범위**: 2025-2026년 e-commerce UX 트렌드

---

## 1. 핵심 요약

- **AI 기반 개인화**: 단순 추천을 넘어 실시간 컨텍스트 인식 개인화가 표준이 됨. 사용자의 현재 상태(시간, 날씨, 최근 행동)를 반영한 동적 추천이 핵심.
- **마이크로 인터랙션**: 스크롤, 필터, 장바구니 추가 등 모든 동작에 즉각적이고 의미 있는 피드백 제공. 지연 시간 100ms 이하가 업계 표준.
- **간소화된 결제**: 원클릭 결제, 게스트 체크아웃 우선, 결제 단계 3단계 이하로 축소. 결제 포기율 최소화가 핵심 KPI.
- **모바일 우선 + 제스처 기반 UI**: 스와이프, 핀치, 롱프레스 등 제스처 인터랙션이 버튼 클릭을 대체. 엄지 도달 영역(Thumb Zone) 최적화 필수.
- **시각적 검색 & AR**: 이미지 기반 검색과 AR 가상 체험이 뷰티/패션 카테고리에서 필수 기능으로 자리잡음.

---

## 2. 상세 내용

### 2.1 브라우징 UX

#### 2.1.1 무한 스크롤 vs 페이지네이션

| 패턴 | 장점 | 단점 | 권장 상황 |
|------|------|------|----------|
| **무한 스크롤** | 몰입감, 발견의 즐거움 | 위치 파악 어려움, 푸터 접근 불가 | 탐색형 브라우징 (SNS, 피드) |
| **페이지네이션** | 명확한 진행 상태, 북마크 가능 | 클릭 필요, 흐름 단절 | 목적형 쇼핑, 검색 결과 |
| **하이브리드** | 양쪽 장점 결합 | 구현 복잡성 | 대부분의 커머스 (권장) |

**2025-2026 트렌드**:
- 초기 20-40개 무한 스크롤
- "더 보기" 버튼 명시적 표시
- 스크롤 위치 기억 (뒤로가기 시)
- URL에 페이지 상태 반영 (SEO + 공유)

#### 2.1.2 그리드 레이아웃

**트렌드**: Spacious + 큰 이미지 + 호버 인터랙션
- Dense Grid (고밀도): 3-4열 (모바일 2열), 가격 중심 쇼핑, 빠른 비교 목적
- Spacious Grid (여백): 2-3열 (모바일 1열), 브랜드/럭셔리, 이미지 강조

#### 2.1.3 퀵뷰 (Quick View)

- **표준 기능화**: 상품 목록에서 전체 페이지 이동 없이 주요 정보 확인
- **구현 패턴**: 호버 시 "빠른 보기" 버튼 표시 (데스크탑), 롱프레스/스와이프 업 (모바일)
- **필수 포함 정보**: 이미지 갤러리, 가격, 옵션 선택, 장바구니 추가

### 2.2 필터/정렬

#### 2.2.1 필터 UI 패턴

| 패턴 | 설명 | 적용 상황 |
|------|------|----------|
| **사이드바 필터** | 좌측 고정, 항상 표시 | 데스크탑, 10개+ 필터 |
| **상단 필터 바** | 수평 배치, 드롭다운 | 5개 이하 핵심 필터 |
| **풀스크린 오버레이** | 모바일 전용 | 복잡한 필터 조합 |
| **스티키 필터 칩** | 적용된 필터 상단 고정 | 필터 상태 명시 |

**모범 사례**:
- 즉시 결과 반영 (필터 선택 즉시 업데이트)
- 결과 수 실시간 표시 ("32개 상품")
- 적용된 필터 칩 표시 + 전체 해제 버튼
- URL에 필터 반영 (공유/북마크)

#### 2.2.2 고급 필터 트렌드

1. **AI 기반 필터 추천**: "당신을 위한 필터 추천" - 퍼스널컬러, 피부 타입, 가격대
2. **시각적 필터**: 색상 스워치, 바디 실루엣, 피부톤 스펙트럼
3. **자연어 필터** (2026 신규): "봄 웜톤에 어울리는 10만원 이하 립스틱" → 자동 필터 적용

#### 2.2.3 정렬 옵션

**표준**: 추천순(기본), 최신순, 가격순, 인기순, 평점순

**이룸 특화**: 피부 적합도순, 퍼스널컬러 매칭순, 성분 안전도순

### 2.3 상세 페이지

#### 2.3.1 레이아웃 구조

```
[갤러리 영역]           [정보 영역]
- 메인 이미지           - 브랜드/상품명
- 썸네일 슬라이더        - 가격 (할인가 강조)
- 줌/360도 뷰           - 별점 + 리뷰 수
- AR 체험 버튼          - 옵션/수량 선택
                       - [장바구니][바로구매]
-------------------------------------------
[AI 분석 배지]
"내 피부에 92% 매칭" | "Spring Warm에 적합"
-------------------------------------------
[탭: 상품정보 | 성분분석 | 리뷰 | Q&A]
```

#### 2.3.2 이미지 갤러리 트렌드

| 기능 | 구현 우선순위 |
|------|-------------|
| 고해상도 줌 | 필수 |
| 360도 뷰 | 권장 |
| AR 가상 체험 (뷰티) | 필수 |
| 비디오 통합 | 권장 |
| UGC 이미지 | 높음 |

#### 2.3.3 신뢰 요소 (Trust Signals)

구매 버튼 근처 배치:
- 무료배송, 교환/반품 정책, 정품 보장
- 리뷰 요약, 베스트셀러 배지
- "지금 N명이 보는 중" (FOMO)

### 2.4 결제 플로우

#### 2.4.1 결제 단계 최적화

**2025-2026 표준 (3단계 이하)**:
1. 장바구니 확인
2. 배송/결제 정보 (한 페이지 통합)
3. 주문 확인 → 완료

#### 2.4.2 장바구니 UX

- **미니 카트**: 호버 프리뷰, 우측 슬라이드 패널, 미니카트에서 바로 결제
- **수량 변경**: 인라인 변경, 즉시 총액 반영, 재고 경고
- **이탈 방지**: 나중에 구매하기, 장바구니 이탈 알림, 가격 인하 알림

#### 2.4.3 결제 폼 최적화

1. 게스트 체크아웃 우선
2. 자동 입력 (주소 완성, 카드 스캔, 이전 정보)
3. 실시간 유효성 검사
4. 간편결제 우선 표시 (카카오페이, 네이버페이)

### 2.5 추천 UX

#### 2.5.1 추천 유형

| 유형 | 배치 위치 |
|------|----------|
| "이 상품을 본 고객이 함께 본" | 상품 상세 |
| "당신을 위한 추천" | 홈, 상세 |
| "지금 인기 있는" | 홈, 카테고리 |
| "함께 구매하면 좋은" | 장바구니 |

#### 2.5.2 이룸 특화 추천

- 매칭 점수 배지: "내 피부에 92% 맞아요"
- 매칭 이유 설명: "Spring Warm에 완벽한 코럴 톤", "지성 피부에 적합한 수분 베이스"
- 경고: 알코올 함유, 인공향료 등

### 2.6 모바일 특화

#### 2.6.1 Thumb Zone 최적화

- 어려움 영역 (상단): 검색, 알림 아이콘
- 쉬움 영역 (하단): 내비게이션, CTA 버튼
- 핵심: 장바구니 추가 하단 고정, 필터 하단 시트

#### 2.6.2 제스처 인터랙션

| 제스처 | 동작 |
|--------|------|
| 스와이프 좌/우 | 이미지 갤러리 탐색 |
| 스와이프 업 | 퀵뷰, 상세 정보 |
| 핀치 | 이미지 확대/축소 |
| 롱프레스 | 위시리스트 추가 |
| 더블탭 | 좋아요, 확대 |

#### 2.6.3 모바일 결제

- 생체인증 (Face ID, 지문)
- 디지털 지갑 우선 (Apple Pay, Google Pay, 카카오페이)
- 숫자 키패드, 자동 필드 이동, 터치 영역 확대

## 3. 이룸 앱 적용 사항

### 3.1 높은 우선순위 (P0)

- [ ] **AI 매칭 배지**: 모든 제품 카드에 "내 피부에 N% 매칭" 표시
- [ ] **퍼스널컬러 필터**: 사용자 진단 결과 기반 자동 필터 제안
- [ ] **성분 안전도 표시**: 피부 타입 기반 성분 경고
- [ ] **간소화된 결제**: 3단계 이하 결제 플로우
- [ ] **하단 고정 CTA**: 장바구니/구매 버튼 Thumb Zone 배치

### 3.2 중간 우선순위 (P1)

- [ ] **AR 립 컬러 테스트**: 퍼스널컬러 기반 가상 체험
- [ ] **하이브리드 스크롤**: 무한 스크롤 + "더 보기" 조합
- [ ] **하단 시트 필터**: 모바일 전용 필터 UI
- [ ] **퀵뷰 모달**: 상품 목록에서 빠른 정보 확인
- [ ] **제스처 인터랙션**: 스와이프 이미지 갤러리

### 3.3 낮은 우선순위 (P2)

- [ ] **자연어 필터**: "봄웜톤 립스틱" 자동 필터 적용
- [ ] **360도 제품 뷰**
- [ ] **PWA 오프라인**: 위시리스트 오프라인 접근
- [ ] **가격 인하 알림**: 위시리스트 푸시 알림
