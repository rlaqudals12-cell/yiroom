## 1. 핵심 요약

이룸의 영양 모듈(N-1)은 사용자의 신체 정보와 활동량을 기반으로 개인화된 영양 목표를 계산하고, 식단 추적을 통해 영양 균형을 분석합니다. 이 리서치는 BMR/TDEE 계산, 한국인 영양섭취기준, 칼로리 추적 방법론의 과학적 근거를 정리합니다.

## 2. 기초대사량(BMR) 계산

### 2.1 Harris-Benedict 공식 (1918, 1984 개정)

**역사적 배경**: 1918년 Harris와 Benedict가 개발한 최초의 BMR 공식. 1984년 Roza와 Shizgal이 현대 인구에 맞게 개정.

**개정 공식 (1984)**:
```
남성: BMR = 88.362 + (13.397 × 체중kg) + (4.799 × 신장cm) - (5.677 × 나이)
여성: BMR = 447.593 + (9.247 × 체중kg) + (3.098 × 신장cm) - (4.330 × 나이)
```

**장점**:
- 가장 널리 사용되는 공식
- 대규모 데이터 기반
- 검증된 역사

**단점**:
- 서양인 기준 (아시아인 과대추정 경향)
- 근육량 미반영
- 20세기 초 데이터 기반

### 2.2 Mifflin-St Jeor 공식 (1990)

**배경**: 현대 인구의 체성분 변화를 반영하여 개발. ADA(미국영양학회)가 권장하는 공식.

**공식**:
```
남성: BMR = (10 × 체중kg) + (6.25 × 신장cm) - (5 × 나이) + 5
여성: BMR = (10 × 체중kg) + (6.25 × 신장cm) - (5 × 나이) - 161
```

**정확도**: Harris-Benedict 대비 약 5% 더 정확 (현대 인구 기준)

**이룸 적용**: 1차 추천 공식으로 사용

### 2.3 Katch-McArdle 공식 (체지방 기반)

**배경**: 제지방량(Lean Body Mass)을 기반으로 계산. 체성분 분석이 가능한 경우 가장 정확.

**공식**:
```
BMR = 370 + (21.6 × LBM)
여기서 LBM = 체중 × (1 - 체지방률)
```

**장점**:
- 근육량 반영
- 운동선수에게 적합

**단점**:
- 체지방률 측정 필요
- 일반 사용자에게 진입장벽

**이룸 적용**: 체지방률 입력 시 2차 검증 공식으로 사용

### 2.4 한국인 보정 계수

**연구 근거**: 2015년 대한비만학회 연구에 따르면, 한국인은 서양인 대비 BMR이 약 5-10% 낮음.

**보정 공식**:
```
한국인 BMR = 기본 BMR × 0.95 (남성)
한국인 BMR = 기본 BMR × 0.93 (여성)
```

**출처**:
- Kim et al., "Resting metabolic rate in Korean adults", Korean J Obes 2015
- 질병관리청 국민건강영양조사 2020-2022

## 3. 총 에너지 소비량(TDEE) 계산

### 3.1 활동 계수 (Activity Multiplier)

| 활동 수준 | 설명 | 계수 | 예시 |
|-----------|------|------|------|
| **좌식** (Sedentary) | 운동 거의 없음, 사무직 | 1.2 | 재택근무, 앉아서 일함 |
| **가벼운 활동** (Light) | 주 1-3회 가벼운 운동 | 1.375 | 가벼운 산책, 요가 |
| **보통 활동** (Moderate) | 주 3-5회 중간 강도 운동 | 1.55 | 조깅, 수영, 자전거 |
| **활발한 활동** (Active) | 주 6-7회 운동 | 1.725 | 헬스장, 스포츠 |
| **매우 활발** (Very Active) | 고강도 운동 또는 육체 노동 | 1.9 | 운동선수, 건설 노동자 |

**TDEE 계산**:
```
TDEE = BMR × 활동 계수
```

### 3.2 운동 칼로리 추가 계산

**MET(Metabolic Equivalent of Task) 기반 계산**:
```
운동 칼로리 = MET × 체중(kg) × 시간(h)
```

| 운동 | MET | 30분 소모량 (70kg 기준) |
|------|-----|------------------------|
| 걷기 (4km/h) | 3.0 | 105 kcal |
| 조깅 (8km/h) | 7.0 | 245 kcal |
| 달리기 (12km/h) | 11.0 | 385 kcal |
| 자전거 (중간 속도) | 6.0 | 210 kcal |
| 수영 (자유형) | 8.0 | 280 kcal |
| 웨이트 트레이닝 | 5.0 | 175 kcal |
| 요가 | 2.5 | 88 kcal |
| HIIT | 12.0 | 420 kcal |

**출처**: Compendium of Physical Activities (2011)

## 5. 칼로리 적자/잉여 계산

### 5.1 체중 변화 원리

**에너지 균형 공식**:
```
체중 변화 = (섭취 칼로리 - 소비 칼로리) / 7,700
```

**1kg 체지방 = 약 7,700 kcal** (지방 9 kcal/g × 85% 순수지방)

### 5.2 안전한 체중 변화 속도

| 목표 | 일일 적자/잉여 | 주간 체중 변화 | 월간 체중 변화 |
|------|--------------|--------------|--------------|
| 느린 감량 | -250 kcal | -0.23 kg | -1 kg |
| 보통 감량 | -500 kcal | -0.45 kg | -2 kg |
| 빠른 감량 | -750 kcal | -0.68 kg | -3 kg |
| 느린 증량 | +250 kcal | +0.23 kg | +1 kg |
| 보통 증량 | +500 kcal | +0.45 kg | +2 kg |

**권장 범위**:
- 최소 섭취: 남성 1,500 kcal, 여성 1,200 kcal
- 최대 적자: 일일 -1,000 kcal
- 최대 잉여: 일일 +750 kcal

### 5.3 적응 대사 (Adaptive Thermogenesis)

**현상**: 장기 칼로리 제한 시 대사율 감소 (10-15%)

**대응 전략**:
1. 다이어트 브레이크 (2주마다 유지 칼로리로 복귀)
2. 리피드 데이 (주 1회 높은 탄수화물 섭취)
3. 점진적 칼로리 조정 (4주마다 재계산)

**이룸 적용**: 4주 주기로 목표 칼로리 자동 재계산 권장

## 7. 식사 시간 및 식사 빈도

### 7.1 식사 타이밍 연구

**간헐적 단식 (Intermittent Fasting) 효과**:
- 16:8 방식: 16시간 공복, 8시간 섭취 창
- 체중 감량에 유의미한 효과 (메타분석, 2020)
- 인슐린 민감성 개선

**식후 대사 반응**:
```
식후 1시간: 혈당 최고점
식후 2-3시간: 인슐린 최고점
식후 4시간: 기저 상태 복귀
```

### 7.2 운동 전후 영양

**운동 전 (1-2시간 전)**:
```
탄수화물: 1-4 g/kg
단백질: 0.15-0.25 g/kg
지방: 최소화
```

**운동 후 (30분-2시간 내)**:
```
단백질: 20-40g (또는 0.25-0.4 g/kg)
탄수화물: 단백질의 2-3배 (근육 회복 시)
```

**이룸 적용**: 운동 기록 연동 시 식사 타이밍 가이드 제공

## 8. 한국인 특화 고려사항

### 8.1 한식 영양 특성

**장점**:
- 다양한 채소 섭취
- 발효식품 (김치, 된장, 청국장)
- 낮은 유제품 섭취 → 칼슘 부족 주의

**단점**:
- 높은 나트륨 섭취 (세계 1-2위)
- 정제 탄수화물 과다 (백미, 라면)
- 과일 섭취 부족

### 8.2 한국인 영양 문제

| 문제 | 현황 | 권장 대응 |
|------|------|----------|
| 나트륨 과다 | 평균 4,000mg/일 (WHO 권장 2배) | 국물 줄이기, 저염 제품 |
| 칼슘 부족 | 권장량 67% 섭취 | 유제품, 뼈째 먹는 생선 |
| 비타민 D 부족 | 성인 72% 결핍 | 보충제 권장 |
| 식이섬유 부족 | 권장량 70% 섭취 | 현미, 채소 증가 |

### 8.3 KDRI 2020 vs 서양 기준

| 영양소 | KDRI 2020 | DRI (미국) | 차이점 |
|--------|-----------|-----------|--------|
| 탄수화물 비율 | 55-65% | 45-65% | 한국 상한 높음 |
| 단백질 (g/kg) | 0.91 | 0.8 | 한국 약간 높음 |
| 나트륨 (mg) | 2,300 상한 | 2,300 상한 | 동일 |
| 칼슘 (mg) | 700-800 | 1,000-1,200 | 한국 낮음 |


### 9.2 영양소 목표 계산

```typescript
interface NutritionGoals {
  calories: number;          // TDEE ± 목표 적자/잉여

  carbs: number;             // (칼로리 × 탄수화물비율) / 4
  fat: number;               // (칼로리 × 지방비율) / 9
  fiber: number;             // 25-38g 고정
  sodium: number;            // 2,300mg 상한
}
```
