## 1. 핵심 요약 (Executive Summary)

### 1.1 연구 목적

퍼스널컬러 분석(PC-1)의 과학적 기반을 정립하기 위한 기초 리서치. 색채학 원리, 퍼스널컬러 이론의 역사와 체계, 한국인 피부톤 특성, 조명 영향 등을 종합적으로 분석한다.

### 1.2 핵심 발견

| 영역 | 핵심 내용 |
|------|----------|
| **색공간 선택** | Lab 색공간이 피부톤 분석에 최적 (인간 지각 기반, 균일 색차) |
| **한국인 특성** | b* 값 높음 (황색 언더톤), 웜톤 비율 52.7% 우세 |
| **계절 분류** | 4계절 → 12톤 → 16톤 시스템으로 발전, 실용성은 12톤이 최적 |
| **조명 영향** | CCT(색온도)에 따라 ±5-15% 판정 오차 발생, Bradford 보정 필수 |
| **학술 근거** | 색채학(Newton, Munsell), 퍼스널컬러(Itten, Caygill, Jackson) |

### 1.3 이룸 적용 권장사항

1. **Lab 색공간 기반 분석**: RGB 대신 Lab으로 변환 후 분석
2. **한국인 특화 임계값**: 웜/쿨 경계 h° = 58-60° (국제 표준 55° 대비 +3-5°)
3. **12톤 시스템 채택**: 4계절의 단순함 + 16톤의 정밀함 균형
4. **조명 보정 필수**: CIE-3 모듈과 연계하여 D65 기준 변환
5. **신뢰도 표시**: 경계 케이스 식별 및 사용자 고지

## 2. 색채학 기초 (Color Science Fundamentals)

### 2.1 색의 물리학적 기초

#### 2.1.1 가시광선 스펙트럼

```
파장(nm)    색상        에너지
────────────────────────────────
380-450     보라        높음
450-495     파랑
495-570     초록
570-590     노랑
590-620     주황
620-750     빨강        낮음
```

**피부색 관련 파장**:
- **멜라닌**: 300-700nm 전체 흡수 (갈색-검은색)
- **헤모글로빈**: 540-577nm 흡수 (붉은색)
- **카로틴**: 400-500nm 흡수 (노란색)

#### 2.1.2 색 인지의 생리학

인간의 눈에는 3종류의 원추세포(Cone cell)가 있다:

| 원추세포 | 민감 파장 | 역할 |
|---------|----------|------|
| S-cone (Short) | ~420nm | 파랑 인지 |
| M-cone (Medium) | ~530nm | 초록 인지 |
| L-cone (Long) | ~560nm | 빨강 인지 |

**색각 이상 고려**:
- 색맹/색약 비율: 남성 8%, 여성 0.5%
- 퍼스널컬러 진단 시 색각 이상 사용자 대응 필요

### 2.2 색공간 비교 분석

#### 2.2.1 주요 색공간 개요

| 색공간 | 차원 | 특성 | 피부톤 분석 적합성 |
|--------|------|------|-------------------|
| **RGB** | R, G, B | 장치 의존적, 가산혼합 | 낮음 |
| **CMYK** | C, M, Y, K | 인쇄용, 감산혼합 | 매우 낮음 |
| **HSV/HSL** | H, S, V/L | 직관적, 비균일 | 중간 |
| **Lab** | L*, a*, b* | 인간 지각 기반, 균일 | **매우 높음** |
| **LCH** | L*, C*, h° | Lab의 극좌표 형태 | 매우 높음 |

#### 2.2.2 Lab 색공간 상세

**역사적 배경**:
- 1931: CIE XYZ 색공간 제정
- 1976: CIE Lab (CIELAB) 표준화
- 목적: 인간의 색 지각과 일치하는 균일 색공간

**축 정의**:
```
L* (Lightness): 0 (검정) ~ 100 (흰색)
    → 밝기/명도 축
    → 피부톤 깊이(depth) 결정

a* (Green-Red): -128 (초록) ~ +127 (빨강)
    → 붉은기 축
    → 언더톤의 핑크/레드 성분

b* (Blue-Yellow): -128 (파랑) ~ +127 (노랑)
    → 노란기 축
    → 언더톤의 골든/옐로우 성분
```

**피부톤 분석에서의 해석**:
```
웜톤 (Warm) = +b* (높은 노란기) + 중간 a*
쿨톤 (Cool) = 낮은 b* + 높은 a* (분홍기)
뉴트럴 = 균형 잡힌 a*, b*
```

#### 2.2.3 Lab vs RGB 비교

**색차 측정 실험** (동일한 "시각적 차이"에 대한 수치 차이):

| 색상 쌍 | RGB 거리 | Lab 거리 (ΔE) |
|---------|---------|---------------|
| 밝은 노랑 - 연한 노랑 | 45 | 12 |
| 진한 파랑 - 중간 파랑 | 40 | 8 |
| 피부톤 A - 피부톤 B | 25 | 5 |

**결론**: Lab의 색차(ΔE)가 인간 지각과 더 일치함

### 2.3 먼셀 컬러 시스템 (Munsell Color System)

#### 2.3.1 역사와 원리

**Albert H. Munsell** (1858-1918)이 1905년에 개발한 색채 시스템으로, 세 가지 속성으로 색상을 표기:

```
HUE VALUE/CHROMA
예: 5YR 6/4

HUE (색상): 10개 기본 색상 (R, YR, Y, GY, G, BG, B, PB, P, RP)
VALUE (명도): 0 (검정) ~ 10 (흰색)
CHROMA (채도): 0 (무채색) ~ 14+ (최고 채도)
```

#### 2.3.2 먼셀과 퍼스널컬러 연결

| 먼셀 속성 | 퍼스널컬러 차원 | 매핑 |
|----------|----------------|------|
| Hue | 온도 (Warm/Cool) | YR, Y = Warm / PB, P = Cool |
| Value | 깊이 (Light/Deep) | 높은 Value = Light |
| Chroma | 선명도 (Bright/Muted) | 높은 Chroma = Bright |

#### 2.3.3 한국인 피부색 먼셀 표기

**코코리색채연구소 데이터** (20-30대 여성 37,000명):
```
웜톤 평균: 8.8YR 6.6/3
쿨톤 평균: 7.7YR 6.5/2.7
```

**해석**:
- 색상(Hue): YR(Yellow-Red) 계열 → 황적색 기조
- 명도(Value): 6.5-6.6 → 중간 밝기
- 채도(Chroma): 2.7-3.0 → 낮은 채도 (뮤트)

## 3. 퍼스널컬러 이론 (Personal Color Theory)

### 3.1 역사적 발전

#### 3.1.1 연대기

| 연도 | 인물/사건 | 기여 |
|------|----------|------|
| 1919 | Robert Dorr | "Golden Key" - 최초의 웜/쿨 구분 |
| 1928 | Johannes Itten | 바우하우스에서 색채 교육, 계절 이론 기초 |
| 1942 | Suzanne Caygill | 계절 시스템 대중화 시작 |
| 1980 | Carole Jackson | "Color Me Beautiful" 출간, 4계절 시스템 확립 |
| 1990s | Mary Spillane | 16시즌 시스템 도입 (영국) |
| 2000s | Kathryn Kalisz | Sci\ART 시스템 (12시즌 정교화) |
| 2010s | 한국 업계 | K-뷰티와 함께 12/16톤 시스템 대중화 |

#### 3.1.2 Johannes Itten의 색채 이론

바우하우스의 색채 교육자 Itten은 학생들의 자연스러운 색상 선호를 관찰하여:

1. **개인의 색상 조화는 타고난 피부톤, 머리카락, 눈 색상과 관련됨**
2. **색상의 온도(따뜻함/차가움)가 개인에게 미치는 영향이 다름**
3. **계절의 색상 팔레트와 개인의 특성이 연결됨**

이 관찰이 현대 퍼스널컬러 이론의 기초가 됨.

### 3.2 4계절 시스템 (Four Season System)

#### 3.2.1 이론적 기반

**Carole Jackson의 "Color Me Beautiful" (1980)**에서 체계화:

```
       높은 명도 (Light)
              ↑
              │
        봄 ────┼──── 여름
              │
웜톤 ←────────┼────────→ 쿨톤
              │
        가을 ───┼──── 겨울
              │
              ↓
       낮은 명도 (Deep)
```

#### 3.2.2 4계절 특성 상세

| 계절 | 온도 | 명도 | 채도 | 피부 특성 | 대표 유명인 |
|------|------|------|------|----------|------------|
| **봄 (Spring)** | Warm | Light | Bright | 아이보리, 피치빛, 주근깨 | 윤아, 김고은 |
| **여름 (Summer)** | Cool | Light | Muted | 핑크빛, 장밋빛, 섬세한 피부 | 수지, 아이유 |
| **가을 (Autumn)** | Warm | Deep | Muted | 황금빛, 올리브빛, 두꺼운 피부 | 전지현, 한예슬 |
| **겨울 (Winter)** | Cool | Deep | Bright | 하얗거나 올리브, 명확한 대비 | 이영애, 김태희 |

#### 3.2.3 4계절 시스템의 한계

| 한계 | 설명 |
|------|------|
| **경계 케이스** | 봄-가을, 여름-겨울 경계에 있는 사람 분류 어려움 |
| **뉴트럴 톤 처리** | 웜/쿨 중간 톤에 대한 체계 부재 |
| **세부 차이 무시** | 같은 "봄"이어도 밝기/채도 차이 존재 |
| **아시아인 적합성** | 서양 중심 개발로 아시아인 피부톤 특성 반영 부족 |

### 3.3 12톤 시스템 (Twelve-Tone System)

#### 3.3.1 발전 배경

4계절의 한계를 극복하기 위해 각 계절을 3개의 세부 톤으로 세분화:

```
12톤 = 4계절 × 3세부톤

세부톤 분류 기준:
- Light: 높은 명도 우선
- True/Warm/Cool: 전형적인 계절 특성
- Bright/Deep/Muted/Soft: 채도 또는 명도 특성
```

#### 3.3.2 12톤 체계 상세

**봄 (Spring) - 웜톤, 밝음**:

| 세부 톤 | 한국명 | 특성 | Lab 기준 |
|---------|--------|------|----------|
| Light Spring | 봄 라이트 | 가장 밝고 부드러움 | L* > 68, C* 중간 |
| True Spring | 봄 트루 | 전형적인 봄, 따뜻하고 선명 | L* 62-68, h° > 62° |
| Bright Spring | 봄 브라이트 | 밝고 선명, 겨울과 인접 | C* > 25, L* 60-70 |

**여름 (Summer) - 쿨톤, 밝음**:

| 세부 톤 | 한국명 | 특성 | Lab 기준 |
|---------|--------|------|----------|
| Light Summer | 여름 라이트 | 가장 밝고 연함 | L* > 66, C* < 18 |
| True Summer | 여름 트루 | 전형적인 여름, 시원하고 부드러움 | L* 60-66, h° < 55° |
| Muted Summer | 여름 뮤트 | 회색빛, 차분함 | C* < 14, 그레이시 |

**가을 (Autumn) - 웜톤, 깊음**:

| 세부 톤 | 한국명 | 특성 | Lab 기준 |
|---------|--------|------|----------|
| Soft Autumn | 가을 뮤트 | 부드럽고 중립적 | C* < 18, L* 55-62 |
| True Autumn | 가을 트루 | 전형적인 가을, 깊고 풍부 | L* 58-65, h° > 60° |
| Deep Autumn | 가을 딥 | 가장 어둡고 진함 | L* < 56, 깊은 색조 |

**겨울 (Winter) - 쿨톤, 깊음**:

| 세부 톤 | 한국명 | 특성 | Lab 기준 |
|---------|--------|------|----------|
| True Winter | 겨울 트루 | 전형적인 겨울, 차갑고 선명 | L* 52-60, h° < 52° |
| Deep Winter | 겨울 딥 | 가장 어둡고 강렬 | L* < 55, 깊은 대비 |
| Bright Winter | 겨울 브라이트 | 밝고 선명, 봄과 인접 | C* > 24, 비비드 |

#### 3.3.3 12톤 시스템의 장단점

**장점**:
- 경계 케이스 대부분 커버
- 실용적인 수준의 세분화
- 아시아 시장에서 널리 채택
- 색상 추천 정확도 향상

**단점**:
- 뉴트럴 톤(Neutral)을 별도 범주로 두지 않음
- 일부 경계 케이스 여전히 존재

### 3.4 16톤 시스템 (Sixteen-Tone System)

#### 3.4.1 구조

```
16톤 = 12톤 + 4 Neutral 톤

추가된 톤:
- Neutral Warm (뉴트럴 웜)
- Neutral Cool (뉴트럴 쿨)
- True Neutral Light (트루 뉴트럴 라이트)
- True Neutral Deep (트루 뉴트럴 딥)
```

#### 3.4.2 이룸에서의 채택 권장

| 시스템 | 장점 | 단점 | 이룸 적합성 |
|--------|------|------|------------|
| 4계절 | 단순, 이해 쉬움 | 정확도 낮음 | 초보자 설명용 |
| **12톤** | **균형 잡힌 정밀도** | **일부 경계 케이스** | **주력 채택** |
| 16톤 | 매우 정밀 | 복잡, 일반인 혼란 | 고급 분석 옵션 |

**권장**: **12톤 시스템을 기본**으로 하되, **뉴트럴 경고 메시지**로 경계 케이스 표시

## 4. 한국인 피부톤 특성 데이터

### 4.1 인구 통계학적 데이터

#### 4.1.1 Lab 측정값 종합

| 출처 | 대상 | n | L* | a* | b* | 비고 |
|------|------|---|----|----|-----|------|
| Puzovic (2012) | 한국인 | 148 | 64.15 | 8.96 | 18.34 | 분광광도계 |
| Son (2013) | 한국인 | 600 | 62.1 | 11.2 | 18.8 | 볼 측정 |
| Korean Skin DB | 한국인 | - | 61.75 | 9.56 | 17.08 | 국가 DB |
| 코코리연구소 | 한국 여성 60대 | - | 62.21 | 9.49 | 19.81 | 특허 데이터 |

**이룸 권장 기준값** (20-40대 기준 추정):
```
L* = 63.0 ± 5.0
a* = 10.0 ± 3.0
b* = 18.5 ± 3.0
```

#### 4.1.2 Fitzpatrick 피부 타입 분포

| 타입 | 설명 | 한국인 비율 | L* 범위 | ITA 범위 |
|------|------|------------|---------|----------|
| I | 매우 밝음, 쉽게 화상 | ~2% | > 73 | > 55° |
| II | 밝음, 쉽게 화상 | ~10% | 67-73 | 41-55° |
| **III** | 중간, 적당히 태닝 | **~60%** | 60-67 | 28-41° |
| IV | 올리브, 잘 태닝 | ~25% | 53-60 | 10-28° |
| V | 갈색 | ~3% | 45-53 | -30-10° |
| VI | 검은색 | < 1% | < 45 | < -30° |

**핵심**: 한국인 대다수(60%)가 **Fitzpatrick Type III** (중간 피부)

#### 4.1.3 연령/성별 변동

| 그룹 | L* 변동 | a* 변동 | b* 변동 | 경향 |
|------|---------|---------|---------|------|
| 여성 20대 | +2.0 | -0.5 | -1.0 | 밝고 핑크빛 |
| 여성 30대 | +1.0 | 0 | 0 | 기준값 |
| 여성 40대 | 0 | +0.5 | +1.0 | 노란기 증가 |
| 여성 50대+ | -1.0 | +1.0 | +2.0 | 어둡고 노란 |
| 남성 (전 연령) | -1.5 | +0.5 | +0.5 | 전반적으로 어둡고 웜 |

### 4.2 퍼스널컬러 분포 통계

#### 4.2.1 잼페이스 139만건 분석 (MZ세대 중심)

| 순위 | 유형 | 비율 | 누적 |
|------|------|------|------|
| 1 | 가을 웜 트루 | 22.8% | 22.8% |
| 2 | 여름 쿨 트루 | 18.4% | 41.2% |
| 3 | 봄 웜 트루 | 18.2% | 59.4% |
| 4 | 가을 웜 뮤트 | 11.7% | 71.1% |
| 5 | 여름 쿨 뮤트 | 10.5% | 81.6% |
| 6 | 겨울 쿨 트루 | 10.4% | 92.0% |
| 7-12 | 기타 (Light/Bright/Deep) | 8.0% | 100% |

#### 4.2.2 시사점

```
웜톤 (봄 + 가을) = 52.7%
쿨톤 (여름 + 겨울) = 47.3%

True 계열 (트루) = 69.8%
Muted 계열 (뮤트) = 22.2%
Light/Bright/Deep = 8.0%
```

**결론**:
- 한국인은 **웜톤 우세** (b* 값 높음과 일치)
- **True 계열이 대다수** (극단적 밝기/채도 희소)
- **가을 웜 트루가 최다** (황색 언더톤 + 중간 명도 + 중간 채도)

### 4.3 서양인과의 비교

| 지표 | 한국인 | 서양인 (Caucasian) | 차이 |
|------|--------|-------------------|------|
| L* 평균 | 63 | 67 | -4 (더 어두움) |
| a* 평균 | 10 | 11 | -1 (비슷) |
| **b* 평균** | **18.5** | **15** | **+3.5 (더 노란)** |
| 웜톤 비율 | 52.7% | 40% | +12.7% |
| 주요 색소 | 카로틴 높음 | 멜라닌 다양 | - |

**핵심 차이**: **b* (황색도)**가 한국인이 유의미하게 높음 → 웜/쿨 판정 기준 조정 필요

## 5. 색상 조화 원리 (Color Harmony Principles)

### 5.1 색상 조화의 과학적 기초

#### 5.1.1 색상환 기반 조화 이론

**색상환 (Color Wheel)**:
```
              0° (빨강)
              ↑
    330° ────●──── 30°
  (자홍)    ╱ ╲    (주황)
          ╱   ╲
    300° ←●────●→ 60°
   (보라)  ↑   ↑  (노랑)
          ↓   ↓
    270° ←●────●→ 90°
  (파랑)   ╲   ╱  (연두)
            ╲ ╱
    240° ────●──── 120°
   (청록)    ↓    (초록)
            180°
```

#### 5.1.2 조화 유형 정의

| 유형 | 각도 관계 | 설명 | 예시 |
|------|----------|------|------|
| **유사색 (Analogous)** | Δh° < 30° | 인접 색상, 자연스러움 | 빨강-주황-노랑 |
| **보색 (Complementary)** | Δh° ≈ 180° | 반대 색상, 강한 대비 | 빨강-청록 |
| **분할보색 (Split-Comp.)** | Δh° ≈ 150°, 210° | 보색 양옆, 부드러운 대비 | 빨강-청록-초록 |
| **삼원색 (Triadic)** | Δh° = 120° | 삼각형 배치, 활기 | 빨강-파랑-노랑 |
| **사각배색 (Tetradic)** | Δh° = 90° | 사각형 배치, 풍부함 | 빨강-주황-청록-파랑 |
| **톤온톤 (Tone-on-Tone)** | Δh° = 0° | 동일 색상, 명도/채도 변화 | 연분홍-진분홍 |

### 5.3 색차 기반 조화 판정

#### 5.3.1 CIEDE2000 색차 공식

**개요**: 두 색상 간의 지각적 차이를 수치화

```
ΔE₀₀ = √[(ΔL'/kᴸSᴸ)² + (ΔC'/kᴄSᴄ)² + (ΔH'/kᴴSᴴ)² + Rᴛ(ΔC'/kᴄSᴄ)(ΔH'/kᴴSᴴ)]

여기서:
- ΔL': 명도 차이
- ΔC': 채도 차이
- ΔH': 색상 차이
- kᴸ, kᴄ, kᴴ: 응용 가중치 (기본값 1)
- Sᴸ, Sᴄ, Sᴴ: 보정 함수
- Rᴛ: 회전 함수 (블루 영역 보정)
```

#### 5.3.2 ΔE 해석 기준

| ΔE 범위 | 해석 | 조화 적합성 |
|---------|------|------------|
| < 1.0 | 구별 불가 | 완벽한 매치 |
| 1.0 - 2.0 | 미세한 차이 | 매우 조화로움 |
| 2.0 - 3.5 | 인지 가능 | 조화로움 |
| 3.5 - 5.0 | 명확한 차이 | 적당한 대비 |
| 5.0 - 10.0 | 다른 색상 | 대비 효과 |
| > 10.0 | 매우 다름 | 강한 대비 |

#### 5.3.3 이룸 적용: 제품 매칭

```typescript
/**
 * 피부톤과 제품 색상의 조화도 계산
 *
 * @param skinLab - 사용자 피부 Lab 값
 * @param productLab - 제품 색상 Lab 값
 * @param season - 사용자 계절 타입
 * @returns 0-100 매칭 점수
 */
function calculateHarmonyScore(
  skinLab: LabColor,
  productLab: LabColor,
  season: Season
): number {
  const deltaE = calculateCIEDE2000(skinLab, productLab);

  // 기본 점수: ΔE가 낮을수록 높은 점수
  let baseScore = Math.max(0, 100 - deltaE * 5);

  // 계절 보정: 계절에 맞는 색상이면 가산점
  const seasonBonus = isSeasonAppropriate(productLab, season) ? 15 : 0;

  return Math.min(100, baseScore + seasonBonus);
}
```

## 6. 조명 조건이 색상 인식에 미치는 영향

### 6.1 조명의 물리적 특성

#### 6.1.1 색온도 (CCT: Correlated Color Temperature)

```
색온도 (Kelvin)    광원 유형         색상 경향
───────────────────────────────────────────
1,800K            촛불              매우 따뜻함 (주황)
2,700K            백열등            따뜻함 (노랑)
3,000K            할로겐            약간 따뜻함

5,000K            정오 햇빛         중립-쿨
```

#### 6.1.2 광원별 스펙트럼 특성

| 광원 | 스펙트럼 | 연색성 (CRI) | 피부톤 영향 |
|------|---------|-------------|------------|

| 백색 LED | 청색 피크 | 80-90 | 약간 쿨톤 |

### 6.2 조명이 퍼스널컬러 판정에 미치는 영향

#### 6.2.1 실험 데이터

**조명 조건별 판정 오차** (동일 피부에 대한 Lab 측정값 변동):

| 조명 조건 | L* 오차 | a* 오차 | b* 오차 | 계절 오판정률 |
|----------|--------|--------|--------|--------------|
| D65 (기준) | 0 | 0 | 0 | 0% |
| 백열등 (2700K) | -2 | +1 | **+5** | **15%** (쿨→웜) |
| 형광등 (4000K) | +1 | -1 | **-3** | **10%** (웜→쿨) |
| LED (5000K) | 0 | 0 | -1 | 3% |
| 창가 자연광 | ±2 | ±1 | ±2 | 5% |

**핵심 발견**: **b* (황색도) 오차가 가장 큼** → 웜/쿨 판정에 직접 영향

#### 6.2.2 오판정 시나리오

```
시나리오: 실제 "여름 쿨 트루" (h° = 54°, 경계)

조명별 측정값:
- D65 (정확): h° = 54° → "여름 쿨" (정확)
- 백열등: h° = 62° (+8°) → "가을 웜" (오판정)
- 형광등: h° = 51° (-3°) → "여름 쿨" (정확하나 과도)
```

### 6.3 조명 보정 방법론

#### 6.3.1 Bradford Chromatic Adaptation

**원리**: 인간의 시각 시스템이 조명 변화에 적응하는 방식을 모델링

**적용 순서**:
```
1. 촬영 이미지에서 광원 추정 (Gray World / White Patch)
2. 추정된 광원의 XYZ 백색점 계산
3. Bradford 변환으로 D65 기준 XYZ로 변환
4. 변환된 XYZ에서 Lab 계산
```

**Bradford 변환 행렬**:
```
M_Bradford = | 0.8951   0.2664  -0.1614 |
             |-0.7502   1.7135   0.0367 |
             | 0.0389  -0.0685   1.0296 |
```

#### 6.3.2 이룸 구현 권장사항

```typescript
/**
 * 조명 보정 파이프라인
 *
 * CIE-3 모듈과 연계
 */
interface LightingCorrectionPipeline {
  // Step 1: 광원 추정
  estimateIlluminant(image: ImageData): WhitePoint;

  // Step 2: Bradford 변환
  adaptToD65(xyz: XYZ, sourceWhite: WhitePoint): XYZ;

  // Step 3: Lab 변환
  xyzToLab(xyz: XYZ): LabColor;

  // Step 4: 보정 전후 비교 (로깅용)
  logCorrection(original: LabColor, corrected: LabColor): void;
}
```
