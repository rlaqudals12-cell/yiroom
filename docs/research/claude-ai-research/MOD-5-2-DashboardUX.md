# MOD-5-2: Dashboard UX 트렌드 (2025-2026)

## 1. 핵심 요약

2025-2026년 대시보드 UX의 핵심 트렌드는 **AI 기반 인사이트 자동화**, **마이크로 인터랙션**, **모바일 퍼스트 설계**입니다. 데이터 시각화는 복잡한 차트보다 **Bento Grid 레이아웃**과 **프로그레스 링**으로 간결해지고, 인사이트는 사용자가 찾지 않아도 **AI가 자동 생성하여 푸시**합니다. 필터링은 **자연어 검색**과 **스마트 기간 프리셋**으로 단순화되며, 알림은 **중요도 기반 우선순위**와 **액션 가능한 CTA**가 필수입니다. 개인화는 **드래그앤드롭 위젯 재배치**와 **AI 학습 기반 자동 최적화**가 표준이 되고 있습니다.

---

## 2. 상세 내용

### 2.1 데이터 시각화

#### Bento Grid 레이아웃

Apple의 iOS 17 이후 확산된 Bento Grid는 2025-2026년 대시보드의 표준 레이아웃으로 자리잡았습니다. 불규칙한 크기의 카드를 그리드에 배치하여 시각적 계층을 명확히 하고, 중요 정보는 큰 카드로, 보조 정보는 작은 카드로 표현합니다.

| 특성 | 2023-2024 | 2025-2026 |
|------|-----------|-----------|
| 레이아웃 | 균일 그리드 | Bento Grid (불규칙) |
| 카드 크기 | 고정 | 콘텐츠 중요도에 따라 가변 |
| 계층 구조 | 위치로 표현 | 크기 + 위치로 표현 |

#### 프로그레스 링 & 아크 차트

복잡한 바 차트 대신 **프로그레스 링**이 주요 지표 표시에 선호됩니다. Apple Watch Activity Ring이 대중화시킨 패턴입니다.

- 직관적 진행률 표시 (0-100%)
- 작은 공간에서 고밀도 정보 전달
- 다중 링으로 여러 지표 비교 가능

#### 스파크라인 & 마이크로 차트

대시보드 카드 내 작은 영역에서 추세를 표시하는 스파크라인이 필수 요소가 되었습니다.

#### 애니메이션 타이밍 권장값

- 마이크로 인터랙션: 100-200ms
- 페이지 전환: 200-300ms
- 데이터 로딩 완료: 300-500ms

### 2.2 인사이트 표시

#### AI 자동 인사이트

사용자가 데이터를 분석하는 것이 아니라, AI가 분석 결과를 제공하는 패러다임 전환입니다.

| 유형 | 아이콘 | 예시 |
|------|--------|------|
| 트렌드 | TrendingUp | "3주 연속 운동 목표 달성" |
| 이상치 | AlertTriangle | "오늘 수분 섭취량이 평소보다 낮아요" |
| 추천 | Lightbulb | "내일 스트레칭 추가 추천" |
| 마일스톤 | Trophy | "100일 연속 기록 달성!" |

#### Cross-Module Insights

여러 분석 모듈의 데이터를 교차 분석하여 통합 인사이트를 제공합니다.

### 2.3 필터링/기간 선택

#### 스마트 기간 프리셋

| 프리셋 | 사용 빈도 | 권장 |
|--------|-----------|------|
| 오늘 | 30% | - |
| 이번 주 | 35% | **기본 선택** |
| 이번 달 | 25% | - |
| 커스텀 | 2% | 숨김 (고급) |

#### 빠른 필터

Chip/Tag 형태의 토글 필터가 표준입니다.

### 2.4 모바일 대시보드

#### 레이아웃

| 화면 크기 | 레이아웃 |
|-----------|----------|
| Mobile (<640px) | 단일 열 스택 |
| Tablet (640-1024px) | 2열 그리드 |
| Desktop (>1024px) | Bento Grid |

#### 터치 타겟

- 버튼 최소: 44x44px, 권장: 48x48px
- 카드 탭 영역: 전체 카드

#### 모바일 전용 패턴

- Bottom Sheet: 상세 정보 표시 (모달 대체)
- FAB: 주요 액션 (분석 시작)
- Pull-to-Refresh: 새로고침 제스처

### 2.5 알림/노티피케이션

| 우선순위 | 표시 방식 | 예시 |
|----------|-----------|------|
| Urgent | 모달 + 푸시 | "계정 보안 확인 필요" |
| High | 푸시 + 인앱 배너 | "첫 분석 완료!" |
| Medium | 인앱 배지 + 토스트 | "새 피부 분석 가능" |
| Low | 알림 센터만 | "오늘의 스타일 팁" |

#### 알림 최적화 원칙

1. 그룹핑: 유사 알림 묶기
2. 타이밍: 사용자 활성 시간대 고려
3. 빈도 제한: 일 최대 3-5개 푸시
4. 액션 가능: 알림에서 바로 액션 실행

### 2.6 개인화

#### 위젯 커스터마이징

드래그앤드롭으로 위젯 순서 변경이 표준입니다.

#### AI 기반 자동 최적화

사용자 행동 패턴을 학습하여 자동으로 대시보드 구성을 최적화합니다.

#### 테마 & 외관

| 설정 | 옵션 |
|------|------|
| 테마 | 라이트 / 다크 / 시스템 |
| 폰트 크기 | 작게 / 보통 / 크게 |
| 애니메이션 | 켜기 / 줄이기 / 끄기 |
