## 개요

### 리서치 워크플로우

```
1. claude.ai에서 자료 조사
   ├── 카테고리별 심층 리서치
   ├── 경쟁사/시장 조사
   └── 데이터 소스 확보
           ↓
2. Claude Code에서 구체화
   ├── SDD 스펙 문서 작성
   ├── 데이터 스키마 설계
   ├── Mock 데이터 구조화
   └── 기존 모듈 연동 계획
           ↓
3. 구현 및 검증
   ├── /qplan → /qcode → /qcheck
   └── typecheck + lint + test
```

## 조사 프롬프트 템플릿

### 피부시술 조사 시

```
이룸(Yiroom) 웰니스 앱의 피부시술 추천 모듈을 위해
[항목ID] [시술명]에 대해 조사해주세요.

필요한 정보:
1. 시술 원리 (1-2문장)
2. 적응증 (어떤 피부 고민에 효과적?)
3. 시술 과정 (소요 시간, 마취 여부)
4. 권장 주기 (몇 주 간격, 총 몇 회?)
5. 다운타임 (회복 기간)
6. 가격대 (서울 기준 범위)
7. 주의사항/부작용
8. 피부타입별 적합도 (건성/지성/민감성)
```
