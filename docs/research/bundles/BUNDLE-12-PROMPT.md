## 프롬프트

### TEAM-SKILL-GAP

```
AI 뷰티 스타트업에 필요한 역량과 스킬 갭을 분석해주세요.

조사 범위:
1. 필수 역할 및 역량 매트릭스
   - 프론트엔드 (React, React Native)
   - 백엔드 (Node.js, Supabase)
   - AI/ML (VLM, 프롬프트 엔지니어링)
   - 디자인 (UI/UX, 뷰티 도메인)
   - 그로스 (마케팅, 분석)
2. AI 도구 활용으로 대체 가능한 역할
3. 외주/파트타임 가능 영역
4. 필수 풀타임 역할
5. 스킬 성장 로드맵
6. 역량 평가 방법

기대 결과:
- 역할별 필수 스킬 매트릭스
- 현재 갭 평가 템플릿
- 스킬 우선순위 (Must/Should/Nice)
- 학습 리소스 추천
```

### TEAM-VELOCITY

```
소규모 팀의 개발 속도 최적화 전략을 조사해주세요.

조사 범위:
1. 린 스타트업 개발 방법론
2. 2-3인 팀 최적 업무 분배
3. AI 도구 활용 생산성 향상
4. 코드 품질 vs 속도 균형
5. 기술 부채 관리
6. 번아웃 예방

기대 결과:
- 주간/스프린트 계획 템플릿
- 업무 우선순위 프레임워크
- AI 도구 활용 가이드
- 생산성 지표 및 측정 방법
```

### ORG-HIRING

```
초기 스타트업의 채용 전략을 조사해주세요.

조사 범위:
1. 초기 핵심 채용 순서
2. 스타트업 채용 채널 (한국)
3. 급여 벤치마크 (시드/시리즈A)
4. 스톡옵션 설계
5. 리모트/하이브리드 정책
6. 채용 브랜딩

기대 결과:
- 채용 우선순위 로드맵
- 직무별 JD 템플릿
- 보상 패키지 설계 가이드
- 채용 프로세스 체크리스트
```

### ORG-CULTURE

```
고성과 스타트업 조직 문화 구축 전략을 조사해주세요.

조사 범위:
1. 미션/비전/가치 설정
2. 의사결정 프레임워크
3. 피드백 문화
4. 1:1 미팅 운영
5. 팀 빌딩 활동
6. 원격 협업 문화

기대 결과:
- 조직 문화 선언문 템플릿
- 의사결정 매트릭스
- 1:1 미팅 가이드
- 원격 협업 도구 추천
```
