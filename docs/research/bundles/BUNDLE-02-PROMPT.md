## 프롬프트

### BIZ-MONETIZATION

```
AI 기반 뷰티/웰니스 앱의 수익화 모델을 조사해주세요.

조사 범위:
1. Freemium 모델 (무료 기능 vs 프리미엄 기능 구분)
2. 구독 모델 (월간/연간, 가격대별 기능 차등)
3. 소비 기반 과금 (분석 횟수, 크레딧 시스템)
4. 제휴 마케팅 (화장품 브랜드, 뷰티 서비스)
5. 광고 모델 (비침투적 광고 전략)
6. B2B 라이선싱 (API, 화이트라벨)

기대 결과:
- 이룸에 최적인 수익화 모델 추천 (순위)
- 각 모델의 장단점 및 리스크
- 단계별 수익화 로드맵 (MVP → 성장 → 성숙)

참고: 이룸 철학 "통합된 자기 이해" - 수익화가 사용자 경험을 해치면 안 됨
```

### BIZ-PRICING

```
한국 시장의 뷰티/웰니스 구독 서비스 가격 전략을 조사해주세요.

조사 범위:
1. 경쟁사 가격 조사 (화해, 글로우픽, 뷰티 앱들)
2. 한국 소비자의 가격 민감도 및 지불 의향
3. 티어 구성 (무료/베이직/프로/엔터프라이즈)
4. 심리적 가격 책정 (9,900원 vs 10,000원)
5. 할인 전략 (연간 결제, 첫 달 할인, 친구 추천)
6. 가격 테스트 A/B 전략

기대 결과:
- 3-4개 티어 구성 제안 (가격, 기능)
- 한국 시장 최적 가격대 분석
- 가격 테스트 방법론
```

### BIZ-PAYMENT

```
한국 결제 시스템 통합 전략을 조사해주세요.

조사 범위:
1. 인앱 결제 (App Store, Google Play)
2. PG사 비교 (토스페이먼츠, 아임포트, 나이스페이)
3. 간편결제 (카카오페이, 네이버페이, 토스)
4. 정기결제 구현 방법
5. 환불/취소 정책 및 구현
6. 세금계산서/영수증 발행

기대 결과:
- 추천 PG사 및 선정 이유
- 결제 플로우 설계 (웹/앱 통합)
- 정기결제 구현 체크리스트
- 한국 결제 규정 준수 사항
```

### OPS-COST-MODEL

```
이룸 서비스의 운영 비용 모델을 조사해주세요.

조사 범위:
1. AI API 비용 (Gemini, 이미지 처리)
2. 인프라 비용 (Vercel, Supabase, Redis)
3. 스토리지 비용 (이미지 저장)
4. 사용자당 비용 (Unit Economics)
5. 스케일에 따른 비용 변화
6. 비용 최적화 전략

기대 결과:
- 사용자 1명당 월 운영비 추정
- 손익분기점 분석
- 비용 절감 기회 식별
- 스케일 시나리오별 비용 예측
```
