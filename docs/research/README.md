# Research 자료 관리

리서치 자료의 수집, 검토, 활용을 위한 폴더 구조입니다.

## 폴더 구조

```
docs/research/
├── raw/           # 원본 리서치 자료 (검토 전)
├── reviewed/      # 검토 완료 + 사용할 자료
├── archive/       # 불필요한 자료 (삭제 대기)
└── README.md      # 이 파일
```

## 워크플로우

```
1. Claude.ai 딥 리서치 결과 → raw/ 에 저장
2. 검토 후 사용할 부분 → reviewed/ 로 이동
3. 불필요한 부분 → archive/ 로 이동 (나중에 삭제)
4. reviewed/ 자료 기반으로 JSON/코드 생성
```

## 파일 네이밍 규칙

```
{날짜}-{주제}-{상태}.md

예시:
- 20251217-pilates-exercises-raw.md      (원본)
- 20251217-pilates-exercises-reviewed.md (검토 완료)
- 20251217-yoga-poses-archive.md         (불필요)
```

## 정리 주기

- **raw/**: 검토 완료 후 즉시 이동 또는 삭제
- **archive/**: 1주일 후 일괄 삭제
- **reviewed/**: JSON 변환 완료 후 archive로 이동

## 심층 리서치 프롬프트 템플릿

### 필라테스 운동 (심층)

```
필라테스 운동 가이드를 다음 형식으로 작성해줘:

1. 필라테스 기본 원리 6가지 (Joseph Pilates 원칙)

2. 레벨별 운동 분류:
   - 초급 (10개): 매트 기본 동작
   - 중급 (10개): 코어 강화 동작
   - 고급 (5개): 전신 통합 동작

3. 각 운동별 정보:
   - 한글명 / 영문명
   - 타겟 부위 (primary, secondary)
   - 난이도 (1-5)
   - MET 값 (칼로리 계산용)
   - 자세 설명 (3-5단계)
   - 호흡법 (들숨/날숨 타이밍)
   - 주의사항/금기
   - 변형 동작 (쉬운/어려운)

4. 추천 시퀀스 (15분/30분/45분)

5. 체형별 추천 운동 조합:
   - X형: 균형 유지
   - A형: 상체 강화
   - Y형: 하체 강화
   - H형: 전체 라인
   - O형: 코어 집중

표 형식으로 정리해줘.
```

### 요가 포즈 (심층)

```
요가 포즈 가이드를 다음 형식으로 작성해줘:

1. 요가 8단계 (Ashtanga) 간단 설명

2. 카테고리별 포즈:
   - 서서 하는 포즈 (7개)
   - 앉아서 하는 포즈 (5개)
   - 누워서 하는 포즈 (5개)
   - 밸런스 포즈 (3개)

3. 각 포즈별 정보:
   - 한글명 / 영문명 / 산스크리트명
   - 타겟 부위
   - 난이도 (1-5)
   - MET 값
   - 자세 설명 (단계별)
   - 호흡법
   - 효과 (신체적/정신적)
   - 금기사항 (부상, 임산부 등)
   - 변형/대체 포즈

4. 추천 플로우:
   - 모닝 요가 (15분)
   - 스트레스 해소 (20분)
   - 취침 전 (10분)

5. 운동 타입별 매칭:
   - flexer 타입: 추천 포즈 조합

표 형식으로 정리해줘.
```

### 브랜딩 리서치 (심층)

```
성별 중립적 웰니스 앱 브랜딩 분석:

1. 사례 분석 (각 앱별):
   - Calm
   - Headspace
   - Nike Training Club
   - Strava
   - MyFitnessPal

   각 앱의:
   - 로고 심볼 분석
   - 색상 팔레트 (primary, secondary)
   - 타이포그래피
   - 타겟 사용자층
   - 성별 중립성 점수 (1-10)

2. 공통 패턴:
   - 성공적인 중립 브랜딩의 공통점
   - 피해야 할 요소

3. 이룸 적용 제안:
   - 심볼 후보 3~5개 (꽃 대체)
   - 색상 조정 제안
   - 전체적인 톤앤매너

4. 2024-2025 트렌드:
   - 로고 디자인 트렌드
   - 앱 아이콘 트렌드
   - 색상 트렌드

표와 이미지 설명으로 정리해줘.
```
