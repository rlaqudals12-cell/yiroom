# AI 피부분석 서비스의 의료기기법 경계

한국 뷰티테크 스타트업 "이룸(Yiroom)"의 AI 피부분석 서비스는 **미용·화장품 추천 목적으로 운영하고, 질병 진단 기능 및 표방을 배제하며, 권고 면책 문구를 표기하는 경우 의료기기에 해당하지 않는 웰니스 제품으로 분류**될 수 있다. 핵심은 "사용목적"과 "표방 내용"에 있다. 2025년 1월 시행된 디지털의료제품법과 식약처 가이드라인에 따르면, 질병 진단·치료·예방 목적이 아닌 건강관리·미용 목적 서비스는 의료기기 규제 대상에서 제외된다. 다만 "진단", "치료" 등 의료 용어 사용 시 의료법·의료기기법 위반 위험이 있으므로 용어 선택과 면책 문구 표기에 세심한 주의가 필요하다.

---

## 의료기기법상 정의와 판단 기준

의료기기법 제2조 제1항은 의료기기를 **"사람이나 동물에게 단독 또는 조합하여 사용되는 기구·기계·장치·재료·소프트웨어"**로서 다음 목적에 해당하는 제품으로 정의한다.

| 호 | 사용목적 |
|---|---------|
| 1호 | **질병**을 진단·치료·경감·처치 또는 예방할 목적 |
| 2호 | **상해 또는 장애**를 진단·치료·경감 또는 보정할 목적 |
| 3호 | **구조 또는 기능**을 검사·대체 또는 변형할 목적 |
| 4호 | **임신**을 조절할 목적 |

식약처의 **"의료기기와 개인용 건강관리(웰니스) 제품 판단기준"(2020.11.30 개정)**에 따르면, 위 4가지 목적에 해당하지 않고 위해도가 낮은 제품은 **웰니스(개인용 건강관리) 제품**으로 분류된다. 웰니스 제품의 정의는 "건강 상태 또는 건강한 활동의 유지·향상 목적으로 사용자 안전에 미치는 위해도가 낮은 제품"이다.

의료기기 해당 여부를 결정하는 5가지 위해도 판단 문항이 있으며, **하나라도 고위해도에 해당하면 의료기기로 분류**된다.

| 문항 | 고위해도 (의료기기) | 저위해도 (웰니스) |
|-----|-------------------|-----------------|
| 사용환경 | 의료기관/의료인 사용 | 일반인/가정 사용 |
| 측정 목적 | 질병 진단용 | 건강관리용 |
| 정확도 의존성 | 진단/치료 결정에 직접 사용 | 참고용 정보 |
| 잘못된 정보의 결과 | 건강 피해 가능 | 경미한 불편 |
| 사용자 개입 | 의료 조치 직접 수행 | 생활습관 참고 |

## 이룸 서비스 기능별 의료기기 해당 여부

| 기능 | 해당 여부 | 판단 근거 |
|-----|----------|----------|
| 퍼스널 컬러 분석 | ❌ 비해당 | 미용·패션 목적, 질병과 무관 |
| 피부 타입 분석 (건성/지성/복합성) | ❌ 비해당 | 화장품 선택 참고용, 미용 목적 |
| 피부 고민 분석 (모공, 주름, 색소) | ⚠️ 조건부 | 미용 목적 표방 시 비해당 |
| 화장품/스킨케어 추천 | ❌ 비해당 | 미용 제품 추천, 치료 목적 아님 |
| 피부 질환 진단 | ✅ **해당** | 질병 진단 목적 (의료기기) |
| 피부암 검출/진단 | ✅ **해당** | 3등급 의료기기 (피부암영상검출·진단보조소프트웨어) |

## "진단" vs "분석" 용어의 법적 차이

의료법 제27조 "무면허 의료행위 금지"에 따르면, **"진단"은 의료행위의 핵심 요소**로서 의료인만 수행할 수 있다. 뷰티테크 서비스에서 "진단"이라는 용어를 사용하면 의료법 위반 위험이 발생한다.

| 구분 | 진단 (금지) | 분석 (허용) |
|-----|------------|------------|
| **목적** | 질환 유무 판정, 치료 방향 결정 | 미용 목적 정보 제공, 제품 추천 |
| **대상** | 피부 질환 (아토피, 건선, 피부염) | 피부 타입 (지성, 건성, 복합성) |
| **결과물** | 진단명, 처방, 치료 권고 | 피부 상태 점수, 제품 추천, 케어 가이드 |
| **법적 지위** | 의료행위 (의료인만 가능) | 뷰티 서비스 (일반 사업자 가능) |

### 허용/금지 표현 비교표

| ❌ 사용 금지 표현 | ✅ 사용 허용 표현 |
|-----------------|-----------------|
| "피부 질환 진단" | "피부 상태 분석" |
| "여드름 피부 진단" | "여드름성 피부 타입 분석" |
| "피부암 검사" | "색소 침착 분석" |
| "피부 문제 진단 결과" | "피부 상태 분석 결과" |
| "치료가 필요합니다" | "관리가 도움이 될 수 있습니다" |
| "OO 질환으로 진단됨" | "OO 피부 타입으로 분류됨" |
| "발병위험도" | "피부 고민 정도" |
| "의학적 효과" | "케어 가이드" |

### 금지되는 표현 범주

| 구분 | 금지 표현 예시 |
|-----|--------------|
| 질병 관련 | 아토피, 건선, 소양증, 여드름(진단/치료), 모낭충 |
| 의학적 효능 | 진단, 치료, 처방, 처치, 경감, 예방 |
| 의약품 연상 | 살균·소독, 항염·진통, 해독, 항암, 항균 |
| 신체 기능 조절 | 면역 강화, 근육 이완, 통증 경감 |
| 과장 표현 | 피부 나이 n세 감소, 완전히 제거, 확실히 보증 |

## 실무 체크리스트

### 의료기기 해당 여부 자가진단

- [ ] 서비스가 질병 진단·치료·예방을 목적으로 하는가?
- [ ] AI 분석 결과가 의학적 판단에 활용되는가?
- [ ] '진단', '치료', '발병위험도' 등의 용어를 사용하는가?
- [ ] 특정 피부질환(아토피, 습진, 피부암 등)을 판별하는가?

**→ 하나라도 "예"인 경우 식약처 의료기기 허가 검토 필요**

### 비의료 서비스 유지 조건

- [ ] 사용목적을 '건강 유지·증진' 또는 '미용·뷰티'로 한정
- [ ] 의료행위로 오인될 수 있는 용어 배제
- [ ] 질병 예방·치료 효능 광고 금지
- [ ] '피부타입 분석', '화장품 추천' 등 웰니스 목적 명시
- [ ] 사용자에게 의료기관 상담 권고 문구 포함

### 앱 내 문구 점검 항목

- [ ] "진단" → "분석" 또는 "확인"으로 변경
- [ ] "치료" → "관리" 또는 "케어"로 변경
- [ ] "의학적", "의료적" 표현 삭제
- [ ] 질병명 직접 언급 삭제 (아토피, 건선 등)
- [ ] "처방", "권고", "필요합니다" 형태 표현 수정
- [ ] 서비스 소개 화면에 면책 문구 삽입
- [ ] 이용약관에 "의료 서비스가 아님" 명시

### 면책 문구 체크리스트

**의료 관련:**
- [ ] "의료기기가 아닙니다" 명시
- [ ] "질병 진단/치료/예방 목적 아님" 명시
- [ ] "전문 의료 조언 대체 불가" 명시
- [ ] "피부과 전문의 상담 권고" 포함

**서비스 정확성 관련:**
- [ ] "AI 분석 결과 정확성 미보증" 명시
- [ ] "촬영 환경에 따른 결과 변동 가능성" 안내
- [ ] "참고용 정보"임을 명확히 표시

**개인정보 관련 (개인정보보호법):**
- [ ] 수집 항목 명시 (필수/선택 구분)
- [ ] 수집 목적 명시
- [ ] 보유 기간 명시
- [ ] 얼굴 이미지 처리 방식 명시

## 유사 서비스 제재 사례

2023년 추석 기간 식약처는 의료기기 거짓·과대광고 **38건**을 적발했다. 주요 위반 유형은 허가·인증받은 사항과 다른 광고(31건), 체험담 이용 광고(4건), '최고, 최상' 등 입증 어려운 표현(2건), 부작용 전부 부정 표현(1건)이었다. 화장품 허위·과대광고 금지 표현도 2025년 확대되어 **'병원전용', '피부과시술용', '피부나이 10년 감소'** 등의 표현이 금지 대상에 포함되었다.

## 결론 및 권고사항

이룸(Yiroom)의 AI 피부분석 서비스는 현재 서비스 구성(퍼스널 컬러 분석, 피부 타입 분석, 화장품 추천)을 **미용·뷰티 목적**으로 운영할 경우 의료기기에 해당하지 않는다. 그러나 **세 가지 핵심 조건**을 반드시 준수해야 한다.

첫째, **용어 관리**다. "진단", "치료", "질환" 대신 "분석", "케어", "타입"을 사용해야 한다. 둘째, **면책 문구 표기**다. 온보딩 화면, 분석 결과 화면, 이용약관에 "의료기기가 아님"과 "전문의 상담 권고" 문구를 필수 포함해야 한다. 셋째, **기능 제한**이다. 피부 질환 진단, 피부암 검출 등 의료 영역 기능은 식약처 허가 없이 제공할 수 없다.

불확실한 사항이 있을 경우 **보건복지부 유권해석 신청**(처리기간 20~37일)을 통해 공식 판단을 받는 것이 안전하며, 최종 이용약관 및 면책 조항은 **법률 전문가 검토**를 권장한다.
