## 🏆 뷰티/패션 경쟁사

| 앱 | 주요 기능 | 이룸 차별점 |
|----|----------|------------|
| **화해** | 리뷰, 성분 확인 | + 퍼스널컬러 통합 |
| **올리브영** | 커머스, 당일배송 | + AI 분석 |
| **잼페이스** | 퍼스널컬러, 닮은 연예인 | + 체형/운동/영양 |

## 🏃 운동/영양 경쟁사

| 앱 | 주요 기능 | 이룸 차별점 |
|----|----------|------------|
| **플랜핏** | AI 운동 루틴, 대체 운동, 헬스장 기구 DB, 영상 가이드, ChatGPT 기반 AI 코치 '맥스' | + 뷰티/패션 통합 |
| **눔** | 식단/다이어트 | + 퍼스널컬러 기반 |

### 플랜핏 상세 분석

```
플랜핏 특징:
- AI 맞춤 루틴
- 대체 운동 추천
- 헬스장 기구 데이터베이스
- 운동 영상 가이드
- ChatGPT 기반 AI 코치 '맥스'

→ 이룸 참고점: 헬스장 기구 DB, 대체 운동 로직
```
