## 👴 50대 우려 해소 (UI/UX 원칙)

> 전 연령 사용자를 위한 접근성 고려

### 핵심 원칙

```
- 단순한 인터페이스
- 큰 글씨/아이콘 옵션
- 친절한 안내
- 신조어 금지 (GMG, HMH 등)
```

---

## ✍️ UI 텍스트 가이드

### ❌ 피하기

```
GMG (Got My Goal)
HMH (Healthy Mind Healthy)
TL;DR
ASAP
```

### ✅ 사용하기

```
목표 달성
건강한 마음, 건강한 몸
요약
빠르게
```
