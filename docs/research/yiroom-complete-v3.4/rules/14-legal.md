## ⚖️ 법적 필수 기능 현황

| 기능 | 법적 근거 | 현재 상태 | 우선순위 |
|------|----------|----------|----------|

| 법정대리인 동의 | 개인정보보호법 제22조 | ⚠️ 미완성 (MVP: 14세 미만 차단) | 🔴 필수 |

## 🌍 국가별 법적 요구사항

| 국가 | 연령 제한 | 동의 요건 | 데이터 삭제 |
|------|----------|----------|------------|
| **한국** | 만 14세 | 법정대리인 동의 | 요청 시 5일 이내 |
| **미국** | 13세 (COPPA) | 부모 동의 | 주별 상이 |
| **EU** | 16세 (GDPR) | 명시적 동의 | 30일 이내 (Right to Erasure) |
| **일본** | 없음 | 이용목적 명시 | 요청 시 |
| **중국** | 14세 | 부모 동의 | 개인정보보호법 |

## 📋 미완성 작업 (Phase N)

### N-1: 만 14세 이상 확인 UI
- 복잡도: 35점 (Light)
- 구현: 생년월일 입력 → 연령 계산 → 14세 미만 차단

### N-2: 14세 미만 회원가입 차단 MVP
- 복잡도: 25점 (Quick)
- 구현: 회원가입 플로우에서 연령 확인 추가

### N-3: GDPR Cron 활성화
- 복잡도: 15점 (Quick)
- 구현: Supabase 대시보드에서 pg_cron 설정

### N-4: 개인정보처리방침 아동조항 (선택)
- 구현: 법률 검토 후 추가
