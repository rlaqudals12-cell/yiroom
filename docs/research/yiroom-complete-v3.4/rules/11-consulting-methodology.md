## 🎨 전문 이미지 컨설팅 방법론 (4가지 역할)

> 이룸은 전문가 수준의 컨설팅 로직을 AI로 구현합니다.

### 1️⃣ 이미지 컨설턴트 (Total Image Consultant)

```
[Input] 라이프스타일, 직업, 추구 분위기, TPO(시간/장소/상황)
[Process] 컨셉 키워드 도출 (예: 지적인, 러블리한, 시크한)
[Output] 퍼스널 브랜딩 + 이미지 설계
```

### 2️⃣ 비주얼 디렉터 (Visual Director)

```
[Input] 사용자 선호 이미지/무드
[Process] 무드보드 생성 → 공통 속성 추출 → 스타일 테마 정의
[Output] 시각적 톤앤매너, 컬러 팔레트
```

### 3️⃣ 스타일리스트 (Stylist)

```
[Input] 옷장 사진, 보유 아이템 목록
[Process] 옷장 점검 → 버릴 옷/남길 옷 분류 → Mix & Match 공식
[Output] 캡슐 옷장 구성, 체형별 스타일링 보완
```

### 4️⃣ 퍼스널 컬러 컨설턴트

```
[Input] 얼굴 사진 (정면, 자연광)
[Process] 디지털 드레이핑 → 4계절/12타입/16타입 분류
[Output] 베스트/워스트 컬러, 화장품/의류 색상 추천
```

## 🔬 디지털 드레이핑 알고리즘 변수

| 변수 | 설명 | 측정 방법 |
|------|------|----------|
| **ITA°** | 피부 밝기 (Individual Typology Angle) | L*a*b* 색공간 |
| **Undertone** | 쿨톤/웜톤 판별 | a* (붉은기), b* (노란기) |
| **Contrast** | 눈-머리-피부 대비 | 명도 차이 분석 |
| **Saturation** | 채도 적합성 | 채도 반응 테스트 |

## 😊 얼굴형 분류 시스템

### 6가지 얼굴형

| 얼굴형 | 특징 | 스타일링 방향 |
|--------|------|--------------|
| 타원형 (Oval) | 황금비율 | 대부분 어울림 |
| 둥근형 (Round) | 가로/세로 비슷 | 세로 강조 |
| 사각형 (Square) | 각진 턱선 | 부드러운 곡선 |
| 긴형 (Oblong) | 세로가 김 | 가로 강조 |
| 하트형 (Heart) | 넓은 이마, 뾰족한 턱 | 하관 볼륨 |
| 다이아몬드 (Diamond) | 좁은 이마/턱, 넓은 광대 | 밸런스 조정 |

### 기술 구현

- **MediaPipe**: 468포인트 얼굴 랜드마크
- **ABCL 프레임워크**: 한국 이미지 컨설팅 표준
