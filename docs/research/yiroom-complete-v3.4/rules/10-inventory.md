## 📦 상품 등록 시스템 (4가지 방법)

| 방법 | 활용 | 기술 |
|------|------|------|
| **바코드 스캔** | 화장품/영양제 빠른 등록 | 바코드 DB 연동 |
| **카메라 AI 인식** | 옷/가방 등 패션 아이템 | Vision AI 모델링 |
| **성분표 OCR** | 화장품 성분, 영양제 상호작용 | Google Vision OCR |
| **직접 입력** | 정보를 알고 있는 경우 | 수동 폼 |

## 🗂️ 인벤토리 카테고리

### 내 옷장 (Wardrobe)
- 가상 드레이핑용
- 캡슐 코디 생성

### 내 화장대 (Cosmetics)
- 성분 분석용
- 피부 타입 매칭

### 내 영양제함 (Supplements)
- 상호작용 체크용
- 복용 스케줄 관리

### 내 운동기구 (Equipment)
- 루틴 생성용
- 캡슐 운동 적용

### 내 주방 (Kitchen) - 신규 제안
```
├── 식재료 (냉장고/냉동실/팬트리)
├── 조미료 (소금, 간장, 고추장...)
└── 조리도구 (냄비, 프라이팬, 오븐...)

→ 레시피 추천 시 보유 재료 우선 활용
```

## 🏋️ 운동 인벤토리 상세

### 운동 보충제
```
- 크레아틴
- 프로틴 (WPC/WPI/카제인)
- BCAA/EAA
- 글루타민
- 베타알라닌
- 카페인
- 프리워크아웃
- 관절 건강 (글루코사민)
```

### 홈트레이닝 기구
```
- 덤벨 (무게별)
- 바벨
- 케틀벨
- 저항 밴드
- 폼롤러
- 요가매트
- 풀업바
```

### 헬스장 기구 (플랜핏 참고)
```
├── 유산소: 러닝머신, 사이클, 일립티컬, 스테어마스터
├── 가슴: 벤치프레스, 체스트프레스, 펙덱, 케이블 크로스오버
├── 등: 랫풀다운, 시티드로우, 케이블로우, 티바로우
├── 어깨: 숄더프레스, 레터럴레이즈 머신, 리어델트
├── 하체: 레그프레스, 레그컬, 레그익스텐션, 힙어덕션
├── 팔: 케이블, 프리처컬 머신
└── 프리웨이트: 덤벨, 바벨, EZ바, 스미스머신
```

## 🔄 캡슐 루틴 연동

```
[활용]
- 보유 기구 기반 운동 루틴 자동 생성
- 기구 없으면 맨몸 운동 대체 추천 (캡슐 루틴)
- "보편적으로 헬스장에 있는 기구"로 기본 추천
```
