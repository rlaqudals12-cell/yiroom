# 08. 체형 교정 운동 시스템

> version: 1.0
> last_updated: 2026-01-14
> source: v2.2 섹션 20

---

## 🦴 패션/체형 분석 → 교정 운동 연계

```
[분석 결과]
"어깨가 말려 있습니다"

[교정 운동 추천]
1. 페이스풀
2. 밴드 풀어파트
3. 월 슬라이드
4. 흉추 스트레칭
```

---

## 📋 체형 문제별 교정 운동

| 체형 문제 | 교정 운동 |
|----------|----------|
| **어깨 말림** | 페이스풀, 밴드 풀어파트, 월 슬라이드, 흉추 스트레칭 |
| **거북목** | 친턱, 넥 스트레칭, 흉추 가동성 |
| **굽은 등** | 흉추 익스텐션, 캣카우, 프론 Y |
| **골반 전방 경사** | 힙 플렉서 스트레칭, 글루트 활성화, 데드버그 |
| **골반 후방 경사** | 힙 힌지, 햄스트링 강화, 코어 안정화 |

---

## 📊 체형 교정 전/후 2D 시각화

스파인앤포스처 스타일 시각화:

```
[현재 자세]        [교정 목표]
    ●                  ●
   /|\               /|\
    │        →        │
   / \               / \
  🔴 문제점        🟢 개선점
```

- 현재 자세 실루엣 (빨간선)
- 교정 후 예상 실루엣 (녹색선)
- 비포/애프터 비교 제공

---

## 🔗 통합 외모 컨설팅 연동

체형 분석 결과 → 운동 + 패션 동시 추천

```
예시: "어깨가 좁고 말려 있습니다"

┌─────────────────────────────────────┐
│ [운동 추천]        [패션 추천]       │
│ • 사이드 레터럴    • 어깨 패드 자켓  │
│ • 페이스풀        • 보트넥 상의     │
│ • 밴드 풀어파트   • 스퀘어넥       │
│ • 숄더프레스(헬스장)• 운동복: 어깨강조│
├─────────────────────────────────────┤
│     장점 극대화 + 단점 극복 동시     │
└─────────────────────────────────────┘
```
