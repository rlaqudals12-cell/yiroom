---

## 🧍 아바타 생성 시스템

### 입력 구조

```
[필수 입력]
├── 키 (cm)
├── 체중 (kg)
└── 전신 사진 → AI가 분석해서 치수 예측!

[선택 입력 - 더 정확하게]
├── 어깨너비
├── 가슴둘레
├── 허리둘레
├── 힙둘레
└── ...
```

---

## 🤖 AI 치수 예측

- 전신 사진 1장 → MediaPipe로 신체 랜드마크 추출
- AI가 어깨/가슴/허리/힙 자동 예측
- 선택 입력 시 정확도 향상

---

## 📷 다각도 촬영 지원

| 모듈 | 정면 | 좌측 | 우측 | 후면 |
|------|------|------|------|------|
| 얼굴/피부 | ✅ 필수 | ⚪ 선택 | ⚪ 선택 | - |
| 체형 | ✅ 필수 | ⚪ 선택 | ⚪ 선택 | ⚪ 선택 |

---

## 🔄 데이터 흐름

```
전신 사진
    ↓
MediaPipe 랜드마크 추출 (468포인트)
    ↓
비율 계산 (어깨/허리/힙)
    ↓
AI 치수 예측
    ↓
사용자 프로필 저장
    ↓
패션/운동 추천에 활용
```
