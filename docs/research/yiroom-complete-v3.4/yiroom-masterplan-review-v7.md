# 이룸(Yiroom) 마스터 플랜 종합 검토 요청 v7

## 검토 목적
작성된 마스터 플랜(계획서)을 기반으로 **완전체(Full Version)** 앱으로서 다음을 검증해줘:

1. **민감 정보 보호** 및 개인정보 처리 정책이 완비되어 있는지
2. **법적 필수 기능** (동의, 철회, 삭제, 탈퇴)이 설계되어 있는지
3. 운동/영양 **경쟁사 대비 차별화** 및 완성도
4. **캡슐 루틴** 개념 (대체 운동/대체 식단) 적용
5. 사용자 **인벤토리(보유 기구/재료) 활용** 추천 시스템
6. 체형 교정 + 장점 극대화/단점 극복 통합 추천
7. **모듈간 데이터 일관성** 및 재사용 UX
8. **기존 구현된 기능**들이 잘 연결/통합되었는지
