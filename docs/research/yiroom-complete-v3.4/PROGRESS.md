## 🧠 핵심 결정 사항

### 지침 구조 (v6.1)

```
<role>           ← 역할 정의
<context>        ← 프로젝트 현황 (파일 참조)
<guidelines>     ← 작업 원칙
<constraints>    ← 제약사항 (파일 참조)
<session_protocol> ← 세션 자동화 (v6.1 신규)
<output_format>  ← 응답 형식
<examples>       ← 예시 6개 (세션 종료 예시 추가)
<file_usage>     ← 파일 참조 방법
```

### 세션 자동화 규칙 (v6.1)

| 트리거 | 동작 |
|--------|------|
| 세션 시작 ("이어서", "계속") | PROGRESS.md 상태 확인 및 요약 |
| 세션 종료 ("오늘 여기까지") | 세션 요약 + 업데이트 내용 생성 |
| 파일 변경 | 변경 추적 및 changelog 기록 |

### 버전 관리 규칙

| 변경 유형 | 버전 증가 |
|----------|----------|
| 오타 수정 | 없음 |
| 내용 추가 | +0.0.1 (patch) |
| 구조 변경 | +0.1 (minor) |
| 대규모 개편 | +1.0 (major) |
