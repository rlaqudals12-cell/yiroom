# 이룸(Yiroom) B2B·파트너십·데이터 수익화 전략 종합 가이드

AI 기반 뷰티/웰니스 통합 분석 앱 "이룸"은 **연간 15~50억원 규모의 B2B 수익**을 창출할 수 있다. 핵심은 화장품 브랜드 API 제공(예상 연 10억원+), 피트니스 센터 제휴(연 3~8억원), 익명화 트렌드 리포트 판매(연 3~10억원)의 3축 전략이다. 다만 의료 클리닉 연계는 의료광고법 규제로 **광고 기반 모델만 합법**이며, 데이터 수익화 시 **민감정보(피부진단, 체형) 별도 동의**가 필수다.

---

## Part 1: 파트너십 전략 — 브랜드·피트니스·의료 3축

### 뷰티 브랜드 협업은 체험단과 API 연동이 핵심

한국 뷰티 앱 시장에서 화해(Hwahae)는 **900만+ 다운로드, 550만 건 리뷰**를 기반으로 브랜드 협업의 교과서를 제시한다. 화해의 수익 모델은 체험단(꼼평단), CPC 검색광고, 어워드 엠블럼 라이선스로 구성되며, 체험단 캠페인당 **500만~1,000만원**, 브랜드 전용 화이트라벨 서비스는 **월 1,000만~3,000만원** 수준이다.

올리브영은 2025년 내 100개 점포에 **AI 피부/퍼스널컬러 진단 기기**를 배치 예정으로, 체험형 서비스와 제품 구매 연결 패키지화 전략을 추진 중이다. 이룸은 이 흐름에서 AI 분석 기술 제휴 기회를 포착할 수 있다.

| 협업 유형 | 예상 단가 | 타겟 브랜드 |
|----------|----------|------------|
| AI 분석 기반 체험단 | 500만~1,000만원/캠페인 | 인디 브랜드 |
| 퍼스널컬러 매칭 리포트 | 300만~500만원/월 | 색조 브랜드 |
| 브랜드 전용 화이트라벨 | 1,000만~3,000만원/월 | 대기업 |
| CPA 제품 추천 | 구매액의 5~15% | 전 브랜드 |

**대기업 vs 인디 브랜드 접근 차이**: 대기업에는 R&D 기간 단축과 맞춤형 서비스를, 인디 브랜드에는 올리브영 입점 전 검증과 타겟 리뷰 확보를 핵심 가치로 제안해야 한다.

### 피트니스 센터 제휴 — 연 19% 성장 시장 공략

한국 피트니스 클럽 시장은 **2024년 약 5,700억원**에서 2030년 **9,960억원**(CAGR 7.64%)으로 성장 전망이며, 피트니스 앱 시장은 **CAGR 19.17%**로 더욱 가파르다. 주요 프랜차이즈로는 스포애니(108개, 유효회원 20만명), 커브스(228개), 에이블짐(42개)이 있다.

| 프랜차이즈 | 가맹점 수 | 특징 |
|-----------|----------|------|
| 스포애니 | 108~110개 | 국내 1위, 24시간 운영 |
| 커브스 | 228개 | 여성전용, 30분 서킷 |
| 에이블짐 | 42개 | 프리미엄 시설 |

**바디코디**가 국내 피트니스 SaaS 1위로 **4,000개+ 센터, 170만 회원** 앱을 보유하고 있어, 이룸의 체형분석 API를 바디코디에 연동하면 빠른 시장 침투가 가능하다. 눔(Noom)은 B2C에서 쌓은 신뢰로 현대해상, 노보 노디스크 등 B2B 확장에 성공했으며, B2B 매출이 전체 **50% 이상**을 차지한다.

**추천 제휴 모델**: PT 연계 CPA(건당 2~5만원), 센터 B2B 라이선스(월 5~10만원/센터), 기업 복지 프로그램(인당 월 5,000~15,000원)

### 의료 클리닉 연계 — 규제 지뢰밭을 피하라

의료법 제27조 제3항은 **영리 목적 환자 소개·알선·유인을 금지**하며, 위반 시 3년 이하 징역 또는 3,000만원 이하 벌금이다. 강남언니(힐링페이퍼) 대표는 2024년 시술쿠폰 결제 수수료(진료비 연동) 모델로 **징역 8개월, 집행유예 2년**을 선고받았다.

| 수익 모델 | 합법 여부 | 근거 |
|----------|----------|------|
| 페이지뷰/클릭당 광고비 | ✅ 합법 | 정보 제공 대가 |
| 월정액 SaaS 이용료 | ✅ 합법 | 서비스 요금 |
| 진료비 % 수수료 | ❌ 불법 | 환자 알선 (대법원 판례) |
| 예약 건당 성공 수수료 | ⚠️ 회색지대 | 중개 성격 인정 시 위험 |

**GLP-1 비만치료제(위고비, 삭센다)는 2024년 12월부터 비대면 처방 전면 금지**되어, 비만클리닉 연계는 정보 제공 수준으로 제한해야 한다. AI 피부분석 결과는 "여드름 유형: 화농성"(✅)처럼 일반 정보로만 표현하고, "레이저 시술 필요"(❌)와 같은 의료적 판단은 배제해야 한다.

## Part 2: B2B 전환 전략 — SaaS·API·화이트라벨

### Perfect Corp 벤치마크 — B2B 마진 90%의 비밀

Perfect Corp(YouCam)은 **800+ 브랜드 파트너(에스티로더, 로레알 등)**, 11억 앱 다운로드를 기반으로 2024년 매출 **$60.2M(약 800억원)**을 달성했다. 핵심은 **B2B 구독 서비스가 매출의 88%, 마진 90%+**라는 점이다.

| Perfect Corp 제공 API | 설명 |
|----------------------|------|
| AI Skin Analysis API | 정밀 피부 평가 |
| AI Face API | 얼굴형/비율 분석 |
| Virtual Try-On API | 메이크업, 주얼리 가상 체험 |
| GenAI Content API | AI 캠페인 자산 생성 |

**API 과금 방식 비교**: 건당 $0.01~0.10(스타트업), 월정액 $200K~500K/년(엔터프라이즈), Pay-as-you-go 종량제(Perfect Corp 방식)

### 뷰티샵 B2B SaaS — 월 2~5만원 가격대가 적합

한국 뷰티샵 SaaS 시장에서 핸드SOS(월 19,900원, 시장점유율 1위), 뷰티인프라(월 22,500원), 헤어사랑넷(월 7,000원)이 경쟁 중이다. 이룸의 퍼스널컬러 진단 도구를 화이트라벨로 제공할 경우, **월 20,000~50,000원/샵** 수준의 가격이 시장에 적합하다.

| B2B 모델 | 타겟 | 예상 수익 |
|---------|------|----------|
| 퍼스널컬러 API | 화장품 브랜드 | 호출당 $0.02~0.10 |
| 뷰티샵 화이트라벨 | 헤어샵/피부관리실 | 월 2~5만원/샵 |
| 피트니스 체형분석 API | 피트니스 SaaS | 월정액 또는 수익쉐어 |
| 엔터프라이즈 라이선스 | 대형 프랜차이즈 | 연 1~5억원 |

### 피트니스 화이트라벨 — 인바디 연동이 관건

체형분석 솔루션에서 **인바디(InBody)**는 API/SDK를 제공하며 LookinBody Web 클라우드로 데이터 연동이 가능하다. 3DLook(FitXpress)은 스마트폰 사진 2장으로 80개 체형 측정치를 제공하는 AI 솔루션으로, 이룸의 경쟁 및 협력 대상이다.

## Part 3: 데이터 수익화 — 규제 준수가 곧 경쟁력

### 트렌드 리포트 판매 — 연 3~10억원 잠재력

화해는 **1,200만 유저 데이터, 940만 실사용 리뷰**를 기반으로 연 2회 트렌드 리포트를 발간하며, 올리브영은 '인사이트 스튜디오'를 통해 **연간 1억 건+ 구매 데이터** 분석 결과를 공개한다. 두 플랫폼 모두 리포트를 무료로 제공하고 광고 상품 연계 판매로 수익화한다.

| 데이터 상품 | 예상 가격 | 구매자 |
|-----------|----------|-------|
| 연간 트렌드 리포트 | 500만~2,000만원/건 | 브랜드사, 리서치사 |
| 맞춤형 인사이트 | 1,000만~5,000만원/건 | 대형 브랜드 |
| 실시간 대시보드 구독 | 월 50만~500만원 | 마케팅 에이전시 |

### 개인정보보호법(PIPA) — 민감정보 별도 동의 필수

**피부진단 결과, 체형 데이터는 '건강정보'로 민감정보에 해당**하며, 별도 명시적 동의 없이 처리 시 **5년 이하 징역 또는 5천만원 이하 벌금**이다. 2025년 12월 개정안은 중대 유출 시 **매출액 최대 10% 과징금**을 부과한다.

| 데이터 유형 | 민감정보 여부 | 조치 |
|-----------|-------------|------|
| 피부 타입 정보 | △ 건강정보 가능성 | 별도 동의 권장 |
| 체형 데이터 | ✅ 건강정보 | 별도 동의 필수 |
| 피부사진(얼굴) | △ 생체인식 가능성 | 분석 후 즉시 삭제 |
| 퍼스널컬러 결과 | ❌ 일반정보 | 일반 동의로 충분 |

**익명화 vs 가명처리**: 익명정보는 개인정보보호법 미적용으로 자유롭게 판매 가능하지만, 가명정보는 통계/과학적 연구/공익 목적으로만 제공 가능하며 **영리 목적 직접 판매는 제한적**이다.

### 앱 내 동의 획득 UI 설계

```
[필수] 서비스 이용 개인정보 수집·이용 동의
[필수] 민감정보(피부분석/체형) 수집·이용 동의 ← 별도 체크박스
[선택] 마케팅 목적 활용 동의
[선택] 익명화 데이터 제3자 제공 동의
```

철회권 보장을 위해 마이페이지에 동의 철회 버튼을 구현하고, 철회 시 **72시간 이내 데이터 처리 중단 및 파기**해야 한다.

## 결론: 단계별 실행 로드맵

**1단계(0~6개월)**: B2C 앱 런칭 + 사용자 기반 확보, 무료 체형분석으로 다운로드 유도
**2단계(6~12개월)**: 인디 브랜드 체험단 협업(캠페인당 500만원), 피트니스 센터 5~10곳 파일럿
**3단계(12~24개월)**: API/화이트라벨 B2B 런칭, 프랜차이즈 본사 영업, 트렌드 리포트 판매 시작
**4단계(24개월+)**: 기업/보험사 B2B 헬스케어 확장, 해외 진출

핵심 성공 요인은 **AI 기술 신뢰성 확보**, **규제 준수 체계 구축**, **화해/올리브영 대비 차별화(통합 분석)**이다. 의료 클리닉 연계는 광고 기반으로만 진행하고, 데이터 수익화는 익명화 통계 리포트부터 시작하는 것이 리스크를 최소화하는 전략이다.
