## 🔴 영구 제외 (Permanent Exclusion)

> **재제안 금지** - 동일/유사 기능 제안 시 이 문서 참조

### 기준

| 코드 | 사유 | 설명 |
|------|------|------|
| `LEGAL` | 법적 리스크 | 저작권, 초상권, 개인정보 침해 |
| `BUSINESS` | 비즈니스 방향 | 수익 모델, 브랜드 정체성과 충돌 |
| `ETHICAL` | 윤리적 문제 | 사용자 신뢰, 차별, 유해 콘텐츠 |

### 현재 목록

| 기능 | 분류 | 제외 사유 | 대안 |
|------|------|----------|------|
| 닮은 연예인/유튜버 | `LEGAL` | 초상권 침해 | 일러스트 가이드 |
| 연예인 사진 사용 | `LEGAL` | 저작권/초상권 | 컬러칩 시각화 |
| 유튜버 리뷰 연동 | `LEGAL` | 저작권 | 자체 리뷰 시스템 |
| 구독 모델 | `BUSINESS` | 수익 방향 불일치 | 구매 전환 |

## 🟡 보류 (Hold)

> **조건 충족 시 재검토** - 트리거 조건 명시 필수

### 보류 유형

| 코드 | 유형 | 재검토 트리거 |
|------|------|--------------|
| `TECH_HOLD` | 기술적 보류 | 기술 성숙도 향상 시 |
| `COST_HOLD` | 재정적 보류 | 예산/수익 확보 시 |
| `PRIORITY_HOLD` | 우선순위 보류 | 선행 작업 완료 시 |
| `REVIEW_HOLD` | 검토 보류 | 추가 조사/판단 필요 |

### 현재 목록

#### 기술적 보류 (`TECH_HOLD`)

| 기능 | 보류 사유 | 재검토 트리거 |
|------|----------|--------------|
| 실시간 자세 교정 | 실시간 영상 처리 기술 | MediaPipe 안정화 |
| AR 가상 피팅 | 3D 모델링 복잡도 | WebXR 보편화 |

#### 재정적 보류 (`COST_HOLD`)

| 기능 | 보류 사유 | 재검토 트리거 |
|------|----------|--------------|
| GPT-4 Vision | API 비용 | MAU 1만+ |
| 실시간 AI 코칭 | 서버 비용 | 수익화 후 |

#### 우선순위 보류 (`PRIORITY_HOLD`)

| 기능 | 보류 사유 | 재검토 트리거 |
|------|----------|--------------|
| 커뮤니티 | 사용자 풀 필요 | MAU 5천+ |
| 챌린지/배틀 | 커뮤니티 의존 | 커뮤니티 출시 후 |
| 헬스장 연동 | 사업 개발 필요 | B2B 계약 체결 |

#### 검토 보류 (`REVIEW_HOLD`)

| 기능 | 보류 사유 | 재검토 트리거 |
|------|----------|--------------|
| `.beads` 이슈 트래킹 | 실제 사용 여부 불명확 | 사용 현황 확인 후 |
| `standup.md` 명령어 | CLAUDE.md 미참조 | 실제 사용 확인 후 |
| `create-feature.md` | 초기 설정 전용? | 실제 사용 확인 후 |

## 분류 프로세스

### 새 기능 제안 시

```
1. FIRST-PRINCIPLES.md 체크
   → "통합된 자기 이해에 기여하는가?"

2. 영구 제외 목록 체크
   → 유사 기능 이미 제외되었는가?

3. 보류 조건 체크
   → 기술/재정/우선순위 제약 있는가?

4. 활성화 조건 체크
   → 조건부 활성화 대상인가?

5. 분류 결정
   → 🟢 구현 / 🟡 보류 / 🔴 제외
```

### 기존 기능 재검토 시

```
1. 사용되고 있는가?
   → 아니오: 삭제 검토

2. 가치가 있는가?
   → 아니오: 삭제 검토

3. 문제가 있는가?
   → 예: 리빌딩 검토

4. 개선 필요한가?
   → 예: 보완 계획
```
