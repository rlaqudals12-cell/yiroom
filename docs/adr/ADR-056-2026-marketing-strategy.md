# ADR-056: 2026 마케팅 전략

## 상태

`accepted`

## 날짜

2026-01-24

## 맥락 (Context)

### 현재 이룸의 마케팅 현황

이룸은 현재 **MVP 완료, PMF 탐색 중** 단계로, CMO 기능이 최소화되어 있습니다.

| 항목 | 현재 상태 | 비고 |
|------|----------|------|
| **제품 단계** | MVP 완성 | 7개 분석 모듈 |
| **사용자 기반** | 0명 (베타 준비 중) | - |
| **마케팅 예산** | 미확정 | Pre-seed 자금 대기 |
| **마케팅 인력** | 1인 창업자 겸임 | - |

### 마케팅 의사결정 필요성

ADR-054 (어필리에이트 우선 전략)에서 **무료 서비스 + 어필리에이트** 모델을 채택했습니다. 이 전략을 실행하기 위한 구체적인 마케팅 로드맵이 필요합니다.

## P1: 궁극의 형태 (Ultimate Form)

> 원리 참조: [00-first-principles.md](../../.claude/rules/00-first-principles.md) P1

### 이상적 최종 상태

**제약이 없다면**:
- MAU 1M+ 글로벌 사용자
- CAC ₩500 이하 지속
- 바이럴 K-Factor 1.0+ (자기 지속 성장)
- 멀티 채널 완벽 통합 마케팅
- AI 기반 개인화 마케팅 자동화
- K-뷰티 대표 AI 앱 브랜딩

### 물리적 한계

| 제약 | 현실 | 완화 |
|------|------|------|
| 마케팅 예산 | Pre-seed 대기 | 오가닉 우선 |
| 마케팅 인력 | 1인 창업자 겸임 | 자동화 도구 활용 |
| 경쟁 심화 | AI 뷰티 앱 증가 | 차별화 콘텐츠 |
| 오가닉 성장 예측 | 불확실성 높음 | 다채널 분산 |

### 100점 기준

| 항목 | 100점 기준 | 현재 목표 |
|------|-----------|----------|
| MAU | 1M+ | 5,000 (Phase 1) |
| CAC | ₩500 이하 | ₩500 |
| K-Factor | 1.0+ | 0.3 |
| SNS 팔로워 | 100K+ | 15K |
| 브랜드 인지도 | Top 3 | 진입 |

### 현재 목표

**Phase 1: 25%** (Foundation)
- MAU 0 → 5,000
- 오가닉 성장 집중
- 나노 인플루언서 20명 협업
- SEO/ASO 기반 구축
- 마케팅 비용 ₩5M (월 ₩1M)

## 대안 (Alternatives Considered)

### 1. 유료 광고 우선 전략

| 대안 | 장점 | 단점 | 제외 사유 |
|------|------|------|----------|
| Meta Ads 집중 | 빠른 사용자 확보 | CAC ₩4,000+ | 예산 대비 비효율 |
| Google Ads 집중 | 구매 의도 높음 | CAC ₩5,000+ | 초기 단계 부적합 |

**제외 사유**: Phase 1에서는 오가닉 성장으로 PMF 검증이 우선. 유료 광고는 Phase 2에서 검증된 메시지로 시작.

### 2. PR/미디어 우선 전략

| 대안 | 장점 | 단점 | 제외 사유 |
|------|------|------|----------|
| PR 에이전시 활용 | 브랜드 인지도 | 월 ₩3M+ 비용 | 예산 초과 |
| 스타트업 미디어 집중 | 무료/저비용 | 일반 사용자 도달 제한 | 보조 수단으로 활용 |

**제외 사유**: PR은 보조 채널로 활용. 인플루언서가 더 직접적인 전환 유도.

### 3. 커뮤니티 우선 전략

| 대안 | 장점 | 단점 | 제외 사유 |
|------|------|------|----------|
| Discord 커뮤니티 구축 | 높은 참여도 | 느린 성장 | 초기 사용자 필요 |
| 카카오 오픈채팅 | 한국 사용자 친숙 | 관리 부담 | Phase 2에서 검토 |

**제외 사유**: 커뮤니티는 최소 1,000명 이상의 활성 사용자가 있어야 효과적. Phase 2에서 도입.

## 결과 (Consequences)

### 긍정적

1. **낮은 CAC**: 오가닉 + 인플루언서로 CAC ₩500 목표 달성 가능
2. **PMF 검증**: 유료 마케팅 전 메시지/채널 검증
3. **지속 가능한 성장**: SEO/콘텐츠 자산 축적
4. **리소스 효율**: 1인 창업자가 관리 가능한 범위

### 부정적

1. **느린 초기 성장**: 유료 광고 대비 성장 속도 느림
2. **결과 불확실성**: 오가닉 성장은 예측 어려움
3. **인플루언서 리스크**: 협업 결과 편차 큼

### 리스크

| 리스크 | 확률 | 영향 | 완화 전략 |
|--------|------|------|----------|
| 인플루언서 협업 실패 | 중간 | 중간 | 다수 협업으로 분산 |
| SEO 경쟁 심화 | 높음 | 낮음 | 롱테일 키워드 집중 |
| 바이럴 저조 | 중간 | 높음 | 결과 카드 UI 최적화 |
| 예산 초과 | 낮음 | 중간 | 월별 예산 관리 |
