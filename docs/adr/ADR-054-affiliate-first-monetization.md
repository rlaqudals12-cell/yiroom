# ADR-054: 어필리에이트 우선 수익화 전략 및 단계적 수익원 도입

## 상태

`accepted` (v2.0)

## 날짜

2026-01-24 (v1.0) → 2026-03-22 (v2.0)

## 변경 이력

| 버전 | 날짜       | 변경 내용                                              |
| ---- | ---------- | ------------------------------------------------------ |
| 1.0  | 2026-01-24 | 초기 버전 (어필리에이트 단일 수익원)                   |

### v2.0 변경 근거

v1.0의 다음 문제를 해결:

1. **ARPU 과대 추정**: ₩500-1,000 → 실제 계산 ₩4 (어필리에이트만)
2. **파트너 불일치**: iHerb(API 없음), 무신사(패션 특화) → ADR-067에서 이미 제외
3. **광고 수익 신뢰도 리스크**: 배민 울트라콜 사례 → 분석 기반 앱에서 더 치명적
4. **비용 모델 부재**: 솔로 파운더 vs 팀 운영 시나리오 구분 없음
5. **LTV/CAC 계산 오류**: ARPU ₩2,000 기반 LTV ₩20,000 → 비현실적

## 결정 (Decision)

### Phase 구조 (4단계)

```
Phase 1A: 수익 학습 기간 (MAU 0 ~ 3,000)
├── 수익원: 어필리에이트만
├── ARPU: ₩2-4 (보수적 3% 시 ₩2, 카테고리 가중평균 6% 시 ₩4. 사실상 무의미)
├── 파트너: 쿠팡 파트너스
├── 목표: PMF 검증 + 리텐션 데이터 확보
└── 결제 시스템: 없음

Phase 1B: 초기 수익화 (MAU 3,000 ~ 14,000)
├── 수익원: 어필리에이트 + 1회 구매
├── ARPU: ₩50-150 (1회 구매 포함, 시간이 지나면 감소)
├── 파트너: 쿠팡 + CJ Affiliate (2026 Fall~)
├── 목표: WTP 검증 + 구독 수요 사전 조사
└── 결제 시스템: PG 연동 (1회 결제만)

Phase 2: 구독 도입 (MRR ₩100만 달성 가능 시점)
├── 수익원: 어필리에이트 + 1회 구매 + 구독
├── ARPU: ₩500-1,500
├── 구독: Premium ₩4,900/월, Pro ₩9,900/월
├── 파트너: 쿠팡 + CJ Affiliate
├── 목표: MRR 성장, LTV/CAC 3:1 달성
└── 전환 기준: 구독 전환율 1.5% × MAU ≥ 204명 (MRR ₩100만)
    → 약 MAU 14,000 시점 (보수적 추정)

Phase 3: B2B 확장 (MAU 30,000+)
├── 수익원: 어필리에이트 + 1회 구매 + 구독 + B2B API
├── ARPU: ₩2,000-3,000
├── B2B: 뷰티 브랜드에 분석 API 라이선싱
├── 파트너: 쿠팡 + CJ + Amazon Creators
└── 목표: 수익원 다각화, 글로벌 확장 준비
```

### ARPU 산출 근거 (보수적)

```
어필리에이트 ARPU 계산 (쿠팡 3% 커미션):
MAU × 분석완료율(60%) × 제품클릭률(15%) × 구매전환율(3%) × 주문액(₩25,000) × 커미션(3%)
= MAU × 0.6 × 0.15 × 0.03 × 25,000 × 0.03
= MAU × ₩2.025
→ ARPU ≈ ₩2/월 (쿠팡 3%)

참고: commission.ts에 카테고리별 3-8% (skincare 7%) 정의되어 있으나,
      쿠팡 파트너스 실제 정책 확인 전까지 보수적 3%로 계산.
      실제 확인 후 상향 가능.
```

### 1회 구매 상품 설계

| 상품                  | 가격   | 갱신 주기                           | 용도                     |
| --------------------- | ------ | ----------------------------------- | ------------------------ |
| AI 분석 리포트 PDF    | ₩2,900 | **분기별 갱신** (시즌 변화 반영)    | 상세 분석 결과 영구 보관 |
| VTO 전체 카테고리     | ₩1,900 | 1회 영구 언락                       | 립/블러셔/헤어 전체      |
| Before/After 타임라인 | ₩2,900 | **월별 갱신** (새 분석 시 업데이트) | 변화 추적 기록           |

**ARPU 0 수렴 대응 전략:**

1. **분기별 갱신 리포트**: 같은 분석이어도 계절 변화를 반영하여 새 리포트 생성 → 반복 구매 유도
2. **구독 전환 퍼널**: 1회 구매 3회 이상 사용자에게 "구독이 더 저렴합니다" 안내
3. **신규 상품 파이프라인**: Phase 2 전까지 2-3개 신규 1회 구매 상품 추가

**1회 구매 법적 요구사항 (전자상거래법):**

| 항목       | 요구사항                                   | 비고                     |
| ---------- | ------------------------------------------ | ------------------------ |
| 청약 철회  | 디지털 콘텐츠는 다운로드 시작 후 철회 불가 | 구매 전 고지 필수        |
| 환불 정책  | 미다운로드 시 7일 이내 환불 가능           | 환불 프로세스 구현       |
| 세금계산서 | 건당 ₩1,000 이상 시 간이영수증 발행        | PG사 통해 자동 처리      |
| 미성년자   | 법정대리인 동의 없이 취소 가능             | 나이 확인 또는 결제 한도 |

**1회 구매 vs 구독 카니발리제이션 방지:**

```
1회 구매: "결과물 소유" (PDF 다운로드, VTO 언락)
구독:     "서비스 이용" (무제한 분석, AI 코치, 히스토리)

→ 다른 가치 제안이므로 직접 경쟁하지 않음
→ 1회 구매 경험이 구독 전환의 징검다리 역할
```

### 광고 영구 삭제 결정

**삭제하는 것 (영구):**

| 유형                       | 삭제 사유                             |
| -------------------------- | ------------------------------------- |
| 디스플레이 광고 (AdMob 등) | UX 저하, 브랜드 이미지 훼손           |
| Promoted Products          | 추천 신뢰도 훼손 (배민 울트라콜 문제) |
| CPC/CPM 네트워크 광고      | 사용자 데이터 제3자 공유 리스크       |

**삭제 근거:**

```
이룸의 핵심 가치: "과학적 분석 기반 추천"
광고가 섞이면:    "이게 과학인지 광고인지 모르겠음"

배민 울트라콜 사례:
- 2020: "돈 내면 상단 노출" → 공정거래위 조사
- 사용자 불신 → 리텐션 하락

이룸 적용 시 더 치명적:
- 건강/신체 데이터 기반 앱에서 광고 = 더 민감한 반응
- "내 피부 데이터로 장사하네?" → 앱 삭제
- 분석 결과의 신뢰도가 무너지면 앱의 존재 이유 소멸
```

**향후 검토 가능한 것 (별도 ADR 필요):**

| 유형                   | 정의                                                  | 검토 시점   |
| ---------------------- | ----------------------------------------------------- | ----------- |
| 브랜드 콘텐츠 파트너십 | 이룸 × 브랜드 공동 콘텐츠 제작 (사용자에게 가치 제공) | MAU 50,000+ |

```
예시: "이니스프리 × 이룸: 봄웜톤을 위한 봄 스킨케어 가이드"
→ 광고가 아니라 콘텐츠 (사용자에게 실질적 가치)
→ 별도 ADR에서 결정 (ADR-054 범위 밖)
```

### 파트너 전략 (ADR-067 정합)

| Phase | 파트너            | 커미션      | 비고                               |
| ----- | ----------------- | ----------- | ---------------------------------- |

| 2     | + CJ Affiliate    | 5-10%       | 2026 Fall Sephora 진출 시          |

**단일 파트너 리스크 완화 (Phase 1A-1B):**

- Phase 1A ARPU ₩4이므로 쿠팡 정책 변경의 재무 영향 미미
- 시드 데이터 폴백 유지
- CJ Affiliate 가입 조건 사전 확인 (최소 트래픽 요구사항)

**제외 파트너 (ADR-067 결정 준용):**

- iHerb: 공식 API 없음, 크롤링은 법적 리스크
- 무신사: 패션 특화, 뷰티 커버리지 낮음
- 올리브영: 직접 API 미제공

### B2B 도입 기준

```
기준: MAU 30,000+ (단일 기준, OR 조건 없음)

근거:
- MAU 30,000 = B2C에서 기술 성숙도 증명
- 투자 유치 여부와 무관하게 적용
- Series A 없이도 MAU 30K이면 B2B 파일럿 가능
- 반대로, MAU 5K인데 Series A 받았다고 B2B 시작하면 고객 없음

예외:
- LVMH 등 외부 기회가 능동적으로 들어올 경우 대응 가능
- 단, 이룸이 능동적으로 B2B 영업하는 것은 MAU 30K+ 이후
```

| 월 비용 | ₩57만        | ₩60만        | ₩80만         | ₩100만+      |

---

## 근거 (Rationale)

### 1. Phase 1A에서 어필리에이트를 "수익원"이 아닌 "학습 도구"로 정의

ARPU ₩4는 MAU 8,000이어도 월 ₩32,000 (커피 6잔). 이것은 수익이 아니라:

- 사용자 구매 행동 데이터 수집
- 제품 추천 CTR/CVR 최적화
- 1회 구매/구독 설계를 위한 WTP 신호 탐지

### 2. 광고 영구 삭제가 장기적으로 유리

| 모델      | 단기 (M1-M24)         | 장기 (M24+)                                |
| --------- | --------------------- | ------------------------------------------ |
| 광고 포함 | 월 +₩200만 (M24 기준) | 신뢰도 하락 → 리텐션 감소 → MAU 정체       |
| 광고 삭제 | ₩0                    | 신뢰도 유지 → K-Factor 2배 → MAU 가속 성장 |

K-Factor (바이럴 계수):

- 광고 포함: 0.2 ("근데 광고 좀 있어")
- 광고 삭제: 0.4 ("진짜 괜찮아, 광고도 없어")
  → 바이럴 2배 차이 → 장기 MAU에 훨씬 큰 영향

### 3. MRR 기반 구독 도입이 MAU 기반보다 합리적

MAU 기준의 문제: MAU 10,000이어도 전환율이 0.5%면 구독 50명 = ₩24.5만/월 (결제 시스템 ROI 부정)
MRR 기준의 장점: "₩100만 MRR 달성 가능"이 확인되면 결제 시스템 구축 정당화

## 영향 (Consequences)

### 긍정적

1. **ARPU 예측 현실화**: 투자자에게 정직한 숫자 제시 가능
2. **핵심 가치 보존**: 광고 없는 순수 AI 분석 → 차별화 유지
3. **단계적 복잡도 관리**: 결제 시스템은 Phase 1B부터 (Phase 1A는 결제 없음)
4. **바이럴 가속**: 광고 없는 앱 → 추천 시 긍정적 반응 → K-Factor 2배

### 부정적

1. **수익 규모 축소**: M24 월 ₩250만 (v1.0 대비 50% 감소)
2. **BEP 지연**: M15 (v1.0 M5 대비 10개월 지연)
3. **14개월 적자**: 누적 최대 적자 -₩490만, 생존 자금 필요
4. **광고 수익 영구 포기**: MAU 100만에서도 광고 ₩0

### 중립적

1. **커미션율 불확실**: commission.ts 3-8% vs 쿠팡 실제 3% — 확인 후 상향 가능
2. **1회 구매 ARPU 감소 추세**: 갱신 상품으로 완화 가능하나 미검증
