# ADR-113: 표현 레이어 분리 — AI 보정·가상 착장

**Status**: Accepted
**Date**: 2026-07-09
**Deciders**: 사용자(창업자) + Claude Code
**관련**: ADR-007(정직성/Mock), ADR-111(One Canon), [리서치](../research/claude-ai-research/2026-07-image-gen-tryon-beautify.md)

---

## 배경

사용자 요구 3종: ①본인 사진에 추천 옷을 입혀보는 가상 착장(바이럴·"AI 추천대로 입으니 이렇게 되네") ②공유용 AI 보정(쌩얼 공유 기피 → 공유율 저하 해소) ③미래 변화 예측. 리서치·사용자 결정: 착장=FASHN.ai(장당 ~110원), 보정=자연 보정만(Gemini 이미지 편집, 장당 ~57원), 예측=이미지 시뮬 보류(화장품법 리스크, 문헌 타임라인 텍스트로 대체 — ADR-112에서 구현 완료).

**핵심 긴장**: 생성·보정 이미지가 분석 파이프라인에 섞이면 이룸의 존재 기반(정직한 진단, ADR-007 재현성)이 무너진다.

## 결정

1. **진실 레이어 / 표현 레이어 물리 분리** — 신규 모듈 `lib/visual-expression/`(배럴 export, P8)에만 이미지 생성·편집 코드를 둔다. 이 모듈의 산출물은:
   - 분석 입력(analyze API·이미지 업로드 경로)으로 **절대 재진입 금지** — 분석 경로와 저장 버킷/경로 분리.
   - 분석 결과·DB 지표에 어떤 값도 쓰지 않는다(읽기만 허용).
   - 산출 이미지에는 `aiEdited: true` 메타를 동반하고, UI에 "AI 보정됨"/"AI 생성 이미지" 라벨을 상시 표시(Gemini SynthID 워터마크 + 한국 AI 표시 의무 — 은폐 불가 전제).
2. **보정 = 자연 보정만** — Gemini 이미지 편집(나노바나나 계열, 표준 클라이언트 경유). 프롬프트는 피부 결·잡티·조명 정돈으로 한정하고 **이목구비·윤곽 변형을 명시적으로 금지**. 공유 흐름 전용(분석 결과 화면의 원본 표시는 불변).
3. **가상 착장 = FASHN.ai 어댑터** — `PartnerAdapter`식(OCP, ADR-032 패턴) `lib/visual-expression/tryon/`. `FASHN_API_KEY` env 없으면 기능 표면 자체를 렌더하지 않음(준비중 배너 금지 — 유령 UI 방지). 입력: 사용자 전신/상반신 사진 + 의류 이미지(옷장/추천 코디), 출력: 착장 이미지(+AI 라벨).
4. **비용 가드**: 세션당 생성 상한(보정+착장 합산 5장) — 파산 방지 상한 정책. 초과 시 안내.
5. **예측 이미지("며칠 후 변해요")는 도입하지 않음** — ADR-112 문헌 타임라인이 대체. 법무 검토 전 재론 금지.

## 대안과 트레이드오프

- **분석 파이프라인에 보정 통합(전처리)** — 기각: 진단 왜곡, 재현성 붕괴. 분리 원칙이 이 ADR의 존재 이유.
- **자체 try-on 모델(IDM-VTON 셀프호스팅)** — 기각: 운영 복잡도, 1인 개발 규모 부적합. API가 P4 부합.
- **보정 강도 사용자 선택(강한 보정)** — 기각(사용자 결정): "이룸 결과가 뽀샵" 인식 리스크.

**수용한 트레이드오프**: FASHN.ai 키·크레딧은 사용자 발급 대기(그동안 착장 표면 비노출) / AI 라벨 상시 노출로 바이럴 카피는 "AI가 입혀준 모습"을 전제로 설계 / 장당 비용 발생(세션 ~200-300원, 정책 내).

## 결과

- 신규 모듈: `lib/visual-expression/` (beautify + tryon 어댑터 + 비용 가드).
- 공유 흐름에 "내 사진으로 공유(자연 보정)" 옵션 추가 — 기존 익명(일러스트) 공유는 유지(정본).
- 착장은 코디 추천 표면(옷장/체형 결과의 코디 카드) 위에만 추가(One Canon — 새 페이지 금지).

## 검증 기준

- 표현 레이어 산출물이 분석 API 입력·DB 지표에 도달하는 경로 0 (import 방향 테스트/리뷰).
- 보정 프롬프트에 이목구비 변형 금지 명시 + AI 라벨 렌더 테스트.
- 키 부재 시 착장 표면 비노출 테스트.
