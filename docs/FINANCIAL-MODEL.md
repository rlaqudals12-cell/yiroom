## 3. 손익분기점(BEP) 분석

### 3.1 BEP 산출 공식

```typescript
interface BEPModel {
  fixedCosts: number;           // 고정비
  variableCostPerUser: number;  // 사용자당 변동비
  arpu: number;                 // 사용자당 평균 매출
}

function calculateBEP(model: BEPModel): number {
  // BEP MAU = 고정비 / (ARPU - 변동비)
  const contributionMargin = model.arpu - model.variableCostPerUser;
  return Math.ceil(model.fixedCosts / contributionMargin);
}

// Phase 1 (Pre-seed) BEP 계산
const preSeedBEP = calculateBEP({
  fixedCosts: 17000000,      // ₩17M
  variableCostPerUser: 15.7, // AI 비용 ₩15.7/MAU
  arpu: 500,                 // 어필리에이트만 (Phase 1)
});
// BEP = 17,000,000 / (500 - 15.7) ≈ 35,100 MAU

// Phase 2 (Seed) BEP 계산 - ARPU 증가
const seedBEP = calculateBEP({
  fixedCosts: 45000000,
  variableCostPerUser: 15.7,
  arpu: 2000,               // 어필리에이트 + 구독
});
// BEP = 45,000,000 / (2,000 - 15.7) ≈ 22,700 MAU
```
