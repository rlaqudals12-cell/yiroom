## 1. Hook Model 개요

### 🔄 4단계 사이클

```yaml
Hook Model:
  1. Trigger (계기)
     → 사용자가 앱을 여는 이유
     
  2. Action (행동)
     → 사용자가 취하는 간단한 행동
     
  3. Reward (보상)
     → 기대 이상의 가치 제공
     
  4. Investment (투자)
     → 다음 사용을 위한 준비

순환 구조:
  Trigger → Action → Reward → Investment → (다시 Trigger)
  
목표:
  ✅ 7일 리텐션 50% 이상
  ✅ 모듈 간 전환율 60% 이상
  ✅ 재방문율 주 2-3회
```

### 🎯 핵심 원칙

```yaml
1. Action은 최대한 간단하게
   ❌ 나쁜 예: 20개 질문 + 5장 사진 + 10분 대기
   ✅ 좋은 예: 1장 사진 + 30초 분석 + 즉시 결과

2. Reward는 예상을 뛰어넘게
   ❌ 나쁜 예: "당신은 건성입니다"
   ✅ 좋은 예: "지난주보다 모공 12% 개선! 수면 패턴이 영향을 줄 수 있어요"

3. Investment는 자연스럽게
   ❌ 나쁜 예: "구독하세요!" (강요)
   ✅ 좋은 예: "결과가 저장되었습니다. 다른 분석도 받아보시겠어요?"

4. Trigger는 적절한 타이밍에
   ❌ 나쁜 예: 하루에 10번 푸시 알림
   ✅ 좋은 예: 아침 세안 후, 주간 변화 알림
```

## 2. 전체 사용자 여정

### 🚀 첫 방문 (Day 1)

#### Trigger: 외부 트리거
```yaml
유입 경로:
  - 인스타 광고: "사진 1장으로 AI 분석"
  - 친구 추천: "이거 정말 신기해, 써봐"
  - 검색: "퍼스널 컬러 진단 무료"

랜딩 메시지:
  "4개 앱 쓸 필요 없어요
   사진 한 장이면 AI가 모든 웰니스를 분석합니다"
```

#### Action: 초간단 가입
```yaml
Step 1: 소셜 로그인
  - 구글 / 카카오 / 애플
  - 클릭 1번으로 가입
  - 이메일 입력 불필요
  
Step 2: 온보딩 선택
  "무엇을 먼저 분석해볼까요?"
  
  [퍼스널 컬러 진단] ← 추천
  [피부 분석]
  [체형 분석]
  
  → 대부분 PC-1을 선택하도록 유도
```

#### Reward: 첫 경험
```yaml
PC-1 진단 (2-3분):
  - 문진 10개 (진행률 표시)
  - 셀카 1장
  - 30초 분석
  
결과 화면:
  🎨 "당신은 Summer 타입입니다!"
  
놀라운 정보 3가지:
  ⭐ "Summer는 전체의 18%만 해당됩니다"
  ⭐ "같은 타입 연예인: 아이유, 윤아, 수지"
  ⭐ "신뢰도 95% - 매우 높음"
  
색상 팔레트 30개
베스트 컬러 TOP 5
```

#### Investment: 다음 단계 유도
```yaml
자동 저장:
  "✅ 결과가 저장되었습니다!"
  
다음 모듈 제안:
  "피부 분석도 받아보시겠어요?
   퍼스널 컬러 기반 파운데이션 추천을 받을 수 있습니다!"
  
  [피부 분석 받기] ← 60% 전환 목표
  [나중에 하기]

Social Investment:
  "친구 초대하면 프리미엄 기능 1주일 무료!"
  [카카오톡으로 공유]
  [링크 복사]
```

### 🔁 재방문 (Day 2-7)

#### Trigger: 내부 트리거 활성화
```yaml
내부 트리거 (감정/상황):
  - "오늘 피부가 안 좋네..." 😟
  - "어제보다 나아졌나?" 🤔
  - "새 옷 샀는데 어울릴까?" 👗
  - "친구 결과랑 비교해보고 싶어" 👥

외부 트리거 (푸시 알림):
  Day 2: "오늘 피부 컨디션 체크해볼까요?"
  Day 5: "지난주보다 얼마나 개선됐는지 확인해보세요!"
  Day 7: "7일 연속 기록하면 프리미엄 리포트 무료!"
```

#### Action: 빠른 재사용
```yaml
S-1 피부 분석:
  - 로그인 유지 (자동)
  - 사진 1장만 찍기
  - 30초 분석
  - 즉시 결과

C-1 체형 분석:
  - 전신 사진 1장
  - 30초 분석
  - 즉시 결과
```

#### Reward: 가변적 보상 (핵심!)
```yaml
S-1 결과:
  고정 보상:
    - 피부 타입, 7가지 지표
    
  가변 보상 (기대를 뛰어넘는):
    ⭐ "지난주보다 모공 12% 개선!"
    ⭐ "수분 부족이 감지되었습니다 (-23%)"
    ⭐ "수면 부족이 피부에 영향을 줄 수 있어요"
    ⭐ "당신 피부톤에 어울리는 파운데이션 3개 추천"

C-1 결과:
  고정 보상:
    - 체형 타입, 치수
    
  가변 보상:
    ⭐ "X자 체형 + Summer = 이 색상 조합이 완벽해요"
    ⭐ "이번 시즌 트렌드 중 어울리는 스타일"
    ⭐ "체중 -1.2kg, 허리 -2cm 변화 감지!"
```

#### Investment: 데이터 축적
```yaml
대시보드에서 확인:
  📸 피부 사진 7장
  📊 변화 그래프 (개선 추세)
  🔥 7일 연속 기록 중
  
Progress Tracker:
  "목표까지 3일 남았어요!"
  "7일 달성 시 프리미엄 인사이트 리포트 제공"
  
데이터 가치 강조:
  "이 기록은 당신만의 웰니스 히스토리입니다
   떠나면 모든 데이터를 잃게 됩니다"
```

## 3. PC-1 퍼스널 컬러 진단

### 전체 Flow
```yaml
온보딩 필수 모듈:
  - 모든 사용자가 가장 먼저 진행
  - S-1, C-1의 기반 데이터 제공
  - 한 번만 하면 평생 활용
```

### 1️⃣ Trigger

#### 내부 트리거 (감정)
```yaml
불안/의심:
  - "내 퍼스널 컬러가 뭘까?"
  - "이 색이 나한테 어울리나?"
  - "립스틱 샀는데 이상한데..."
  
호기심:
  - "연예인이랑 같은 타입일까?"
  - "친구는 Spring이래, 나는 뭘까?"
  
FOMO (놓치고 싶지 않음):
  - "요즘 다들 퍼스널 컬러 하던데"
```

#### 외부 트리거
```yaml
광고 카피:
  "사진 한 장으로 퍼스널 컬러 진단
   정확도 95% AI 분석"

친구 추천:
  "이거 해봐, 정말 신기해
   나랑 같은 타입인지 확인해봐"

SNS 공유:
  "나 Summer 타입 나왔어!
   너도 해봐 👇 링크"
```

### 2️⃣ Action

#### 설계 원칙
```yaml
최대한 간단하게:
  ❌ 30분 문진 + 5장 사진
  ✅ 3분 문진 + 1장 사진
  
명확한 진행률:
  "1/10" "2/10" ... "완료!"
  
즉시 피드백:
  질문 답변 → 즉시 다음 질문
  사진 업로드 → 즉시 미리보기
```

#### 구현 상세

**Step 1: 문진 (2-3분)**
```yaml
화면 설계:
  - 큰 질문 텍스트
  - 2-4개 선택지 (버튼 크게)
  - 진행률 바 (1/10, 2/10...)
  - 뒤로가기 가능
  
질문 10개:
  1. 손목 혈관 색은?
  2. 금 vs 은 장신구?
  3. 피부 톤은?
  4. 헤어 컬러는?
  5. 눈동자 색은?
  6. 홍조가 잘 올라오나요?
  7. 햇빛에 노출되면?
  8. 입술 색은? (립스틱 안 바른)
  9. 어울리는 색상 계열은?
  10. 성별/나이대

UX 최적화:
  - 각 질문 5초 이내 답변 가능
  - 총 2-3분 소요
  - 중간 저장 (나갔다 와도 이어서)
```

**Step 2: 사진 업로드 (30초)**
```yaml
화면 안내:
  "셀카 1장만 찍어주세요
   
   팁:
   ✅ 자연광에서
   ✅ 정면 얼굴
   ✅ 화장 안 한 상태"

촬영 옵션:
  [카메라로 찍기]
  [갤러리에서 선택]

즉시 피드백:
  - 업로드 즉시 미리보기
  - "분석 중... 30초만 기다려주세요"
  - 로딩 중 팁 표시:
    "퍼스널 컬러는 태어날 때부터 결정됩니다"
    "4계절 중 하나에 꼭 속합니다"
    "90% 이상의 정확도로 분석됩니다"
```

### 3️⃣ Reward

#### 결과 화면 설계 (핵심!)

**메인 결과**
```yaml
큰 글씨 + 애니메이션:
  🎨 "당신은 Summer 타입입니다!"
  
계절별 이미지/컬러:
  - Spring: 따뜻하고 밝은 이미지
  - Summer: 시원하고 부드러운 이미지
  - Autumn: 따뜻하고 깊은 이미지
  - Winter: 차갑고 선명한 이미지
```

**가변 보상 1: 희소성**
```yaml
통계 정보:
  📊 "Summer 타입은 한국인의 18%만 해당됩니다"
  
  Spring:  25%
  Summer:  18% ← 당신
  Autumn:  30%
  Winter:  27%
  
메시지:
  "특별한 당신을 위한 맞춤 추천을 준비했어요!"
```

**가변 보상 2: 사회적 증거**
```yaml
같은 타입 연예인:
  ⭐ "Summer 타입 연예인"
  
  - 아이유
  - 윤아 (소녀시대)
  - 수지
  - 한소희
  - 공유
  
  사진 3-5장 표시
  
메시지:
  "이 분들과 같은 컬러를 공유하고 있어요!"
```

**가변 보상 3: 신뢰도**
```yaml
정확도 표시:
  ✅ 신뢰도 95%
  
  문진 점수: 88점 (Summer)
  이미지 분석: 92점 (Summer)
  통합 신뢰도: 95% ⭐
  
메시지:
  "매우 높은 정확도로 분석되었습니다!"
```

**핵심 추천**
```yaml
베스트 컬러 TOP 5:
  [컬러칩 1] Soft Rose #FFB6C1
  [컬러칩 2] Lavender #E6E6FA
  [컬러칩 3] Sky Blue #87CEEB
  [컬러칩 4] Mint #98FB98
  [컬러칩 5] Dusty Pink #FFCCE5
  
어울리는 립스틱:
  - 로즈 핑크
  - 라벤더 핑크
  - 코랄 핑크 (피하기)
  
어울리는 옷:
  - 파스텔 톤
  - 차분한 색상
  - 피할 색: 네온, 오렌지
```

### 4️⃣ Investment

#### 즉시 투자

**자동 저장**
```yaml
알림 (애니메이션):
  ✅ "결과가 저장되었습니다!"
  
  "이 정보는 앱 전체에서 활용됩니다:
   - 피부 분석 시 파운데이션 추천
   - 체형 분석 시 코디 색상 추천
   - 영양/운동 모듈에서도 활용 예정"
```

**다음 단계 유도**
```yaml
카드 형태로 제안:
  
  💡 추천: 피부 분석 받기
  
  "Summer 타입에 맞는 파운데이션을
   정확하게 추천받을 수 있어요!
   
   피부 상태 + 퍼스널 컬러 = 완벽한 매칭"
  
  [피부 분석 받기] ← 큰 버튼
  [나중에 하기]
  
전환율 목표: 60%
```

**Social Investment**
```yaml
공유 유도:
  "친구도 자신의 타입이 궁금할 거예요!"
  
  혜택:
    - 친구 1명 가입 → 프리미엄 기능 3일
    - 친구 3명 가입 → 프리미엄 기능 1주일
    - 친구 5명 가입 → 프리미엄 기능 1개월
  
  [카카오톡으로 공유]
  [인스타 스토리로 공유]
  [링크 복사]
```

## 4. S-1 피부 분석

### 전체 Flow
```yaml
반복 사용 모듈:
  - 매일/매주 사용
  - 변화 추적이 핵심
  - PC-1 데이터 활용
```

### 1️⃣ Trigger

#### 내부 트리거 (일상적 감정)
```yaml
아침 루틴:
  - "오늘 피부 어떤가?" (거울 보면서)
  - "어제보다 나아졌나?"
  
불안/걱정:
  - "뾰루지 났네..." 😰
  - "피부가 거칠어진 것 같아" 😟
  - "모공이 커진 것 같은데" 😥
  
호기심:
  - "어제 치킨 먹었는데 영향 있나?"
  - "새 화장품 써도 될까?"
  - "1주일 지났는데 얼마나 개선됐지?"
```

#### 외부 트리거 (푸시 알림)
```yaml
타이밍 최적화:

아침 루틴:
  시간: 08:00
  메시지: "좋은 아침이에요! 오늘 피부 컨디션 체크해볼까요?"
  빈도: 매일

변화 감지:
  조건: 3일 연속 개선
  메시지: "피부가 좋아지고 있어요! 확인해보세요"
  빈도: 변화 시

재참여 유도:
  조건: 7일 미사용
  메시지: "피부 변화를 놓치지 마세요! 1주일째 분석 안 하셨네요"
  빈도: 1회

연속 기록 (Streak):
  조건: 6일 연속 기록
  메시지: "내일 분석하면 7일 연속! 프리미엄 리포트 받으세요"
  빈도: Streak 발생 시

주간 리포트:
  시간: 일요일 20:00
  메시지: "이번 주 피부 변화 리포트가 도착했어요!"
  빈도: 주 1회
```

### 2️⃣ Action

#### 설계 원칙
```yaml
초간단 사용:
  - 사진 1장만
  - 30초 분석
  - 즉시 결과
```
