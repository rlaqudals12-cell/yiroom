## 1. 핵심 개념

### 1.1 피부 시술의 4대 에너지 원리

모든 피부 시술은 다음 4가지 에너지 전달 방식 중 하나 이상을 활용한다:

| 에너지 유형 | 대표 시술 | 타겟층 | 작용 메커니즘 |
|------------|----------|--------|--------------|
| **광(레이저)** | 클라리티, 레이저토닝 | 표피-진피 | 선택적 광열분해 |
| **전자기파(RF)** | 써마지, 볼뉴머 | 진피-피하 | 분자 마찰열 |
| **음파(HIFU)** | 울쎄라, 슈링크 | SMAS층 | 초점 열응고 |
| **화학적** | 주사, 필링 | 표피-진피 | 분자 작용 |

### 1.2 콜라겐 재생의 생물학적 타임라인

열에너지가 조직에 도달하면 두 단계의 효과가 순차적으로 발생:

**즉각 효과 (Immediate Effect)**
- 65°C에서 콜라겐 삼중나선 구조의 수소결합 파괴
- 65°C: 11% 수축 / 80°C: 최대 59% 수축
- 시술 직후 느껴지는 당김감의 원인

**지연 효과 (Delayed Effect)** - 상처 치유 반응 3단계:

| 시기 | 단계 | 조직 반응 |
|------|------|----------|
| Day 2-10 | 염증기 | 대식세포 탐식, HSP47 발현 시작 |
| Day 28 | 증식기 | 섬유아세포 활성화, Type III 콜라겐 합성 |
| 2-3개월 | 성숙기 | 콜라겐 생성 피크, 가시적 개선 |
| 3-6개월 | 리모델링 완료 | Type III → Type I 콜라겐 교체, **최대 효과** |
| 10-12개월 | 완전 성숙 | Type I 콜라겐 교차결합 강화 |

### 1.3 SMAS층의 구조적 중요성

**SMAS (Superficial Musculo-Aponeurotic System)**:
- 피하지방 아래 **3-4.5mm 깊이**에 위치
- 콜라겐 + 탄성섬유 + 표정근육으로 구성된 근건막 시스템
- 피부와 수직 격막으로 연결되어 표정 움직임 전달
- 수술적 페이스리프트의 핵심 타겟
- HIFU(4.5mm 카트리지)로 비침습적 접근 가능

## 2. 물리학적 기반

### 2.1 선택적 광열분해 (Selective Photothermolysis)

1983년 Anderson & Parrish (Science)가 발표한 현대 레이저 피부과학의 근간:

```
핵심 3요소:
1. 타겟 발색단이 우선 흡수하는 파장 선택
2. 발색단의 열 확산 시간(TRT)보다 짧은 펄스 폭
3. 타겟 파괴에 충분한 에너지 밀도(Fluence)
```

**열 확산 시간 (Thermal Relaxation Time)** 계산:

```
TRT = d² / 16α

d = 타겟 직경
α = 열 확산 계수

예시:
- 멜라노좀(0.5-1μm): TRT = 50-500 나노초
- 모낭(200-300μm): TRT = 10-100 밀리초
```

**발색단별 최적 파장**:

| 파장 범위 | 침투 깊이 | 주요 발색단 | 대표 시술 |
|-----------|----------|------------|----------|
| 532nm | 0.8mm | 멜라닌, 헤모글로빈 | IPL, KTP |
| 755nm | 2.5-3mm | 멜라닌 | 클라리티(알렉산드라이트) |
| 1064nm | 5-6mm | 멜라닌(약), 헤모글로빈 | 레이저토닝, Nd:YAG |
| 1927nm | 표재성 | 수분 | 프락셀 |
| 10,600nm | 0.1mm | 수분 | CO2 레이저 |

멜라닌은 **300-500nm**에서 최대 흡수, 파장이 길어질수록 흡수율 지수적 감소.
→ 1064nm는 표피 멜라닌을 우회하여 어두운 피부에도 안전.

### 2.2 고주파 (Radiofrequency) 원리

**기술 사양**:
- 주파수: **6.78MHz** (의료용 표준)
- 최대 출력: 400W
- 침투 깊이: 진피층 및 피하조직 (약 4.3mm)

**작용 메커니즘**:
```
전자기파 → 조직 통과 → 이온/수분 분자 진동 → 분자 마찰열 (Joule heating)
→ 볼류메트릭 히팅 (3D 입체 가열)
```

**Monopolar vs Bipolar 비교**:

| 항목 | Monopolar (써마지) | Bipolar (인모드) |
|------|-------------------|-----------------|
| 전극 배치 | 핸드피스 + 접지패드 | 핸드피스 내 양극 |
| 에너지 경로 | 깊고 넓음 | 표면 집중 |
| 침투 깊이 | 4.3mm | 2-4mm |
| 장점 | 체적 가열 | 정밀 제어 |

**콜라겐 변성 온도**: 60-70°C
→ 즉각적 수축 + 2-6개월 신콜라겐 생성

### 2.3 집속초음파 (HIFU) 원리

**기술 사양**:
- 주파수: **4-10MHz** (기계적 음파)
- TCP 크기: 직경 0.5-1mm
- 초점 온도: **65-75°C**

**작용 메커니즘**:
```
초음파 빔 집속 → 초점에서 분자 진동 + 마찰
→ 열응고점 (TCP, Thermal Coagulation Point) 형성
→ 1mm³ 이하의 미세한 국소 열 손상
→ 콜라겐 수축 + 상처 치유 반응
```

**깊이별 트랜스듀서 (울쎄라 기준)**:

| 트랜스듀서 | 주파수 | 타겟층 | 주요 적응증 |
|-----------|--------|--------|------------|
| 1.5mm | 10MHz | 진피 상부 | 잔주름, 피부결 |
| 3.0mm | 7MHz | 진피 심부 | 탄력, 콜라겐 재생 |
| 4.5mm | 4MHz | SMAS층 | 턱선 리프팅, 처진 피부 |

**RF vs HIFU 핵심 차이**:

| 비교 항목 | RF (써마지) | HIFU (울쎄라) |
|-----------|------------|--------------|
| 에너지 | 전자기파 6.78MHz | 음파 4-10MHz |
| 가열 방식 | 체적 가열 (Volume) | 초점 가열 (Focal) |
| 타겟층 | 진피층 (얕음) | SMAS층 (깊음) |
| 주 효과 | 피부 탄력, 피부결 | 리프팅, 윤곽 개선 |
| 적합 대상 | 얇은 피부, 지방 적은 얼굴 | 두꺼운 피부, 처짐 심한 얼굴 |

## 3. 레이저 시술 원리

### 3.1 레이저토닝 - 서브셀룰러 선택적 광열분해

**작용 메커니즘**:
Q-switched 1064nm Nd:YAG를 **낮은 에너지 밀도(0.5-3.8 J/cm²)**로 반복 조사.
멜라닌 세포 자체는 보존하면서 세포 내 **성숙 멜라노좀(Stage IV)**만 선택적 파괴.

```
저출력에서:
1. 성숙 멜라노좀만 파괴, 초기 멜라노좀 보존
2. 멜라닌 세포 수지상돌기 감소
3. TRP-1, tyrosinase 발현 억제
4. 멜라닌 세포 수 유지 → 기능적 억제 상태
```

**반점성 저색소증 (Mottled Hypopigmentation) 위험 관리**:

| 위험 요인 | 저위험 | 고위험 |
|-----------|--------|--------|
| 시술 횟수 | ≤10회 | >20회 |
| 시술 간격 | ≥2주 | <1주 |
| Fluence | <2 J/cm² | >3 J/cm² |
| 피부 타입 | Type III | Type V-VI |

### 3.2 듀얼 파장 레이저 (클라리티)

**755nm + 1064nm 조합**:
- 755nm (알렉산드라이트): 멜라닌 흡수율 높음, 밝은 피부(Type I-III)
- 1064nm (Nd:YAG): 깊은 침투, 어두운 피부(Type IV-VI) 안전

**한국인 피부(Type III-IV) 적용**:
- 기본적으로 **1064nm 우선** 권장
- 과색소침착(PIH) 발생률이 Type I-III 대비 **25-47% 높음**
- 보수적 에너지 설정, 테스트 패치, 자외선 차단 필수

## 4. 주사 시술 원리

### 4.1 보툴리눔 톡신 A형 (BoNT/A)

**작용 메커니즘**:
신경근접합부에서 **SNAP-25 단백질 선택적 절단** → 아세틸콜린 분비 차단

```
1. 중쇄(Heavy Chain) → SV2 수용체 결합
2. 수용체 매개 내포작용 → 신경세포 내부 유입
3. 경쇄(Light Chain) → SNAP-25의 C-말단 9개 아미노산 절단
4. SNARE 복합체 기능 상실 → 신경전달물질 방출 차단
```

**회복 과정 (Axonal Sprouting)**:
- 2-4주: 측부 발아 형성
- 3-4개월: 원래 신경말단에서 SNARE 단백질 재합성
- 효과 지속: 3-6개월

**스킨보톡스 vs 일반 보톡스**:

| 항목 | 일반 보톡스 | 스킨보톡스 |
|------|------------|------------|
| 주입 깊이 | 근육층 | 진피층 (1-2mm) |
| 목적 | 근육 마비 → 주름 제거 | 피지/모공 조절 |
| 희석 비율 | 표준 | 2-4배 더 희석 |
| 지속 기간 | 3-6개월 | 2-4개월 |

### 4.2 히알루론산 (HA) 필러

**점탄성 특성**:

| 파라미터 | 정의 | 임상적 의미 |
|----------|------|-------------|
| **G' (탄성계수)** | 형태 유지 능력 | 높을수록 리프팅 효과 ↑ |
| **G'' (점성계수)** | 흐름 저항 | 주입 용이성 결정 |
| **응집력 (Cohesivity)** | 젤 입자 결합력 | 이동(migration) 방지 |
| **수분흡수율** | 팽윤 정도 | 눈밑은 **<150%** 권장 |

**부위별 최적 G' 매칭**:

| 부위 | 권장 G' (Pa) | 권장 수분흡수율 | 추천 제품 |
|------|-------------|-----------------|-----------|
| 볼 | 350-500+ | 저~중 | Voluma (398), Lyft (977) |
| 코 | 400+ | 저 | Volux (665), Lyft |
| 팔자 | 250-400 | 중 | Volift (340), Defyne |
| 입술 | 150-300 | 저 | Volbella (271), Kysse |
| 눈밑 | <200 | **<150%** | Volbella, Redensity II |

**혈관 폐색 응급 대처**:
- 골든타임: **90분 이내**
- 초기 증상: 창백/청색 변화, 극심한 통증
- 응급 조치: 히알루로니다제 150-300 IU 즉시 투여

### 4.3 성장인자 기반 재생 치료

**핵심 성장인자**:

| 성장인자 | 주요 역할 | 피부 재생 효과 |
|----------|-----------|----------------|
| **PDGF** | 섬유아세포 증식/이동 | ECM 합성 촉진 |
| **TGF-β** | 콜라겐 생성 촉진 | 근섬유아세포 분화 |
| **VEGF** | 신생혈관 형성 | 영양 공급, 피부 광채 |
| **EGF** | 각질세포 증식 | 재상피화 |

**PRP (Platelet-Rich Plasma)**:
- Double spin 방식: 100g 15분 → 1600g 20분
- 혈소판 농축: 4-7배
- 임상 근거: 환자 만족도 RR 1.34 (P=0.01)

### 4.4 PN (Polynucleotide) - 리쥬란

**A2A 수용체 활성화 경로**:
```
PN → A2A 아데노신 수용체 결합 → cAMP → PKA → ERK/AKT 경로 활성화
→ Type I/III 콜라겐, 엘라스틴 합성 촉진
→ IL-1β, IL-6, TNF-α 감소 (항염증)
→ VEGF 매개 신생혈관 형성
```

**제품 라인업**:

| 제품 | 특징 | 적응증 |
|------|------|--------|
| 리쥬란 힐러 | 원조, 고점도 | 전체 피부 재생 |
| 리쥬란 i | 저점도 | 눈가 전용 |
| 리쥬란 S | 고농축 | 흉터 전용 |
| 리쥬란 HB plus | PN+HA+리도카인 | 통증 감소+물광 |

## 5. 스킨케어 시술 원리

### 5.1 각질 제거의 두 가지 경로

**물리적 박리**:
- 시술: MTS, 초음파 클렌징, 아쿠아필
- 원리: 마찰력/진공 흡입으로 각질세포 직접 제거
- 주의: 과도한 사용 시 피부 장벽 손상

**화학적 박리**:
- 시술: AHA/BHA 필링
- 원리: 칼리크레인(kallikrein) 효소 활성화 → 데스모솜 결합 분해

| 필링제 | 분자량 | 용해성 | 특징 |
|--------|--------|--------|------|
| 글리콜산 (AHA) | 76Da | 수용성 | 가장 높은 침투력 |
| 살리실산 (BHA) | 138Da | 지용성 | 모공 내부 도달, 피지 용해 |
| PHA | >200Da | - | 표면 작용, 민감성 피부 적합 |

### 5.2 경피 흡수의 500 Dalton Rule

**Brick and Mortar 모델**:
- 각질세포(벽돌) + 세라마이드/콜레스테롤/유리지방산(모르타르)
- 분자량 **500Da 미만**만 효과적 투과

**흡수 증진 기술**:

| 기술 | 원리 | 흡수율 증가 |
|------|------|-------------|
| **MTS** | 미세 채널 형성 | **최대 10,000배** |
| **이온토포레시스** | 직류 전류로 이온성 약물 전기영동 | 유의미 |
| **초음파(소노포레시스)** | 캐비테이션 효과로 지질층 교란 | 중간 |

### 5.3 광생물조절 (Photobiomodulation)

**파장별 타겟과 효과**:

| 파장 | 침투 깊이 | 타겟 | 주요 효과 |
|------|-----------|------|-----------|
| 청색(415nm) | ~1mm | P. acnes 포르피린 | 살균, 여드름 치료 |
| 황색(590nm) | ~1.6mm | 멜라닌세포, 혈관 | 기미, 홍조 완화 |
| 적색(630-660nm) | 1-3mm | 섬유아세포 | 콜라겐 합성, 항염 |
| NIR(830nm) | 2-10mm | 심부 조직 | 통증 완화, 조직 재생 |

**Arndt-Schulz 법칙**:
```
약한 자극 → 세포 활동 촉진
강한 자극 → 세포 활동 억제
과도한 자극 → 세포 사멸

최적 용량: 1-20 J/cm²
과량 조사 시 효과 오히려 감소
```

### 5.4 카복시테라피 - Bohr 효과

**작용 메커니즘**:
```
CO2 주입 → 혈액 pH 감소 → 헤모글로빈 산소 해리 촉진
→ 조직 산소화 증가 → 프롤린 수산화 효소 활성
→ 콜라겐 합성 촉진
```

**혈관 변화**:
- 전모세혈관 직경: 26.9μm → 34.3μm
- VEGF 방출 → 신생혈관 형성

### 6.1 Fitzpatrick 피부 타입별 시술 적합도

| 시술 | Type I-II | Type III | Type IV | Type V-VI |
|------|----------|----------|---------|-----------|
| 레이저 755nm | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐ | ⭐⭐ |
| 레이저 1064nm | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ |
| RF (써마지/볼뉴머) | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| HIFU (울쎄라/슈링크) | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| 레이저토닝 | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐ |

**한국인(주로 Type III-IV) 권장**:
- RF/HIFU 계열: 색소 부작용 없이 안전
- 레이저 계열: 1064nm 파장 우선, 보수적 에너지 설정

### 6.2 피부 고민별 추천 매칭 로직

| 주요 고민 | 1순위 추천 | 2순위 추천 | 주의사항 |

```typescript
function getProcedureWarnings(
  procedure: string,
  userProfile: UserProfile
): string[] {
  const warnings: string[] = [];

  // 볼패임 위험
  if (['ulthera', 'shurink'].includes(procedure) && userProfile.fatVolume === 'low') {
    warnings.push('얼굴 지방이 적은 경우 볼패임 위험이 있습니다. 전문의 상담을 권장합니다.');
  }

  // PIH 위험
  if (['clarity_755', 'laser_toning'].includes(procedure) && userProfile.fitzpatrickType >= 4) {
    warnings.push('색소침착(PIH) 위험이 있습니다. 보수적 에너지 설정과 자외선 차단이 필수입니다.');
  }

  // 저색소증 위험
  if (procedure === 'laser_toning') {
    warnings.push('반복 시술 시 저색소증 위험이 있습니다. 10회 단위 평가를 권장합니다.');
  }

  return warnings;
}
```
