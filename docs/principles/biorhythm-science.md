# 바이오리듬 과학 (Biorhythm Science)

> 이 문서는 #2 웰니스/바이오리듬 통합의 기반이 되는 원리를 설명한다.

## 1. 핵심 개념

### 1.1 일주기 리듬 (Circadian Rhythm)

인체의 24시간 생체시계. 수면-각성, 호르몬 분비, 체온 변화를 조절한다.

- **멜라토닌**: 21-23시 분비 시작 → 수면 유도
- **코르티솔**: 06-08시 최고 → 각성/에너지
- **성장호르몬**: 수면 중 (특히 NREM 3-4단계) 분비 → 피부 재생

### 1.2 수면→피부 연결

수면 부족은 피부에 직접적 영향을 미친다:

| 수면 시간  | 피부 영향                             |
| ---------- | ------------------------------------- |
| 7-9시간    | 정상 재생 (최적)                      |
| 5-7시간    | 재생 능력 15-25% 감소                 |
| 5시간 미만 | 재생 능력 40%+ 감소, 콜라겐 합성 저하 |

**근거**: Oyetakin-White et al. (2015) "Sleep Quality and Skin Aging" - 수면 질이 낮은 그룹에서 TEWL(경피수분손실) 30% 증가

### 1.3 스트레스→뷰티 연결

코르티솔 과다 분비(만성 스트레스)의 뷰티 영향:

- 피지 분비 증가 → 트러블
- 콜라겐/엘라스틴 분해 촉진 → 주름
- 모발 성장 주기 교란 → 탈모
- 멜라닌 과다 생성 → 색소침착

### 1.4 신체 주기 (Body Cycles)

#### 여성 (생리 주기)

28±7일 주기, 4단계:

| 단계   | 기간      | 호르몬          | 피부/운동 영향               |
| ------ | --------- | --------------- | ---------------------------- |
| 생리기 | 1-5일     | 낮은 에스트로겐 | 피부 민감↑, 저강도 운동 권장 |
| 여포기 | 6-13일    | 에스트로겐↑     | 피부 최상, 고강도 운동 가능  |
| 배란기 | 14일 전후 | 에스트로겐 최고 | 피부 윤기↑, 최고 운동 성과   |
| 황체기 | 15-28일   | 프로게스테론↑   | 피지↑, 트러블 주의, 중강도   |

#### 남성/공통 (계절/환경 리듬)

- 계절: 자외선→색소, 건조→보습, 꽃가루→민감
- 수면 패턴: 일관된 취침/기상 시간의 중요성
- 운동 주기: 과훈련 방지를 위한 휴식일

## 2. 바이오리듬 점수 공식

### 2.1 수면 점수 (0-30)

```
sleepScore = sleepDurationScore(hours) + sleepQualityScore(quality) + consistencyScore(variance)
```

| 항목        | 범위       | 최대 점수 |
| ----------- | ---------- | --------- |
| 수면 시간   | 7-9h 최적  | 12        |
| 수면 질     | 1-5 스케일 | 10        |
| 수면 일관성 | 시간 편차  | 8         |

### 2.2 스트레스 점수 (0-25, 역방향)

```
stressScore = max(25 - normalizedStress * 2.5, 0)
```

스트레스 1-10 → 0이 최적 (25점), 10이 최악 (0점)

### 2.3 에너지 점수 (0-20)

```
energyScore = energyLevel * 4  (1-5 스케일)
```

### 2.4 기분 점수 (0-25)

```
moodScore = moodLevel * 5  (1-5 스케일)
```

### 2.5 총 바이오리듬 점수

```
totalBiorhythm = sleepScore + stressScore + energyScore + moodScore  (0-100)
```

## 3. 웰니스 스코어 연동

바이오리듬은 기존 웰니스 4영역(운동/영양/피부/체형)에 **보정 계수**로 적용:

```
adjustedWellness = baseWellness * biorhythmModifier
biorhythmModifier = 0.85 + (biorhythmScore / 100) * 0.30
// 범위: 0.85 (바이오리듬 0점) ~ 1.15 (바이오리듬 100점)
```

즉, 바이오리듬이 좋으면 웰니스 점수가 최대 15% 상승, 나쁘면 최대 15% 하락.

## 4. 피부 연결 인사이트

| 바이오리듬 상태 | 피부 영향             | 추천                |
| --------------- | --------------------- | ------------------- |
| 수면 부족       | 재생 저하, 다크서클   | 보습 강화, 아이크림 |
| 고스트레스      | 피지↑, 트러블         | 진정 케어, 클렌징   |
| 저에너지        | 혈액순환 저하, 칙칙함 | 마사지, 비타민C     |
| 여포기(여성)    | 피부 최상             | 새 제품 시도 적합   |
| 황체기(여성)    | 피지↑                 | 각질 관리, 비타민A  |
