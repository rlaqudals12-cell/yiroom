# 성분 안전 판정 · 효과 타임라인 원리

> 제품 스캔 "나와의 적합도" 판정(ADR-112)의 과학적·규제적 기반.
> 리서치: [2026-07-product-scan-grading-medical.md](../research/claude-ai-research/2026-07-product-scan-grading-medical.md)

## 1. 핵심 개념

### 1.1 위해성(risk) ≠ 유해성(hazard)

독성학의 기본: 위해성 = 유해성 × 노출량(농도·빈도). 성분 하나의 유해성만으로 제품을 판정하는 것(Yuka 방식)은 농도를 무시해 비과학적이다. 소비자 앱은 제품별 농도를 알 수 없으므로, **성분 단위의 "절대 안전등급" 단정은 원리적으로 불가능**하다.

### 1.2 판정 가능한 두 가지 사실

농도를 몰라도 과학적으로 말할 수 있는 것:

1. **규제 사실** — 식약처가 배합금지/제한으로 고시한 성분, 알레르기 유발 착향제 25종(고시 기반). 이는 판단이 아니라 인용이다.
2. **개인 맥락 반응성** — 특정 피부 타입/민감도에서 자극 보고가 많은 성분(예: 건성·민감 피부에 변성알코올). "제품이 나쁘다"가 아니라 "당신 피부 조건에서 자극 가능성"이라는 조건부 명제.

→ 이룸의 판정은 이 두 사실 위에서만 성립한다. 절대 등급은 원리적으로도, 법적으로도 배제.

## 2. 4-레이어 판정 모델

```
입력: 성분 리스트(바코드 조회 or OCR) + 사용자 5축 프로필
L1 성분 안전(사실):   식약처 규제·알레르겐 25종 매칭 → "규제 정보" 표시 (판단 아님)
L2 개인 적합도(차별): 피부타입·민감도·고민·PC 톤 × 성분 반응성 → 적합도 점수+사유
L3 성분 궁합:         제품 내/내 화장대 조합 충돌(레티놀+AHA 등) 경고
L4 효과 타임라인:     활성 성분 → 임상 문헌상 발현 시기 안내 (§3)
```

L2가 이룸만의 차별점(사용자 분석 프로필 보유). L1·L3은 사실 인용, L4는 문헌 인용 — 전 레이어가 "출처 있는 진술"로 구성된다.

## 3. 효과 발현 타임라인의 약리학적 근거

피부 세포 턴오버(~28일, 연령에 따라 증가)와 성분별 작용 기전 때문에 효과 발현에는 생리학적 최소 시간이 있다. 임상 문헌 기준:

| 성분                | 기전                                        | 발현 시기(임상 보고)         | 근거                        |
| ------------------- | ------------------------------------------- | ---------------------------- | --------------------------- |
| 비타민C             | 티로시나제 억제·항산화·콜라겐 보조          | 광채 3-7일, 톤·색소 8-12주   | PubMed 27050703             |
| 레티놀              | 턴오버 촉진·콜라겐 합성 유도                | 결 2-4주, 주름 8-12주        | PubMed 27050703 (12주 임상) |
| 나이아신아마이드    | 피지 조절·장벽 지질 합성·멜라노좀 이동 억제 | 장벽 2-4주, 색소 8-12주      | Bissett 2005 계열           |
| 살리실릭애씨드(BHA) | 지용성 각질 용해·모공 내 침투               | 21일 유의 개선(피지 -23.65%) | PMC12274963                 |
| AHA                 | 각질 접합 분해(표면)                        | 결·톤 2-4주                  | PMC12274963                 |
| 세라마이드          | 장벽 지질 보충                              | 장벽·수분 4주                | PMC12945871                 |
| 히알루론산          | 수분 결합(표면 보습)                        | 즉시~수시간                  | 산 계열 종설                |

### 3.1 표현 원리 (화장품법 정합)

- ❌ 예측·보장: "2주 후 주름이 사라져요" (효능 보장 + 수치 단정 = 금지 표현)
- ✅ 문헌 인용: "임상 연구에서는 보통 8-12주 후 톤 개선이 보고돼요 (출처)"
- ❌ 의약품 오인: "치료", "재생" → ✅ "개선에 도움을 줄 수 있어요"
- 개인차 면책 + 정보 제공 도구 성격 명시 상시 유지.

### 3.2 변화 추적과의 결합

타임라인은 이룸의 피부 변화 추적(직전 대비 칩)과 결합할 때 완결된다: "이 성분을 쓰기 시작했다면 N주 후 재분석으로 확인해보세요" — 문헌 안내가 재방문 트리거이자 자기 검증 수단이 된다(정직성: 효과가 없으면 없다고 나온다).

## 4. 구현 도출

- L1: `cosmetic_ingredients`(EWG 점수·주의20·알레르겐 플래그 스키마 기존) + 식약처 벌크 임포트로 확충 → 성분명 정규화(fuzzy) 후 조인.
- L2: 기존 `lib/scan/compatibility.ts` `analyzeCompatibility()` 유지·보강 (사유에 출처 필드 추가).
- L3: 기존 `lib/scan/ingredient-interactions.ts` 25규칙 유지.
- L4: 신규 `lib/scan/ingredient-timeline.ts` — §3 표를 코드 상수로(성분 INCI alias 포함), 문헌 URL 동봉.

## 5. 검증 방법

- L1 판정이 항상 출처(식약처 고시/EWG 참고)를 동반하는가.
- L4 문구가 금지 표현(치료/보장/기간 단정)을 포함하지 않는가 — 테스트로 고정.
- 동일 성분 리스트 입력 → 동일 판정(결정론).
