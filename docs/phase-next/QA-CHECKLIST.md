## 테스트 진행 방법

### 수동 테스트 절차

1. **서버 시작**

   ```bash
   cd apps/web
   npx supabase start
   npm run dev
   ```

2. **브라우저 접속**
   - http://localhost:3000

3. **체크리스트 순서대로 테스트**
   - 각 항목 ⬜ → ✅ (성공) 또는 ❌ (실패) 업데이트

4. **버그 발견 시**
   - GitHub Issue 생성
   - 스크린샷 첨부
   - 재현 단계 기록
