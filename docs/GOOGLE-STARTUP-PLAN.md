## 1. 개요

### 1.1 목적

이 문서는 Google for Startups Cloud Program 및 Google AI Tier 혜택을 최대한 활용하기 위한 전략을 정의합니다.

## 2. AI 티어 신청 요건

### 2.1 기본 자격 요건

| 요건 | 이룸 상태 | 충족 |
|------|----------|------|
| 설립 10년 이내 | 2025년 설립 | ✅ |
| 펀딩 시리즈 A 이하 | Pre-Seed | ✅ |
| 제품/서비스 출시 | MVP 준비 중 | ⚠️ |
| AI/ML 핵심 사용 | Gemini 기반 분석 | ✅ |
| VC 후원 (선호) | 준비 중 | ⚠️ |

### 2.2 AI 티어 추가 요건

```typescript
// AI 티어 심사 기준
const aiTierCriteria = {
  // 기술적 요건
  technical: {
    aiCoreToProduct: true,      // AI가 제품의 핵심
    scalablePotential: true,    // 확장 가능한 아키텍처
    dataPrivacy: true,          // 데이터 보호 정책
  },

  // 비즈니스 요건
  business: {
    clearGtmStrategy: true,     // 명확한 시장 진입 전략
    differentiatedValue: true,  // 차별화된 가치 제안
    potentialScale: '50K MAU',  // 성장 잠재력
  },

  // 우대 사항
  preferential: {
    vcBacked: false,            // VC 투자 (준비 중)
    acceleratorAlumni: false,   // 액셀러레이터 졸업
    existingGcpUsage: true,     // 기존 GCP 사용
  },
};
```

│   Year 1 ($150K)                                                 │

### 3.2 서비스별 크레딧 할당

| 서비스 | 비중 | 연간 예상 | 용도 |
|--------|------|----------|------|
| **Vertex AI / Gemini** | 60% | $90K | AI 분석 API |
| **Cloud Run** | 15% | $22.5K | 서버리스 컴퓨팅 |
| **Cloud Storage** | 10% | $15K | 이미지 저장 |
| **BigQuery** | 8% | $12K | 분석/데이터 |
| **기타** | 7% | $10.5K | 모니터링, 로깅 |
