## 3. 이미지 저장 동의 카드

| 테스트         | 예상 결과                              | 통과 |
| -------------- | -------------------------------------- | ---- |
| 카드 렌더링    | "피부 분석 이미지 저장 동의" 제목 표시 | ⬜   |
| 동의 상태 표시 | ConsentStatus 컴포넌트 표시            | ⬜   |
| 동의일 표시    | YYYY-MM-DD 형식                        | ⬜   |
| 보관 기한 표시 | "XX일 남음" 또는 "만료됨"              | ⬜   |
| 철회 버튼 클릭 | 확인 다이얼로그 표시                   | ⬜   |
| 철회 확인      | 성공 토스트 + 상태 업데이트            | ⬜   |
| 미동의 상태    | 노란색 안내 박스 표시                  | ⬜   |

## 4. 마케팅 동의 토글 카드

| 테스트             | 예상 결과                              | 통과 |
| ------------------ | -------------------------------------- | ---- |
| 카드 렌더링        | "마케팅 정보 수신 동의" 제목 표시      | ⬜   |
| 토글 초기 상태     | DB 값에 따라 ON/OFF                    | ⬜   |
| 토글 ON → OFF      | 즉시 UI 변경 + API 호출                | ⬜   |
| 토글 OFF → ON      | 즉시 UI 변경 + API 호출                | ⬜   |
| 성공 토스트 (동의) | "마케팅 정보 수신에 동의했습니다"      | ⬜   |
| 성공 토스트 (철회) | "마케팅 정보 수신 동의를 철회했습니다" | ⬜   |
| 동의일 표시        | 동의 상태일 때 날짜 표시               | ⬜   |
| 철회일 표시        | 미동의 상태일 때 날짜 표시             | ⬜   |
| 로딩 중 비활성화   | API 호출 중 토글 disabled              | ⬜   |
| API 실패 시 롤백   | 이전 상태로 복원 + 에러 토스트         | ⬜   |
