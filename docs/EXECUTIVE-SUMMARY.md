## Problem

```
"나에게 맞는 제품/스타일을 어떻게 찾지?"

현재 상황:
├── 퍼스널컬러, 피부 타입을 모르는 사용자 85%
├── 화장품 구매 후 불만족률 60%+
├── 전문가 진단 비용 10만원+, 접근성 낮음
└── 단편적 정보 (피부만, 컬러만, 체형만 따로)
```

---

## Solution

```
"사진 한 장으로 온전한 나를 이해한다"

이룸이 제공하는 것:
├── AI 기반 7대 분석 (퍼스널컬러, 피부, 체형, 영양, 운동, 헤어, 메이크업)
├── 크로스모듈 인사이트 (피부+영양, 체형+운동 연계)
├── 개인화 제품 추천 (어필리에이트 연동)
└── 변화 추적 및 동기부여
```

### 핵심 차별화: **크로스모듈 인사이트**

```
"피부가 건조한 이유는 수분 섭취 부족 + 운동 후 케어 미흡"
"봄 웜톤에 맞는 헤어컬러와 메이크업 팔레트 연동"

단편적 분석 → 통합적 자기 이해
```
