# 이룸(Yiroom) AI 이미지 컨설팅 통합 방법론

## 프로젝트 개요

- **플랫폼**: AI 기반 통합 웰니스/뷰티 플랫폼
- **타겟**: 한국 여성 10-30대
- **핵심 차별점**: 퍼스널컬러 + 피부 + 체형 + 얼굴형 통합 분석
- **슬로건**: "온전한 나는?" (What is my whole self?)

# PART 1: 이미지 컨설팅 방법론

## 1. 이미지 컨설턴트 방법론

### Input (수집 데이터)

```
- 라이프스타일: "일주일 일상이 어떻게 되세요?", "토요일 저녁 식사는 주로 어디서?"
- 페인포인트: "옷장 열었을 때 어떤 기분?", "쇼핑 시 언제 가장 좌절?"
- 목표: "어떤 이미지로 기억되고 싶으세요?", "인스타에서 누구 스타일 좋아하세요?"
- 퍼스널컬러/체형 진단 결과
```

### Process (처리 로직)

#### TPO 매트릭스 (5가지 상황)

| 상황        | Time      | Place    | Occasion  | 스타일 방향           |
| ----------- | --------- | -------- | --------- | --------------------- |
| 출근 포멀   | 평일/주간 | 회의실   | 중요 미팅 | 정장, 네이비/그레이   |
| 출근 캐주얼 | 평일/주간 | 오피스   | 일상 업무 | 블라우스, 슬랙스      |
| 데이트      | 저녁/주말 | 레스토랑 | 로맨틱    | 원피스, 페미닌 컬러   |
| 면접        | 평일/주간 | 회사     | 공식 평가 | 정장, 미니멀 액세서리 |
| 하객        | 주말/낮   | 웨딩홀   | 축하      | 하객룩, 적절한 화려함 |

#### 3-키워드 이미지 도출 공식

```
키워드1(실루엣) + 키워드2(미학) + 키워드3(감정) = 퍼스널 이미지

예시:
- 봄웜라이트 + 웨이브체형 + 창의직 → "페미닌 + 아티스틱 + 플레이풀"
- 겨울쿨딥 + 스트레이트체형 + 경영직 → "시크 + 파워풀 + 소피스티케이티드"
```

#### 7가지 스타일 퍼스널리티

| 유형     | 키워드           | 특징               |
| -------- | ---------------- | ------------------ |
| Classic  | 세련된, 정돈된   | 품위, 우아함       |
| Dramatic | 강렬한, 임팩트   | 대담함, 자신감     |
| Natural  | 내추럴, 캐주얼   | 편안함, 자연스러움 |
| Elegant  | 우아한, 시크     | 세련미             |
| Romantic | 러블리, 페미닌   | 부드러움           |
| Sexy     | 글래머러스       | 매혹적             |
| Creative | 유니크, 에클레틱 | 독창적             |

### Output (제공 솔루션)

```
✓ 퍼스널 이미지 키워드 3개
✓ TPO별 스타일링 가이드 (5-7개 상황)
✓ 피해야 할 스타일 정의
✓ 시그니처 룩 제안
```

## 2. 스타일리스트 방법론

### 믹스앤매치 공식

#### 60-30-10 컬러 비율

```
60% = 메인 컬러 (하의, 재킷)
30% = 세컨더리 (상의, 니트)
10% = 악센트 (액세서리, 신발)
```

#### 컬러 매칭 4가지 방법

| 방법          | 설명                | 예시                  |
| ------------- | ------------------- | --------------------- |
| 모노크롬      | 같은 색 다른 톤     | 베이지+카멜+브라운    |
| 보색 대비     | 컬러휠 반대 (70:30) | 네이비+오렌지         |
| 유사색        | 컬러휠 인접         | 블루+그린             |
| 뉴트럴+포인트 | 무채색+컬러         | 블랙&화이트+레드 가방 |

#### 패턴 매칭 규칙

```
"Two Plain + One Fancy" = 3개 중 2개 무지, 1개만 패턴
```

#### 비율/실루엣 공식

```
1/3 법칙: 허리 위 1/3 + 허리 아래 2/3 → 다리 길어 보임
볼륨 밸런스: 상의 오버핏 → 하의 슬림핏 (또는 반대)
```

### 캡슐 옷장 구성 (총 30-50개)

| 카테고리    | 수량 | 필수 아이템              |
| ----------- | ---- | ------------------------ |
| 하의        | 5-7  | 데일리 3-4 + 포멀 1-2    |
| 티/탱크탑   | 5-7  | 화이트/블랙/뉴트럴 필수  |
| 탑/블라우스 | 3-5  | 화이트 셔츠 필수         |
| 니트        | 3-5  | 가디건 1-2 포함          |
| 원피스      | 2-3  | 캐주얼 + 포멀            |
| 아우터      | 3-5  | 블레이저, 코트, 재킷     |
| 신발        | 4-5  | 스니커즈, 플랫, 부츠, 힐 |

**컬러 비율**: 50% 뉴트럴 + 40% 토널 뉴트럴 + 10% 포인트

### 체형별 스타일링 보완 공식

| 체형        | 강조         | 완화   | 추천                         | 피할 것               |
| ----------- | ------------ | ------ | ---------------------------- | --------------------- |
| 🍎 사과형   | 다리, 데콜테 | 복부   | V넥, 엠파이어라인, A라인     | 타이트 바디콘, 하이넥 |
| 🍐 배형     | 상체, 허리선 | 엉덩이 | 보트넥, 퍼프소매, 밝은 상의  | 힙라인 패턴, 스키니진 |
| ⌛ 모래시계 | 허리선, 곡선 | -      | 랩원피스, 벨티드, 하이웨스트 | 박시한 옷             |
| 📐 직사각형 | 허리 만들기  | -      | 벨트, 페플럼, 레이어드       | 일자형 원피스         |
| 🔻 역삼각형 | 하체, 힙라인 | 어깨   | V넥, 플레어스커트, 밝은 하의 | 어깨패드, 보트넥      |

## 3. 퍼스널 컬러 컨설턴트 방법론

### 디지털 드레이핑 5가지 핵심 변수

| 변수      | 측정 방법                                           | 분류 기준                                                      |
| --------- | --------------------------------------------------- | -------------------------------------------------------------- |
| 피부 밝기 | ITA° = arctan((L*-50)/b*) × (180/π)                 | Very Light(>55), Light(41-55), Intermediate(28-41), Tan(10-28) |
| 언더톤    | Lab a*, b* 값 분석                                  | 웜(b* 높음), 쿨(a* 높음), 뉴트럴(균형)                         |
| 대비감    | Contrast = \|L*skin - L*hair\| + \|L*skin - L*eye\| | High(>40), Medium(20-40), Low(<20)                             |
| 채도      | Chroma = √(a*² + b*²)                               | Bright/Vivid(고채도), Muted/Soft(저채도)                       |
| 두피색    | 두피색 분석                                         | 한국 연구에서 최고 상관관계                                    |

### 4계절 기본 분류

| 계절    | 톤  | 명도      | 채도          | 특성                   |
| ------- | --- | --------- | ------------- | ---------------------- |
| 🌸 봄   | 웜  | 고명도    | 중-고채도     | 밝고 따뜻하며 선명     |
| 🌿 여름 | 쿨  | 고-중명도 | 저-중채도     | 부드럽고 시원하며 탁함 |
| 🍂 가을 | 웜  | 중명도    | 중채도        | 깊고 따뜻하며 부드러움 |
| ❄️ 겨울 | 쿨  | 극단적    | 극고/극저채도 | 강렬하고 차갑고 선명   |

### 12타입 세분화

| 계절 | 3가지 세부 유형                                                        |
| ---- | ---------------------------------------------------------------------- |
| 봄   | Light(파스텔), Bright(선명 원색), Warm/True(골든)                      |
| 여름 | Light(연한 파스텔), Muted/Soft(그레이시), Cool/True(블루 베이스)       |
| 가을 | Muted/Soft(부드러운 어스톤), Deep/Dark(깊은 갈색), Warm/True(머스타드) |
| 겨울 | Deep(주얼리 톤), Bright/Clear(선명 원색), Cool/True(아이시)            |

### 계절별 베스트/워스트 컬러

| 타입             | 베스트                         | 워스트               |
| ---------------- | ------------------------------ | -------------------- |
| 🌸 봄 라이트     | 피치, 코랄핑크, 아이보리, 민트 | 버건디, 블랙         |
| 🌿 여름 뮤트     | 라벤더, 더스티로즈, 세이지그린 | 오렌지, 골드         |
| 🍂 가을 딥       | 테라코타, 올리브그린, 머스타드 | 네온핑크, 아이시블루 |
| ❄️ 겨울 브라이트 | 퓨어화이트, 트루레드, 로열블루 | 베이지, 살몬         |

# PART 2: 얼굴형 진단 방법론

## 1. 얼굴형(Face Shape) 분류 체계

### 6가지 표준 얼굴형

| 얼굴형                | 특징                                  | 판별 기준                       | 인상/이미지          |
| --------------------- | ------------------------------------- | ------------------------------- | -------------------- |
| 계란형(Oval)          | 길이가 너비의 약 1.5배, 턱선 부드러움 | 길이/너비 ≈ 1.5, 이마 > 턱      | 부드러움, 균형       |
| 둥근형(Round)         | 가로와 세로 거의 동일, 턱선 둥금      | 길이/너비 ≈ 1.0, 광대 가장 넓음 | 동안, 친근함, 귀여움 |
| 각진형(Square)        | 이마·광대·턱선 비슷, 턱선 강하고 각짐 | 이마 ≈ 광대 ≈ 턱선, 각진 턱     | 강인함, 신뢰감       |
| 긴형(Oblong)          | 세로가 현저히 김, 뺨 라인 직선적      | 길이/너비 > 1.5                 | 지적, 세련됨, 성숙함 |
| 하트형(Heart)         | 넓은 이마·광대, V라인 턱              | 이마 ≥ 광대 > 턱선              | 사랑스러움, 여성적   |
| 다이아몬드형(Diamond) | 광대뼈 가장 넓음, 이마와 턱 좁음      | 광대 > 이마, 광대 > 턱선        | 강렬함, 세련됨       |

### 얼굴형 측정 기준

| 측정 부위   | 기준점                | 측정 방법                             |
| ----------- | --------------------- | ------------------------------------- |
| 이마 너비   | 눈썹 위 아치 정점     | 헤어라인과 눈썹 중간의 가장 넓은 지점 |
| 광대뼈 너비 | 눈 바깥쪽 아래 돌출점 | 양쪽 가장 두드러진 광대 지점 간 거리  |
| 턱선 너비   | 귀밑 턱 꺾이는 지점   | 턱 끝에서 귀 아래까지 × 2             |
| 얼굴 길이   | 헤어라인 중심~턱 끝   | 수직 거리 측정                        |

### 비율 계산 공식

```python
이마_비율 = 이마_너비 ÷ 얼굴_길이
광대_비율 = 광대_너비 ÷ 얼굴_길이
턱선_비율 = 턱선_너비 ÷ 얼굴_길이
길이너비_비율 = 얼굴_길이 ÷ max(이마, 광대, 턱선)
```

### 한국인 평균치 (20대 여성)

- 얼굴 길이: 18.2cm
- 얼굴 폭: 13.4cm
- 길이/폭 지수: 1.35 (서양인 1.39보다 낮음)

## 2. 이목구비(Facial Features) 분석

### 쌍꺼풀 유형 (한국인 특화)

| 유형       | 특징                                    | 인상/이미지                |
| ---------- | --------------------------------------- | -------------------------- |
| 아웃라인   | 쌍꺼풀 앞머리가 눈 앞머리와 만나지 않음 | 화려함, 서구적, 큰 눈      |
| 인아웃라인 | 쌍꺼풀이 눈 앞머리와 만남               | 또렷함, 가장 선호되는 라인 |
| 인라인     | 쌍꺼풀 앞머리가 안쪽으로 말려들어감     | 여리여리함, 자연스러움     |
| 속쌍꺼풀   | 눈 뜰 때 쌍꺼풀 보이지 않음             | 날카로움, 시원한 눈매      |
| 무쌍       | 쌍꺼풀 없음                             | 동양적, 새침한 매력        |

### 눈 모양 분류

| 유형     | 판별 기준                              | 인상             |
| -------- | -------------------------------------- | ---------------- |
| 아몬드형 | 홍채가 위아래 눈꺼풀에 닿음, 양끝 뾰족 | 균형미           |
| 둥근형   | 홍채 주변 흰자 보임, 큰 눈             | 귀여움, 순수함   |
| 처진눈   | 눈꼬리가 수평선보다 아래               | 온화함, 부드러움 |
| 올라간눈 | 눈꼬리가 수평선보다 위                 | 날카로움, 세련됨 |

### 눈 사이 거리 분류

```

가까운 눈: ES Ratio < 0.44
표준: ES Ratio 0.45-0.47
떨어진 눈: ES Ratio > 0.48
```

## 3. 이미지 유형 분류 체계

### 4유형 분류 (Active/Beauty/Cute/Luxury)

| 유형   | 정의                | 얼굴 특성                          | 스타일링 방향              |
| ------ | ------------------- | ---------------------------------- | -------------------------- |
| Active | 활동적, 건강한      | 직선적 골격, 선명한 이목구비       | 스포티 캐주얼, 선명한 색상 |
| Beauty | 전통적 미인, 청순한 | 곡선적 골격, 계란형, 부드러운 눈매 | 페미닌, 파스텔 톤          |
| Cute   | 귀엽고 사랑스러운   | 둥근 얼굴, 동안, 큰 눈             | 러블리, 밝은 색상          |
| Luxury | 고급스럽고 세련된   | 긴 얼굴, 성숙함, 선명한 대비       | 엘레강스, 모노톤/딥톤      |

### 첫인상 키워드 매트릭스

| 조합          | 이미지 키워드            |
| ------------- | ------------------------ |

### 얼굴형 분류 알고리즘

```python
def classify_face_shape(measurements):
    """
    Input: 이마너비, 광대너비, 턱선너비, 얼굴길이, 턱각도
    Output: {'shape': 얼굴형, 'confidence': 신뢰도}
    """
    fw, cw, jw, fl = measurements['forehead'], measurements['cheekbone'],
                     measurements['jawline'], measurements['face_length']

    length_ratio = fl / max(fw, cw, jw)

    if length_ratio > 1.4 and fw > jw and measurements['jaw_angle'] == 'round':
        return {'shape': 'Oval', 'confidence': 0.90}

    elif length_ratio < 1.2 and cw >= fw and cw >= jw:
        if measurements['jaw_angle'] == 'round':
            return {'shape': 'Round', 'confidence': 0.88}
        else:
            return {'shape': 'Square', 'confidence': 0.90}

    elif cw > fw and cw > jw and jw < fw:
        return {'shape': 'Diamond', 'confidence': 0.87}

    elif fw >= cw and fw > jw and jw < cw * 0.7:
        return {'shape': 'Heart', 'confidence': 0.85}

    elif length_ratio > 1.5:
        return {'shape': 'Oblong', 'confidence': 0.85}
```
