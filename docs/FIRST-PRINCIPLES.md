## 왜 통합했는가?

P0 (요구사항 의심) 원칙 적용:

1. **중복 문제**: 같은 역할의 문서가 2개 존재 (97줄 vs 605줄)
2. **버전 충돌**: docs 버전에 P3-P8이 누락되어 정보 불일치
3. **Single Source of Truth**: 하나의 정규 문서만 유지
