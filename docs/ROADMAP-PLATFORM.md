# 이룸 플랫폼 수익화 로드맵

> 저위험, 사용자 신뢰 중심의 수익 모델
> **최종 업데이트**: 2026-01-11

## 핵심 원칙

```yaml
신뢰 최우선:
  - 광고 없는 100% AI 매칭 추천
  - 상위 노출/프로모션 배너 없음
  - 수수료가 추천에 영향 주지 않음

저위험 운영:
  - 자체 브랜드/서비스 제외
  - 재고 보유 없음 (중개 전용)
  - 업계 최저 수수료 (3%)

성장 우선:
  - 사용자(소비자) 먼저 확보
  - 임계량 달성 후 입점 유치
  - 수수료보다 거래량 중심
```

### 예상 수익 (MAU 1만 기준)

```
월간 활성 사용자: 10,000명
제품 클릭률: 5% = 500명/월
구매 전환율: 10% = 50건/월
평균 구매가: 30,000원
수수료율: 3%

월 수익 = 50 × 30,000 × 0.03 = 45,000원
```

## Phase 2: 입점 플랫폼 (MAU 10만+)

### CPS 수수료 모델

```yaml
수수료율: 3% (업계 최저)

비교:
  - 에이블리: 6.96%
  - 지그재그: 8.5%
  - 무신사: 9.4%
  - 이룸: 3% ← 차별점
```

### 입점 대상

| 카테고리 | 타겟 브랜드      | 예시                   |
| -------- | ---------------- | ---------------------- |
| 스킨케어 | 클린뷰티 브랜드  | 아로마티카, 라운드랩   |
| 영양제   | 이너뷰티         | 뉴트리코어, 덴프스     |
| 패션     | 체형 특화 브랜드 | 플러스사이즈, 톨사이즈 |
| 운동용품 | 홈트 브랜드      | 반석스포츠, 코어밸런스 |

### 입점 혜택 (수수료 외 비용 0원)

- 입점비: 무료
- 광고비: 없음 (광고 시스템 자체가 없음)
- 프로모션: AI 매칭 점수로만 노출

### 예상 수익 (MAU 10만 기준)

```
월간 활성 사용자: 100,000명
입점 제품 구매율: 2% = 2,000건/월
평균 구매가: 50,000원
수수료율: 3%

월 수익 = 2,000 × 50,000 × 0.03 = 3,000,000원
```

### 구현 항목

- [ ] 브랜드 입점 신청 페이지
- [ ] 브랜드 대시보드 (판매 통계)
- [ ] 제품 등록 API
- [ ] 주문 연동 시스템

## Chicken-Egg 해결 전략

### 소비자 먼저 (Subsidy Side)

```
1. 무료 AI 분석 → 사용자 유입
2. 분석 결과 공유 → 바이럴
3. 제품 추천 클릭 → 어필리에이트 수익
4. 데이터 축적 → AI 정확도 향상
```

### 입점 유치 (Money Side)

```
임계량 달성 후:
1. "MAU 10만 사용자에게 노출"
2. "광고비 0원, 수수료 3%만"
3. "AI 매칭으로 타겟 고객만"
4. "브랜드 대시보드 무료 제공"
```
